调试测试

edit

如果你的测试没有像预期的那样工作，你可以在浏览器中检查和调试它们。

在浏览器中调试规范，就像调试应用一样。

  1. 显示 Karma 浏览器窗口。 如果你在这一步需要帮助，请参阅 建立测试环境。

  2. 点击 DEBUG 按钮以打开新的浏览器标签页并重新运行测试。

  3. 打开浏览器的 开发者工具。在 Windows 上，按 Ctrl-Shift-I。在 macOS 上，按 Command-Option-I。

  4. 选择 Sources 部分。

  5. 按 Control/Command-P，然后开始输入你的测试文件的名称以打开它。

  6. 在测试中设置一个断点。

  7. 刷新浏览器，并注意它是如何在断点处停止的。
