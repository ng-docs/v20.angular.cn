在 Angular 中使用 Tailwind CSS

edit

Tailwind CSS 是一个工具优先的 CSS 框架，您可以在不离开 HTML 的情况下构建现代网站。本指南将引导您在 Angular 项目中设置 Tailwind CSS。

设置 Tailwind CSS

1. 创建一个 Angular 项目

首先，如果您还没有设置项目，请创建一个新的 Angular 项目。

ng new my-projectcd my-project

2. 安装 Tailwind CSS

接下来，在您的 Angular 项目的根目录中打开终端，并运行以下命令来安装 Tailwind CSS 及其对等依赖项：

npm install tailwindcss @tailwindcss/postcss postcss

3. 配置 PostCSS 插件

接下来，在项目的根目录下添加一个 .postcssrc.json 文件。 将 @tailwindcss/postcss 插件添加到您的 PostCSS 配置中。

.postcssrc.json

{  "plugins": {    "@tailwindcss/postcss": {}  }}

4. 导入 Tailwind CSS

在 ./src/styles.css 中添加一个导入 Tailwind CSS 的 @import。

src/styles.css

@import "tailwindcss";

如果您正在使用 SCSS，请在 ./src/styles.scss 中添加 @use。

src/styles.scss

@use "tailwindcss";

5. 开始在您的项目中使用 Tailwind

您现在可以在组件模板中使用 Tailwind 的工具类来设置应用程序的样式了。

例如，您可以将以下内容添加到您的 app.html 文件中：

<h1 class="text-3xl font-bold underline">  Hello world!</h1>

6. 开始在您的项目中使用 Tailwind

使用 ng serve 运行您的构建过程，您应该会看到已设置样式的标题。
