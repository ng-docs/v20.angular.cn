绑定动态文本、属性（Property）和属性（Attribute）

edit

在 Angular 中，绑定在组件的模板和数据之间创建动态连接。这种连接确保了组件数据的更改会自动更新渲染后的模板。

使用文本插值渲染动态文本

你可以在模板中使用双花括号绑定动态文本，这告诉 Angular 它负责处理花括号内的表达式，并确保它被正确更新。这被称为文本插值。

@Component({  template: `    <p>Your color preference is {{ theme }}.</p>  `,  ...})export class AppComponent {  theme = 'dark';}

在这个范例中，当代码片段渲染到页面上时，Angular 会将 {{ theme }} 替换为 dark。

<!-- Rendered Output --><p>Your color preference is dark.</p>

随时间变化的绑定应从 signals 读取值。Angular 会跟踪模板中读取的 signals，并在这些 signal 值更改时更新已渲染的页面。

@Component({  template: `    <!-- Does not necessarily update when `welcomeMessage` changes. -->     <p>{{ welcomeMessage }}</p>     <p>Your color preference is {{ theme() }}.</p> <!-- Always updates when the value of the `name` signal changes. -->  `  ...})export class AppComponent {  welcomeMessage = "Welcome, enjoy this app that we built for you";   theme = signal('dark');}

有关更多详细信息，请参阅 Signals 指南。

继续以主题为例，如果在页面加载后用户单击一个按钮将 theme signal 更新为 'light'，则页面将相应更新为：

<!-- Rendered Output --><p>Your color preference is light.</p>

你可以在通常在 HTML 中编写文本的任何地方使用文本插值。

所有表达式的值都会被转换为字符串。对象和数组会使用值的 toString 方法进行转换。

绑定动态属性（Property）和属性（Attribute）

Angular 支持使用方括号将动态值绑定到对象属性（Property）和 HTML 属性（Attribute）。

你可以绑定到 HTML 元素的 DOM 实例、组件实例或 指令实例上的属性（Property）。

原生元素属性（Property）

每个 HTML 元素都有一个对应的 DOM 表示形式。例如，每个 <button> HTML 元素都对应于 DOM 中 HTMLButtonElement 的一个实例。在 Angular 中，你可以使用属性（Property）绑定将值直接设置到元素的 DOM 表示形式。

<!-- Bind the `disabled` property on the button element's DOM object --><button [disabled]="isFormValid()">Save</button>

在这个范例中，每次 isFormValid 更改时，Angular 都会自动设置 HTMLButtonElement 实例的 disabled 属性（Property）。

组件和指令属性（Property）

当元素是 Angular 组件时，你可以使用属性（Property）绑定，使用相同的方括号语法来设置组件的输入属性（Property）。

<!-- Bind the `value` property on the `MyListbox` component instance. --><my-listbox [value]="mySelection()" />

在这个范例中，每次 mySelection 更改时，Angular 都会自动设置 MyListbox 实例的 value 属性（Property）。

你也可以绑定到指令属性（Property）。

<!-- Bind to the `ngSrc` property of the `NgOptimizedImage` directive  --><img [ngSrc]="profilePhotoUrl()" alt="The current user's profile photo">

属性（Attribute）

当你需要设置没有对应 DOM 属性（Property）的 HTML 属性（Attribute）时（例如 ARIA 属性（Attribute）或 SVG 属性（Attribute）），你可以使用 attr. 前缀将属性（Attribute）绑定到模板中的元素。

<!-- Bind the `role` attribute on the `<ul>` element to the component's `listRole` property. --><ul [attr.role]="listRole()">

在这个范例中，每次 listRole 更改时，Angular 都会通过调用 setAttribute 自动设置 <ul> 元素的 role 属性（Attribute）。

如果属性（Attribute）绑定的值为 null，Angular 会通过调用 removeAttribute 删除该属性（Attribute）。

在属性（Property）和属性（Attribute）中使用文本插值

你还可以在属性（Property）和属性（Attribute）中使用文本插值语法，方法是使用双花括号语法而不是属性（Property）或属性（Attribute）名称周围的方括号。当使用此语法时，Angular 会将赋值视为属性（Property）绑定。

<!-- Binds a value to the `alt` property of the image element's DOM object. --><img src="profile-photo.jpg" alt="Profile photo of {{ firstName() }}" >

要使用文本插值语法绑定到 Attribute，请在 Attribute 名称前加上 attr. 前缀

<button attr.aria-label="Save changes to {{ objectType() }}">

CSS 类和样式属性（Property）绑定

Angular 支持将 CSS 类和 CSS 样式属性（Property）绑定到元素的附加特性。

CSS 类

你可以创建一个 CSS 类绑定，以根据绑定值是 真值与否，有条件地在元素上添加或删除 CSS 类。

<!-- When `isExpanded` is truthy, add the `expanded` CSS class. --><ul [class.expanded]="isExpanded()">

你也可以直接绑定到 class 属性（Property）。Angular 接受三种类型的值：

class 值的说明 TypeScript 类型
一个包含一个或多个 CSS 类（以空格分隔）的字符串 string
CSS 类字符串的数组 string[]
一个对象，其中每个属性（Property）名称都是一个 CSS 类名，每个对应的值根据真值与否决定是否将该类应用于元素。 Record<string, any> 
@Component({  template: `    <ul [class]="listClasses"> ... </ul>    <section [class]="sectionClasses()"> ... </section>    <button [class]="buttonClasses()"> ... </button>  `,  ...})export class UserProfile {  listClasses = 'full-width outlined';  sectionClasses = signal(['expandable', 'elevated']);  buttonClasses = ({    highlighted: true,    embiggened: false,  });}

上面的范例渲染以下 DOM：

<ul class="full-width outlined"> ... </ul><section class="expandable elevated"> ... </section><button class="highlighted"> ... </button>

Angular 忽略任何不是有效 CSS 类名的字符串值。

当使用静态 CSS 类、直接绑定 class 以及绑定特定类时，Angular 会智能地将所有类组合在渲染结果中。

@Component({  template: `<ul class="list" [class]="listType()" [class.expanded]="isExpanded()"> ...`,  ...})export class Listbox {  listType = signal('box');  isExpanded = signal(true);}

在上面的范例中，Angular 使用所有三个 CSS 类渲染 ul 元素。

<ul class="list box expanded">

Angular 不保证渲染元素上 CSS 类的任何特定顺序。

当将 class 绑定到数组或对象时，Angular 使用三等号运算符 (===) 将先前的值与当前值进行比较。当你修改这些值时，必须创建一个新的对象或数组实例，以便 Angular 应用任何更新。

如果一个元素对同一个 CSS 类有多个绑定，Angular 会按照其样式优先级顺序来求解冲突。

注意：类绑定不支持单个键中的空格分隔的类名。它们也不支持对象上的变异，因为绑定的引用保持不变。如果您需要其中之一，请使用 ngClass 指令。

CSS 样式属性（Property）

你也可以直接绑定到元素上的 CSS 样式属性（Property）。

<!-- Set the CSS `display` property based on the `isExpanded` property. --><section [style.display]="isExpanded() ? 'block' : 'none'">

你可以进一步为接受单位的 CSS 属性（Property）指定单位。

<!-- Set the CSS `height` property to a pixel value based on the `sectionHeightInPixels` property. --><section [style.height.px]="sectionHeightInPixels()">

你还可以在一个绑定中设置多个样式值。Angular 接受以下类型的值：

style 值的描述 TypeScript 类型
一个包含零个或多个 CSS 声明的字符串，例如 "display: flex; margin: 8px"。 string
一个对象，其中每个属性（Property）名称都是一个 CSS 属性（Property）名称，每个对应的值都是该 CSS 属性（Property）的值。 Record<string, any> 
@Component({  template: `    <ul [style]="listStyles()"> ... </ul>    <section [style]="sectionStyles()"> ... </section>  `,  ...})export class UserProfile {  listStyles = signal('display: flex; padding: 8px');  sectionStyles = signal({    border: '1px solid black',    'font-weight': 'bold',  });}

上面的范例渲染出如下 DOM。

<ul style="display: flex; padding: 8px"> ... </ul><section style="border: 1px solid black; font-weight: bold"> ... </section>

当将 style 绑定到对象时，Angular 使用三等号运算符 (===) 将先前的值与当前值进行比较。在修改这些值时，您必须创建一个新的对象实例，以便 Angular 应用任何更新。

如果一个元素针对同一个 style 属性有多个绑定，Angular 会按照其 style 优先级顺序来解决冲突。
