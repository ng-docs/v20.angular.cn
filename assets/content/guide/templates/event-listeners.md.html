添加事件监听器

edit

Angular 支持在模板中通过在圆括号内指定事件名称以及在每次事件发生时运行的语句，在元素上定义事件监听器。

监听原生事件

当你想为 HTML 元素添加事件监听器时，可以用圆括号 () 把事件名称括起来，这样你就可以指定一个监听器语句。

@Component({  template: `    <input type="text" (keyup)="updateField()" />  `,  ...})export class AppComponent{  updateField(): void {    console.log('Field is updated!');  }}

在这个例子中，每当 <input> 元素发出 keyup 事件时，Angular 都会调用 updateField。

你可以为任何原生事件添加监听器，例如：click、keydown、mouseover 等。要了解更多信息，请查看 MDN 上 元素上所有可用的事件。

访问事件参数

在每个模板事件监听器中，Angular 都会提供一个名为 $event 的变量，其中包含对事件对象的引用。

@Component({  template: `    <input type="text" (keyup)="updateField($event)" />  `,  ...})export class AppComponent {  updateField(event: KeyboardEvent): void {    console.log(`The user pressed: ${event.key}`);  }}

使用按键修饰符

当你想捕获特定按键的特定键盘事件时，你可能会编写如下代码：

@Component({  template: `    <input type="text" (keyup)="updateField($event)" />  `,  ...})export class AppComponent {  updateField(event: KeyboardEvent): void {    if (event.key === 'Enter') {      console.log('The user pressed enter in the text field.');    }  }}

然而，由于这是一个常见的场景，Angular 允许你通过使用句点 (.) 字符指定特定按键来过滤事件。 这样做，代码可以简化为：

@Component({  template: `    <input type="text" (keyup.enter)="updateField($event)" />  `,  ...})export class AppComponent{  updateField(event: KeyboardEvent): void {    console.log('The user pressed enter in the text field.');  }}

你还可以添加额外的按键修饰符：

<!-- Matches shift and enter --><input type="text" (keyup.shift.enter)="updateField($event)" />

Angular 支持修饰符 alt、control、meta 和 shift。

你可以指定要绑定到键盘事件的 key 或 code。 key 和 code 字段是浏览器键盘事件对象的原生组成部分。 默认情况下，事件绑定假定你想使用 键盘事件的 Key 值。

Angular 还允许你通过提供内置的 code 后缀来指定 键盘事件的 Code 值。

<!-- Matches alt and left shift --><input type="text" (keydown.code.alt.shiftleft)="updateField($event)" />

这对于跨不同操作系统一致地处理键盘事件非常有用。 例如，在 MacOS 设备上使用 Alt 键时，key 属性会基于已被 Alt 键修改过的字符报告按键。 这意味着像 Alt + S 这样的组合会报告 'ß' 的 key 值。 但是，code 属性对应于按下的物理或虚拟按钮，而不是产生的字符。

阻止事件默认行为

如果您的事件处理程序应替换浏览器的原生行为，您可以使用事件对象的 preventDefault 方法：

@Component({  template: `    <a href="#overlay" (click)="showOverlay($event)">  `,  ...})export class AppComponent{  showOverlay(event: PointerEvent): void {    event.preventDefault();    console.log('Show overlay without updating the URL!');  }}

如果事件处理程序语句的计算结果为 false，Angular 会自动调用 preventDefault()，这与 原生事件处理程序属性 类似。始终优先显式调用 preventDefault，因为这种方法使代码的意图更加明显。
