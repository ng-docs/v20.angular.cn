基于语言环境格式化数据

edit

Angular 提供以下内置数据转换管道。 数据转换管道使用 LOCALE_ID 令牌根据每个语言环境的规则来格式化数据。

数据转换管道 详情
DatePipe 格式化日期值。
CurrencyPipe 将数字转换为货币字符串。
DecimalPipe 将数字转换为十进制数字字符串。
PercentPipe 将数字转换为百分比字符串。

使用 DatePipe 显示当前日期

要以当前语言环境的格式显示当前日期，请为 DatePipe 使用以下格式。

{{ today | date }}

为 CurrencyPipe 覆盖当前语言环境

将 locale 参数添加到管道以覆盖 LOCALE_ID 令牌的当前值。

要强制货币使用美式英语 (en-US)，请为 CurrencyPipe 使用以下格式

{{ amount | currency : 'en-US' }}

有帮助：为 CurrencyPipe 指定的语言环境会覆盖您应用程序的全局 LOCALE_ID 令牌。
