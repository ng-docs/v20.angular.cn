构建你的第一个 Angular 应用

edit

本教程包含多个课程，它们将介绍开始使用 Angular 编码所需的 Angular 概念。

你可以根据自己的意愿选择学习任意数量的课程，并且可以按任何顺序学习。

提示： 喜欢视频？我们还为本教程准备了完整的 YouTube 课程！

开始之前

为了获得最佳的教程体验，请查看以下要求，确保你已做好充分准备。

你的经验

本教程中的课程假定你具有以下经验：

  1. 通过直接编辑 HTML 创建了 HTML 网页。
  2. 使用 JavaScript 编写过网站内容。
  3. 阅读过层叠样式表 (CSS) 内容，并了解选择器的使用方式。
  4. 使用过命令行指令在计算机上执行任务。

你的设备

这些课程可以使用 Angular 工具的本地安装或在我们的嵌入式编辑器中完成。本地 Angular 开发可以在 Windows、MacOS 或基于 Linux 的系统上完成。

注意： 请留意像这样的警报，它们会指出可能仅适用于本地编辑器的步骤。

你的第一个 Angular 应用的概念预览

本教程中的课程将创建一个 Angular 应用，用于列出租房并显示各个房屋的详细信息。 此应用使用了许多 Angular 应用通用的特性。

本地开发环境

注意： 此步骤仅适用于你的本地环境！

在你想要用于本教程的计算机上的命令行工具中执行这些步骤。

  1. 确定 Angular 所需的 node.js 版本

    Angular 需要 Node 的 active LTS 或 maintenance LTS 版本。让我们确认你的 node.js 版本。有关特定版本要求的信息，请参阅 package.json 文件中的 engines 属性。

    从 终端 窗口中：

    1. 运行以下命令：node --version
    2. 确认显示的版本号符合要求。

  2. 安装适用于 Angular 的正确版本的 node.js

    如果你没有安装 node.js 版本，请按照 nodejs.org 上的安装说明进行安装

  3. 安装最新版本的 Angular

    在安装了 node.js 和 npm 之后，下一步是安装 Angular CLI，它为高效的 Angular 开发提供了工具。

    从 终端 窗口运行以下命令：npm install -g @angular/cli。

  4. 安装集成开发环境 (IDE)

    你可以自由使用任何你喜欢的工具来构建 Angular 应用。我们推荐以下工具：

    1. Visual Studio Code
    2. 作为一个可选但推荐的步骤，你可以通过安装 Angular 语言服务来进一步改善你的开发者体验
    3. WebStorm

  5. 可选：设置您的人工智能驱动的 IDE

    如果您在本教程中使用您偏好的人工智能驱动的 IDE，请查看 Angular 的提示规则和最佳实践。
