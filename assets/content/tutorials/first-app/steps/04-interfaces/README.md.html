创建接口

edit

本教程课程演示了如何创建一个接口并将其包含在你的应用组件中。

你将学到什么

  • 你的应用有了一个新的接口，可以将其用作数据类型。
  • 你的应用拥有一个新接口的实例，其中包含示例数据。

接口的概念预览

接口是你的应用的自定义数据类型。

Angular 使用 TypeScript 以充分利用在强类型编程环境中工作。 强类型检查降低了你的应用中的一个元素向另一个元素发送格式不正确的数据的可能性。 这种类型不匹配错误会被 TypeScript 编译器捕获，并且许多此类错误也可以在你的 IDE 中捕获。

在本课程中，你将创建一个接口来定义表示单个住房位置数据的属性（Property）。

  1. 创建一个新的 Angular 接口

    此步骤在你的应用中创建一个新接口。

    在你的 IDE 的 终端 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 在 first-app 目录中，运行此命令以创建新接口。

      ng generate interface housinglocation

    3. 运行 ng serve 以构建应用并为 http://localhost:4200 启动服务器。

    4. 在浏览器中，打开 http://localhost:4200 以查看你的应用。

    5. 确认应用构建时没有错误。 在继续下一步之前，更正所有错误。

  2. 向新接口添加属性（Property）

    此步骤将属性（Property）添加到你的应用所需的接口，以表示住房位置。

    1. 在你的 IDE 的 终端 窗格中，启动 ng serve 命令（如果尚未运行），以构建应用并为 http://localhost:4200 启动服务器。

    2. 在你的 IDE 的 编辑 窗格中，打开 src/app/housinglocation.ts 文件。

    3. 在 housinglocation.ts 中，将默认内容替换为以下代码，以使你的新接口与此范例匹配。

      更新 src/app/housinglocation.ts 以匹配此代码

      export interface HousingLocationInfo {  id: number;  name: string;  city: string;  state: string;  photo: string;  availableUnits: number;  wifi: boolean;  laundry: boolean;}

    4. 保存你的更改并确认应用未显示任何错误。 在继续下一步之前，更正所有错误。

    至此，你已经定义了一个接口，该接口表示有关住房位置的数据，包括 id、name 和位置信息。

  3. 为你的应用建立测试环境

    你有一个接口，但你尚未使用它。

    在此步骤中，你将创建一个接口的实例，并为其赋予一些示例数据。 你还不会在你的应用中看到此示例数据。 在该示例数据出现之前，还需要完成一些课程。

    1. 在你的 IDE 的 终端 窗格中，运行 ng serve 命令（如果尚未运行），以构建应用并为你的应用启动服务器，地址为 http://localhost:4200。

    2. 在 IDE 的编辑窗格中，打开 src/app/home/home.ts。

    3. 在 src/app/home/home.ts 中，在现有的 import 语句之后添加此导入语句，以便 Home 可以使用新接口。

      在 src/app/home/home.ts 中导入 Home

      import {Component} from '@angular/core';import {HousingLocation} from '../housing-location/housing-location';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-home',  imports: [HousingLocation],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.css'],})export class Home {  readonly baseUrl = 'https://angular.dev/assets/images/tutorials/common';  housingLocation: HousingLocationInfo = {    id: 9999,    name: 'Test Home',    city: 'Test city',    state: 'ST',    photo: `${this.baseUrl}/example-house.jpg`,    availableUnits: 99,    wifi: true,    laundry: false,  };}

    4. 在 src/app/home/home.ts 中，用以下代码替换空的 export class Home {} 定义，以在组件中创建新接口的单个实例。

      向 src/app/home/home.ts 添加示例数据

      import {Component} from '@angular/core';import {HousingLocation} from '../housing-location/housing-location';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-home',  imports: [HousingLocation],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.css'],})export class Home {  readonly baseUrl = 'https://angular.dev/assets/images/tutorials/common';  housingLocation: HousingLocationInfo = {    id: 9999,    name: 'Test Home',    city: 'Test city',    state: 'ST',    photo: `${this.baseUrl}/example-house.jpg`,    availableUnits: 99,    wifi: true,    laundry: false,  };}

    5. 确认你的 home.ts 文件与此示例匹配。

      src/app/home/home.ts

      import {Component} from '@angular/core';import {HousingLocation} from '../housing-location/housing-location';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-home',  imports: [HousingLocation],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.css'],})export class Home {  readonly baseUrl = 'https://angular.dev/assets/images/tutorials/common';  housingLocation: HousingLocationInfo = {    id: 9999,    name: 'Test Home',    city: 'Test city',    state: 'ST',    photo: `${this.baseUrl}/example-house.jpg`,    availableUnits: 99,    wifi: true,    laundry: false,  };}

      通过将 HousingLocation 类型的 housingLocation 属性添加到 Home 类，我们可以确认数据与接口描述匹配。如果数据不满足接口描述，IDE 将有足够的信息提供有用的错误提示。

    6. 保存你的更改并确认应用没有任何错误。 打开浏览器并确认你的应用仍然显示消息“housing-location works!”。

    7. 在继续下一步之前，请更正所有错误。

总结： 在本课程中，你创建了一个接口，该接口为你的应用创建了一个新的数据类型。 这种新的数据类型使你可以指定在何处需要 HousingLocation 数据。 这种新的数据类型还使你的 IDE 和 TypeScript 编译器可以确保在需要 HousingLocation 数据的地方使用它。
