为应用添加路由

edit

本教程课程演示了如何为你的应用添加路由。

重要提示： 我们建议使用本地环境来学习路由。

你将学到什么

在本课程结束时，你的应用将支持路由。

路由的概念预览

本教程介绍了 Angular 中的路由。路由是指从应用中的一个组件导航到另一个组件的能力。在单页应用（SPA）中，只会更新页面的部分内容，以呈现用户所请求的视图。

Angular 路由器户能够声明路由，并指定当应用请求该路由时，应在屏幕上显示哪个组件。

在本课程中，你将在你的应用中启用路由，以便导航到详情页面。

  1. 创建一个默认的详情组件

    1. 在终端中，输入以下命令来创建 Details：

      ng generate component details

      此组件将代表详情页面，其中提供了有关给定住房地点的更多信息。

  2. 为应用添加路由

    1. 在 src/app 目录下，创建一个名为 routes.ts 的文件。我们将在该文件中定义应用中的路由。

    2. 在 main.ts 中，进行以下更新以在应用中启用路由：

      1. 导入路由文件和 provideRouter 函数：

        在 src/main.ts 中导入路由详情

        /* *  Protractor support is deprecated in Angular. *  Protractor is used in this example for compatibility with Angular documentation tools. */import {bootstrapApplication, provideProtractorTestingSupport} from '@angular/platform-browser';import {App} from './app/app';import {provideRouter} from '@angular/router';import routeConfig from './app/routes';bootstrapApplication(App, {  providers: [provideProtractorTestingSupport(), provideRouter(routeConfig)],}).catch((err) => console.error(err));

      2. 更新对 bootstrapApplication 的调用，以包含路由配置：

        在 src/main.ts 中添加路由器配置

        /* *  Protractor support is deprecated in Angular. *  Protractor is used in this example for compatibility with Angular documentation tools. */import {bootstrapApplication, provideProtractorTestingSupport} from '@angular/platform-browser';import {App} from './app/app';import {provideRouter} from '@angular/router';import routeConfig from './app/routes';bootstrapApplication(App, {  providers: [provideProtractorTestingSupport(), provideRouter(routeConfig)],}).catch((err) => console.error(err));

    3. 在 src/app/app.ts 中，更新组件以使用路由：

      1. 为 RoutingModule 添加文件级导入：

        在 src/app/app.ts 中导入 RouterModule

        import {Component} from '@angular/core';import {Home} from './home/home';import {RouterModule} from '@angular/router';@Component({  selector: 'app-root',  imports: [Home, RouterModule],  template: `    <main>      <a [routerLink]="['/']">        <header class="brand-name">          <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />        </header>      </a>      <section class="content">        <router-outlet></router-outlet>      </section>    </main>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

      2. 将 RouterModule 添加到 @Component 元数据的 imports 中

        在 src/app/app.ts 中导入 RouterModule

        import {Component} from '@angular/core';import {Home} from './home/home';import {RouterModule} from '@angular/router';@Component({  selector: 'app-root',  imports: [Home, RouterModule],  template: `    <main>      <a [routerLink]="['/']">        <header class="brand-name">          <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />        </header>      </a>      <section class="content">        <router-outlet></router-outlet>      </section>    </main>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

      3. 在 template 属性中，将 <app-home></app-home> 标签替换为 <router-outlet> 指令，并添加一个返回主页的链接。你的代码应与此代码匹配：

        在 src/app/app.ts 中添加 router-outlet

        import {Component} from '@angular/core';import {Home} from './home/home';import {RouterModule} from '@angular/router';@Component({  selector: 'app-root',  imports: [Home, RouterModule],  template: `    <main>      <a [routerLink]="['/']">        <header class="brand-name">          <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />        </header>      </a>      <section class="content">        <router-outlet></router-outlet>      </section>    </main>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

  3. 为新组件添加路由

    在上一步中，你移除了模板中对 <app-home> 组件的引用。在这一步中，你将为该组件添加一个新的路由。

    1. 在 routes.ts 中，执行以下更新以创建一个路由。

      1. 为 Home、Details 和你将在路由定义中使用的 Routes 类型添加文件级导入。

        导入组件和路由

        import {Routes} from '@angular/router';import {Home} from './home/home';import {Details} from './details/details';const routeConfig: Routes = [  {    path: '',    component: Home,    title: 'Home page',  },  {    path: 'details/:id',    component: Details,    title: 'Home details',  },];export default routeConfig;

      2. 定义一个名为 routeConfig 的变量，类型为 Routes，并为应用定义两个路由：

        向应用添加路由

        import {Routes} from '@angular/router';import {Home} from './home/home';import {Details} from './details/details';const routeConfig: Routes = [  {    path: '',    component: Home,    title: 'Home page',  },  {    path: 'details/:id',    component: Details,    title: 'Home details',  },];export default routeConfig;

        routeConfig 数组中的条目代表应用中的路由。当 URL 匹配 '' 时，第一个条目导航到 Home。第二个条目使用了一些特殊格式，将在未来的课程中重新讨论。

    2. 保存所有更改并确认应用在浏览器中工作正常。应用应仍然显示住房位置列表。

总结： 在本课程中，你在你的应用中启用了路由，并定义了新的路由。现在你的应用可以支持在视图之间导航。在下一课中，你将学习导航到给定房屋位置的“详情页面”。
