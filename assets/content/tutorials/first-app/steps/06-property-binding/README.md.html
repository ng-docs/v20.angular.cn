向组件的模板添加属性绑定

edit

本教程课程演示了如何向模板添加属性绑定，并使用它将动态数据传递给组件。

你将学到什么

  • 您的应用在 Home 模板中具有数据绑定。
  • 您的应用将数据从 Home 发送到 HousingLocation。

对输入（Input）的概念性预览

在本课程中，你将继续执行从父组件向子组件共享数据的过程，方法是使用属性绑定将数据绑定到模板中的那些属性（Property）。

属性绑定使您能够将变量连接到 Angular 模板中的 Input。然后，数据将动态绑定到 Input。

要获得更深入的解释，请参阅属性绑定指南。

  1. 更新 Home 模板

    此步骤将属性绑定添加到 <app-housing-location> 标签。

    在代码编辑器中：

    1. 导航到 src/app/home/home.ts

    2. 在 @Component 装饰器的 template 属性中，将代码更新为与下方代码匹配：

      添加 housingLocation 属性绑定

      import {Component} from '@angular/core';import {HousingLocation} from '../housing-location/housing-location';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-home',  imports: [HousingLocation],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location [housingLocation]="housingLocation"></app-housing-location>    </section>  `,  styleUrls: ['./home.css'],})export class Home {  readonly baseUrl = 'https://angular.dev/assets/images/tutorials/common';  housingLocation: HousingLocationInfo = {    id: 9999,    name: 'Test Home',    city: 'Test city',    state: 'ST',    photo: `${this.baseUrl}/example-house.jpg`,    availableUnits: 99,    wifi: true,    laundry: false,  };}

      当向组件标签添加属性绑定时，我们使用 [attribute] = "value" 语法来通知 Angular，分配的值应被视为来自组件类的属性（Property），而不是字符串值。

      右侧的值是 Home 中属性的名称。

  2. 确认代码仍然有效

    1. 保存你的更改并确认应用没有任何错误。
    2. 在继续下一步之前，请更正任何错误。

摘要：在本课中，您添加了一个新的属性绑定，并传入了一个类属性的引用。现在，HousingLocation 可以访问数据，并使用这些数据来自定义组件的显示。
