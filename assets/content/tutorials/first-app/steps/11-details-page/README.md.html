将详情页面集成到应用中

edit

本教程课程演示了如何将详情页面连接到你的应用。

重要提示： 我们建议使用本地环境来学习路由。

你将学到什么

在本课程结束时，你的应用将支持路由到详情页面。

使用路由参数进行路由的概念预览

每个住房位置都有特定的详情，当用户导航到该条目的详情页面时，应该显示这些详情。为了实现这个目标，你需要使用路由参数。

路由参数使你能够将动态信息作为路由 URL 的一部分包含进来。为了标识用户点击了哪个住房位置，你将使用 HousingLocation 类型的 id 属性。

  1. 使用 routerLink 进行动态导航

    在第 10 课中，你向 src/app/routes.ts 添加了第二条路由，其中包括一个用于标识路由参数的特殊片段，即 id：

    'details/:id'

    在本例中，:id 是动态的，并将根据代码请求路由的方式而变化。

    1. 在 src/app/housing-location/housing-location.ts 中，向 section 元素添加一个锚点标签并包含 routerLink 指令：

      将带 routerLink 指令的锚点添加到 housing-location.ts

      import {Component, input} from '@angular/core';import {HousingLocationInfo} from '../housinglocation';import {RouterModule} from '@angular/router';@Component({  selector: 'app-housing-location',  imports: [RouterModule],  template: `    <section class="listing">      <img        class="listing-photo"        [src]="housingLocation.photo"        alt="Exterior photo of {{ housingLocation().name }}"        crossorigin      />      <h2 class="listing-heading">{{ housingLocation().name }}</h2>      <p class="listing-location">{{ housingLocation().city }}, {{ housingLocation().state }}</p>      <a [routerLink]="['/details', housingLocation().id]">Learn More</a>    </section>  `,  styleUrls: ['./housing-location.css'],})export class HousingLocation {  housingLocation = input.required<HousingLocationInfo>();}

      routerLink 指令使 Angular 路由器能够在应用中创建动态链接。赋予 routerLink 的值是一个包含两个条目的数组：路径的静态部分和动态数据。

      为了让 routerLink 在模板中工作，请从 '@angular/router' 添加文件级别的 RouterLink 和 RouterOutlet 的导入，然后更新组件的 imports 数组以同时包含 RouterLink 和 RouterOutlet。

    2. 此时，你可以确认路由在你的应用中工作正常。在浏览器中，刷新主页，然后单击住房位置的“了解更多”按钮。

  2. 获取路由参数

    在此步骤中，你将在 Details 中获取路由参数。目前，应用显示 details works!。接下来，你将更新代码以显示通过路由参数传递的 id 值。

    1. 在 src/app/details/details.ts 中更新模板，导入你需要在 Details 中使用的函数、类和服务：

      更新文件级别的导入

      import {Component, inject} from '@angular/core';import {ActivatedRoute} from '@angular/router';import {HousingService} from '../housing.service';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-details',  template: `    <article>      <img        class="listing-photo"        [src]="housingLocation?.photo"        alt="Exterior photo of {{ housingLocation?.name }}"        crossorigin      />      <section class="listing-description">        <h2 class="listing-heading">{{ housingLocation?.name }}</h2>        <p class="listing-location">{{ housingLocation?.city }}, {{ housingLocation?.state }}</p>      </section>      <section class="listing-features">        <h2 class="section-heading">About this housing location</h2>        <ul>          <li>Units available: {{ housingLocation?.availableUnits }}</li>          <li>Does this location have wifi: {{ housingLocation?.wifi }}</li>          <li>Does this location have laundry: {{ housingLocation?.laundry }}</li>        </ul>      </section>    </article>  `,  styleUrls: ['./details.css'],})export class Details {  route: ActivatedRoute = inject(ActivatedRoute);  housingService = inject(HousingService);  housingLocation: HousingLocationInfo | undefined;  constructor() {    const housingLocationId = Number(this.route.snapshot.params['id']);    this.housingLocation = this.housingService.getHousingLocationById(housingLocationId);  }}

    2. 更新 @Component 装饰器的 template 属性以显示 housingLocationId 的值：

      template: `<p>details works! {{ housingLocationId }}</p>`,

    3. 使用以下代码更新 Details 类的正文：

      export class Details {   route: ActivatedRoute = inject(ActivatedRoute);   housingLocationId = -1;   constructor() {     this.housingLocationId = Number(this.route.snapshot.params['id']);   } }

      此代码使 Details 可以访问 ActivatedRoute 路由功能，使你能够访问有关当前路由的数据。在 constructor 中，代码将从路由获取的 id 参数从字符串转换为数字。

    4. 保存所有更改。

    5. 在浏览器中，单击其中一个住房位置的“了解更多”链接，并确认页面上显示的数值与数据中该位置的 id 属性匹配。

  3. 自定义 Details

    现在路由在应用程序中正常工作，是时候更新 Details 的模板以显示路由参数所代表的住房位置的特定数据了。

    要访问数据，你将添加对 HousingService 的调用。

    1. 更新模板代码以匹配以下代码：

      更新 src/app/details/details.ts 中的 Details 模板

      import {Component, inject} from '@angular/core';import {ActivatedRoute} from '@angular/router';import {HousingService} from '../housing.service';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-details',  template: `    <article>      <img        class="listing-photo"        [src]="housingLocation?.photo"        alt="Exterior photo of {{ housingLocation?.name }}"        crossorigin      />      <section class="listing-description">        <h2 class="listing-heading">{{ housingLocation?.name }}</h2>        <p class="listing-location">{{ housingLocation?.city }}, {{ housingLocation?.state }}</p>      </section>      <section class="listing-features">        <h2 class="section-heading">About this housing location</h2>        <ul>          <li>Units available: {{ housingLocation?.availableUnits }}</li>          <li>Does this location have wifi: {{ housingLocation?.wifi }}</li>          <li>Does this location have laundry: {{ housingLocation?.laundry }}</li>        </ul>      </section>    </article>  `,  styleUrls: ['./details.css'],})export class Details {  route: ActivatedRoute = inject(ActivatedRoute);  housingService = inject(HousingService);  housingLocation: HousingLocationInfo | undefined;  constructor() {    const housingLocationId = Number(this.route.snapshot.params['id']);    this.housingLocation = this.housingService.getHousingLocationById(housingLocationId);  }}

      请注意，housingLocation 属性正在使用可选链式运算符 ? 进行访问。这确保了如果 housingLocation 值为 null 或 undefined，应用不会崩溃。

    2. 更新 Details 类的正文以匹配以下代码：

      更新 src/app/details/details.ts 中的 Details 类

      import {Component, inject} from '@angular/core';import {ActivatedRoute} from '@angular/router';import {HousingService} from '../housing.service';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-details',  template: `    <article>      <img        class="listing-photo"        [src]="housingLocation?.photo"        alt="Exterior photo of {{ housingLocation?.name }}"        crossorigin      />      <section class="listing-description">        <h2 class="listing-heading">{{ housingLocation?.name }}</h2>        <p class="listing-location">{{ housingLocation?.city }}, {{ housingLocation?.state }}</p>      </section>      <section class="listing-features">        <h2 class="section-heading">About this housing location</h2>        <ul>          <li>Units available: {{ housingLocation?.availableUnits }}</li>          <li>Does this location have wifi: {{ housingLocation?.wifi }}</li>          <li>Does this location have laundry: {{ housingLocation?.laundry }}</li>        </ul>      </section>    </article>  `,  styleUrls: ['./details.css'],})export class Details {  route: ActivatedRoute = inject(ActivatedRoute);  housingService = inject(HousingService);  housingLocation: HousingLocationInfo | undefined;  constructor() {    const housingLocationId = Number(this.route.snapshot.params['id']);    this.housingLocation = this.housingService.getHousingLocationById(housingLocationId);  }}

      现在，该组件具有根据所选住房位置显示正确信息的代码。现在，constructor 包括对 HousingService 的调用，以将路由参数作为参数传递给 getHousingLocationById 服务函数。

    3. 将以下样式复制到 src/app/details/details.css 文件中：

      为 Details 添加样式

      .listing-photo {  height: 600px;  width: 50%;  object-fit: cover;  border-radius: 30px;  float: right;}.listing-heading {  font-size: 48pt;  font-weight: bold;  margin-bottom: 15px;}.listing-location::before {  content: url('/assets/location-pin.svg') / '';}.listing-location {  font-size: 24pt;  margin-bottom: 15px;}.listing-features > .section-heading {  color: var(--secondary-color);  font-size: 24pt;  margin-bottom: 15px;}.listing-features {  margin-bottom: 20px;}.listing-features li {  font-size: 14pt;}li {  list-style-type: none;}.listing-apply .section-heading {  font-size: 18pt;  margin-bottom: 15px;}label, input {  display: block;}label {  color: var(--secondary-color);  font-weight: bold;  text-transform: uppercase;  font-size: 12pt;}input {  font-size: 16pt;  margin-bottom: 15px;  padding: 10px;  width: 400px;  border-top: none;  border-right: none;  border-left: none;  border-bottom: solid .3px;}@media (max-width: 1024px) {  .listing-photo {    width: 100%;    height: 400px;  }}

      并保存您的更改

    4. 在 Details 中，使用刚刚创建的 details.css 文件作为样式的来源：

      更新 details.ts 以使用创建的 css 文件

      import {Component, inject} from '@angular/core';import {ActivatedRoute} from '@angular/router';import {HousingService} from '../housing.service';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-details',  template: `    <article>      <img        class="listing-photo"        [src]="housingLocation?.photo"        alt="Exterior photo of {{ housingLocation?.name }}"        crossorigin      />      <section class="listing-description">        <h2 class="listing-heading">{{ housingLocation?.name }}</h2>        <p class="listing-location">{{ housingLocation?.city }}, {{ housingLocation?.state }}</p>      </section>      <section class="listing-features">        <h2 class="section-heading">About this housing location</h2>        <ul>          <li>Units available: {{ housingLocation?.availableUnits }}</li>          <li>Does this location have wifi: {{ housingLocation?.wifi }}</li>          <li>Does this location have laundry: {{ housingLocation?.laundry }}</li>        </ul>      </section>    </article>  `,  styleUrls: ['./details.css'],})export class Details {  route: ActivatedRoute = inject(ActivatedRoute);  housingService = inject(HousingService);  housingLocation: HousingLocationInfo | undefined;  constructor() {    const housingLocationId = Number(this.route.snapshot.params['id']);    this.housingLocation = this.housingService.getHousingLocationById(housingLocationId);  }}

    5. 在浏览器中刷新页面，并确认当你单击给定住房位置的“了解更多”链接时，详情页面会根据所选条目的数据，显示正确的信息。

  4. 检查 Home 中的导航

    在之前的课程中，您更新了 App 模板以包含一个 routerLink。添加该代码后，您的应用将在每次单击徽标时启用导航回 Home。

    1. 确认你的代码与以下内容匹配：

      确认 app.ts 中的 routerLink

      import {Component} from '@angular/core';import {Home} from './home/home';import {RouterLink, RouterOutlet} from '@angular/router';@Component({  selector: 'app-root',  imports: [Home, RouterLink, RouterOutlet],  template: `    <main>      <a [routerLink]="['/']">        <header class="brand-name">          <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />        </header>      </a>      <section class="content">        <router-outlet></router-outlet>      </section>    </main>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

      您的代码应该已是最新版本，但请确认以确保。
