为你的应用添加 HTTP 通信

edit

本指南演示了如何将 HTTP 和 API 集成到你的应用中。

到目前为止，你的应用一直是从 Angular 服务中的静态数组读取数据。下一步是使用 JSON 服务器，你的应用将通过 HTTP 与之通信。HTTP 请求将模拟使用来自服务器的数据的体验。

重要提示： 我们建议在本指南的此步骤中使用你的本地环境。
