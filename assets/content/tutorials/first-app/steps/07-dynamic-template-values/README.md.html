向组件的模板添加插值

edit

本教程课程演示了如何向 Angular 模板添加插值，以便在模板中显示动态数据。

你将学到什么

  • 你的应用将在 HousingLocation 模板中显示插值。
  • 你的应用会将房屋位置数据渲染到浏览器。

插值的概念预览

在此步骤中，你将使用插值在模板中显示从 input 属性读取的值。

在 Angular 模板中使用 {{ expression }}，你可以渲染属性、inputs 和有效的 JavaScript 表达式中的值。

要获得更深入的解释，请参阅使用插值显示值指南。

  1. 更新 HousingLocation 模板以包含插值

    此步骤在 HousingLocation 模板中添加了新的 HTML 结构和插值。

    在代码编辑器中：

    1. 导航到 src/app/housing-location/housing-location.ts
    2. 在 @Component 装饰器的 template 属性中，用以下代码替换现有的 HTML 标记：

    在 housing-location.ts 中更新 HousingLocation 模板

    import {Component, input} from '@angular/core';import {HousingLocationInfo} from '../housinglocation';@Component({  selector: 'app-housing-location',  template: `    <section class="listing">      <img        class="listing-photo"        [src]="housingLocation().photo"        alt="Exterior photo of {{ housingLocation().name }}"        crossorigin      />      <h2 class="listing-heading">{{ housingLocation().name }}</h2>      <p class="listing-location">{{ housingLocation().city }}, {{ housingLocation().state }}</p>    </section>  `,  styleUrls: ['./housing-location.css'],})export class HousingLocation {  housingLocation = input.required<HousingLocationInfo>();}

    在这个更新后的模板代码中，你使用了属性绑定将 housingLocation.photo 绑定到 src Attribute。alt Attribute 使用插值来为图像的 alt 文本提供更多上下文。

    你使用插值来包含 housingLocation 属性的 name、city 和 state 的值。

  2. 确认更改在浏览器中渲染

    1. 保存所有更改。
    2. 打开浏览器并确认应用渲染了照片、城市和州示例数据。

总结： 在本课程中，你添加了一个新的 HTML 结构，并使用 Angular 模板语法来渲染 HousingLocation 模板中的值。

现在，你掌握了两项重要的技能：

  • 将数据传递给组件
  • 将值插值到模板中

有了这些技能，你的应用现在可以共享数据并在浏览器中显示动态值了。目前为止做得很好。
