Hello world

edit

本节课是本教程的起点，本教程的后续课程都会在本节课的基础上增加新特性，以构建一个完整的 Angular 应用。在本节课中，我们将更新这个应用，以显示著名的文本 “Hello World”。

你将学到什么

本节课之后，更新后的应用会确认你和你的 IDE 已准备就绪，可以开始创建 Angular 应用了。

注意： 如果你正在使用嵌入式编辑器，请跳至第三步。 在浏览器 playground 中工作时，你无需 ng serve 即可运行应用。其他命令（如 ng generate）可以在右侧的控制台窗口中完成。

  1. 下载默认应用

    首先，单击代码编辑器右上窗格中的 “下载” 图标。这将下载一个 .zip 文件，其中包含本教程的源码。在本地终端和 IDE 中打开它，然后继续测试默认应用。

    在本教程的任何步骤中，你都可以单击此图标来下载该步骤的源码并从那里开始。

  2. 测试默认应用

    在此步骤中，下载默认起始应用后，你将构建默认的 Angular 应用。 这确认你的开发环境具备继续本教程所需的条件。

    在 IDE 的 终端 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 运行此命令以安装运行应用所需的依赖项。

      npm install

    3. 运行此命令以构建并为默认应用启动服务器。

      ng serve

      应用应该可以成功构建，且不报错。

    4. 在开发计算机的 Web 浏览器中，打开 http://localhost:4200。

    5. 确认默认网站出现在浏览器中。

    6. 在完成后续步骤时，你可以保持 ng serve 运行。

  3. 审阅项目中的文件

    在此步骤中，你将了解构成默认 Angular 应用的文件。

    在 IDE 的 资源管理器 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。
    2. 打开 src 目录以查看这些文件。
      1. 在文件资源管理器中，找到 Angular 应用文件 (/src)。
        1. index.html 是应用的顶层 HTML 模板。
        2. styles.css 是应用的顶层样式表。
        3. main.ts 是应用开始运行的地方。
        4. favicon.ico 是应用的图标，就像你在任何网站中找到的那样。
      2. 在文件资源管理器中，找到 Angular 应用的组件文件 (/app)。

        1. app.ts 是描述 app-root 组件的源文件。 这是应用中的顶层 Angular 组件。组件是 Angular 应用的基本构建块。 组件描述包括组件的代码、HTML 模板和样式，这些可以在此文件中或在单独的文件中进行描述。

          在本应用中，样式位于单独的文件中，而组件的代码和 HTML 模板则位于此文件中。

        2. app.css 是此组件的样式表。

        3. 新组件会添加到此目录中。

      3. 在文件资源管理器中，找到图像目录 (/assets)，其中包含应用使用的图像。
      4. 在文件资源管理器中，找到 Angular 应用构建和运行所需的文件和目录，但它们不是你通常会交互的文件。
        1. .angular 包含构建 Angular 应用所需的文件。
        2. .e2e 包含用于测试应用的文件。
        3. .node_modules 包含应用使用的 node.js 包。
        4. angular.json 向应用构建工具描述 Angular 应用。
        5. package.json 供 npm（node 包管理器）用来运行已完成的应用。
        6. tsconfig.* 是向 TypeScript 编译器描述应用配置的文件。

    审阅构成 Angular 应用项目的文件后，继续下一步。

  4. 创建 Hello World

    在此步骤中，你将更新 Angular 项目文件以更改显示的内容。

    在你的 IDE 中：

    1. 打开 first-app/src/index.html。 注意：此步骤和下一步仅适用于你的本地环境！

    2. 在 index.html 中，将 <title> 元素替换为以下代码以更新应用的标题。

      在 src/index.html 中替换

      <!DOCTYPE html><html lang="en"><head>  <meta charset="utf-8">  <title>Homes</title>  <base href="/">  <meta name="viewport" content="width=device-width, initial-scale=1">  <link rel="icon" type="image/x-icon" href="favicon.ico">  <link href="/assets/css/be-vietnam-pro.css" rel="stylesheet"></head><body>  <app-root></app-root></body></html>

      然后，保存你刚刚对 index.html 所做的更改。

    3. 接下来，打开 first-app/src/app/app.ts。

    4. 在 app.ts 的 @Component 定义中，将 template 行替换为以下代码，以更改应用组件中的文本。

      在 src/app/app.ts 中替换

      import {Component} from '@angular/core';@Component({  selector: 'app-root',  imports: [],  template: `    <h1>Hello world!</h1>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

    5. 在 app.ts 的 App 类定义中，将 title 行替换为以下代码，以更改组件标题。

      在 src/app/app.ts 中替换

      import {Component} from '@angular/core';@Component({  selector: 'app-root',  imports: [],  template: `    <h1>Hello world!</h1>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

      然后，保存你对 app.ts 所做的更改。

    6. 如果你在步骤 1 中停止了 ng serve 命令，在你的 IDE 的 Terminal 窗口中，再次运行 ng serve。

    7. 打开你的浏览器并导航到 localhost:4200，确认应用能够成功构建并且在应用标题中显示 Homes，在应用正文中显示 Hello world：

总结： 在本课程中，你更新了一个默认的 Angular 应用以显示Hello world。 在此过程中，你了解了 ng serve 命令，用于在本地启动服务器以测试你的应用。
