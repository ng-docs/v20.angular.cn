向你的 Angular 应用添加表单

edit

本教程课程演示了如何向 Angular 应用添加一个收集用户数据的表单。 本课程从一个功能正常的 Angular 应用开始，并展示如何向其添加表单。

表单收集的数据仅发送到应用的 Service，Service 会将其写入浏览器的控制台。 本课程不介绍如何使用 REST API 发送和接收表单数据。

重要提示： 我们建议在本教程的此步骤中使用你的本地环境。

你将学到什么

  • 你的应用有一个表单，用户可以在其中输入数据，这些数据将发送到你的应用的 Service。
  • Service 将表单中的数据写入浏览器的控制台日志。

  4. 测试你的应用的新表单

    此步骤测试新表单，以查看当表单数据提交到应用时，表单数据是否出现在控制台日志中。

    1. 在 IDE 的 终端 窗格中，运行 ng serve（如果尚未运行）。
    2. 在浏览器中，在 http://localhost:4200 打开你的应用。
    3. 在浏览器中右键单击应用，然后从上下文菜单中选择 检查。
    4. 在开发者工具窗口中，选择 控制台 选项卡。 确保开发者工具窗口在后续步骤中可见
    5. 在你的应用中：
      1. 选择一个住房地点，然后单击 了解更多，以查看有关该房屋的详细信息。
      2. 在该房屋的详情页面中，滚动到底部以找到新表单。
      3. 在表单的字段中输入数据 - 任何数据都可以。
      4. 选择 立即申请 以提交数据。
    6. 在开发者工具窗口中，查看日志输出以查找你的表单数据。
