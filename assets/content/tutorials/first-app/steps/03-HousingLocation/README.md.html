创建应用的 HousingLocation 组件

edit

本教程课程演示了如何将 HousingLocation 组件添加到你的 Angular 应用中。

你将学到什么

  • 你的应用有了一个新组件：HousingLocation，它显示一条消息，确认该组件已添加到你的应用中。

  1. 创建 HousingLocation

    在此步骤中，你将为你的应用创建一个新组件。

    在你的 IDE 的 终端 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 运行此命令来创建一个新的 HousingLocation

      ng generate component housingLocation

    3. 运行此命令以构建并为你的应用启动服务器。

      ng serve

      注意：此步骤仅适用于你的本地环境！

    4. 打开浏览器并导航到 http://localhost:4200 以查找该应用。

    5. 确认该应用构建时没有错误。

      提示：它应该渲染得和上一课中一样，因为即使你添加了一个新组件，你还没有把它包含在应用的任何模板中。

    6. 在完成后续步骤时，保持 ng serve 运行。

  2. 将新组件添加到应用的布局中

    在此步骤中，你将新组件 HousingLocation 添加到你的应用的 Home 中，以便它显示在你的应用布局中。

    在你的 IDE 的 编辑 窗格中：

    1. 在编辑器中打开 home.ts。

    2. 在 home.ts 中，通过将此行添加到文件级别的导入中来导入 HousingLocation。

      在 src/app/home/home.ts 中导入 HousingLocation

      import {Component} from '@angular/core';import {HousingLocation} from '../housing-location/housing-location';@Component({  selector: 'app-home',  imports: [HousingLocation],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.css'],})export class Home {}

    3. 接下来，通过将 HousingLocation 添加到数组中来更新 @Component 元数据的 imports 属性。

      在 src/app/home/home.ts 的 imports 数组中添加 HousingLocation

      import {Component} from '@angular/core';import {HousingLocation} from '../housing-location/housing-location';@Component({  selector: 'app-home',  imports: [HousingLocation],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.css'],})export class Home {}

    4. 现在组件已准备好在 Home 的模板中使用。更新 @Component 元数据的 template 属性，以包含对 <app-housing-location> 标签的引用。

      在 src/app/home/home.ts 的组件模板中添加住房位置

      import {Component} from '@angular/core';import {HousingLocation} from '../housing-location/housing-location';@Component({  selector: 'app-home',  imports: [HousingLocation],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.css'],})export class Home {}

  3. 为组件添加样式

    在此步骤中，你将把 HousingLocation 的预写样式复制到你的应用中，以便应用能够正确渲染。

    1. 打开 src/app/housing-location/housing-location.css，并将下面的样式粘贴到文件中：

      注意：在浏览器中，这些可以放在 src/app/housing-location/housing-location.ts 的 styles 数组中。

      在 src/app/housing-location/housing-location.css 中为组件的住房位置添加 CSS 样式

      .listing {  background: var(--accent-color);  border-radius: 30px;  padding-bottom: 30px;}.listing-heading {  color: var(--primary-color);  padding: 10px 20px 0 20px;}.listing-photo {  height: 250px;  width: 100%;  object-fit: cover;  border-radius: 30px 30px 0 0;}.listing-location {  padding: 10px 20px 20px 20px;}.listing-location::before {  content: url("/assets/location-pin.svg") / "";}section.listing a {  padding-left: 20px;  text-decoration: none;  color: var(--primary-color);}section.listing a::after {  content: "\203A";  margin-left: 5px;}

    2. 保存你的代码，返回到浏览器并确认该应用构建时没有错误。你应该看到消息 “housing-location works!” 渲染到屏幕上。在继续下一步之前，请纠正任何错误。

总结： 在本课程中，你为你的应用创建了一个新组件，并将其添加到应用的布局中。
