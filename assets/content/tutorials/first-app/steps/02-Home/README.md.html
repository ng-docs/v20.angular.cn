创建 Home 组件

edit

本教程演示了如何为你的 Angular 应用创建一个新的 组件。

你将学到什么

你的应用有了一个新组件：Home。

Angular 组件的概念预览

Angular 应用围绕组件构建，组件是 Angular 的构建块。 组件包含代码、HTML 布局和 CSS 样式信息，这些信息提供了应用中元素的函数和外观。 在 Angular 中，组件可以包含其他组件。应用的功能和外观可以被划分和分割成组件。

在 Angular 中，组件具有定义其属性的元数据。 创建 Home 组件时，你会用到这些属性：

  • selector：描述 Angular 在模板中如何引用该组件。
  • standalone：描述该组件是否需要 NgModule。
  • imports：描述组件的依赖项。
  • template：描述组件的 HTML 标记和布局。
  • styleUrls：以数组形式列出组件使用的 CSS 文件的 URL。

  1. 创建 Home 组件

    在此步骤中，你将为你的应用创建一个新组件。

    在 IDE 的终端窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 运行此命令来创建一个新的 Home 组件

      ng generate component home

    3. 运行此命令来构建和运行你的应用。

      注意：此步骤仅适用于你的本地环境！

      ng serve

    4. 打开浏览器并导航到 http://localhost:4200 来查找该应用。

    5. 确认应用构建没有错误。

      有帮助：它应该像上一课一样渲染，因为尽管你添加了一个新组件，但你还没有在应用的任何模板中包含它。

    6. 在完成后续步骤时，请保持 ng serve 运行。

  2. 将新组件添加到你的应用布局中

    在此步骤中，你将新组件 Home 添加到应用的根组件 App 中，以便它显示在应用的布局中。

    在 IDE 的编辑窗格中：

    1. 在编辑器中打开 app.ts。

    2. 在 app.ts 中，通过将此行添加到文件级别的导入中来导入 Home 组件。

      在 src/app/app.ts 中导入 Home

      import {Component} from '@angular/core';import {Home} from './home/home';@Component({  selector: 'app-root',  imports: [Home],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

    3. 在 app.ts 的 @Component 中，更新 imports 数组属性并添加 Home。

      替换 src/app/app.ts 中的内容

      import {Component} from '@angular/core';import {Home} from './home/home';@Component({  selector: 'app-root',  imports: [Home],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

    4. 在 app.ts 的 @Component 中，更新 template 属性以包含以下 HTML 代码。

      替换 src/app/app.ts 中的内容

      import {Component} from '@angular/core';import {Home} from './home/home';@Component({  selector: 'app-root',  imports: [Home],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.css'],})export class App {  title = 'homes';}

    5. 保存对 app.ts 的更改。

    6. 如果 ng serve 正在运行，应用应该会更新。 如果 ng serve 未运行，请重新启动它。 应用中的Hello world应该会变成来自Home组件的home works!。

    7. 在浏览器中检查正在运行的应用，并确认应用已更新。

  3. 为 Home 组件添加功能

    在此步骤中，你将为 Home 组件添加功能。

    在上一步中，你将默认的 Home 组件添加到应用的模板中，使其默认 HTML 出现在应用中。 在此步骤中，你将添加一个搜索过滤器和按钮，它们将在后续课程中使用。 目前，Home 组件只有这些。 请注意，此步骤只是将搜索元素添加到布局中，目前还没有任何功能。

    在 IDE 的编辑窗格中：

    1. 在 first-app 目录中，在编辑器中打开 home.ts。

    2. 在 home.ts 的 @Component 中，使用此代码更新 template 属性。

      替换 src/app/home/home.ts 中的内容

      import {Component} from '@angular/core';@Component({  selector: 'app-home',  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>  `,  styleUrls: ['./home.css'],})export class Home {}

    3. 接下来，在编辑器中打开 home.css 并使用这些样式更新内容。

      注意：在浏览器中，这些可以放在 src/app/home/home.ts 的 styles 数组中。

      替换 src/app/home/home.css 中的内容

      .results {  display: grid;  column-gap: 14px;  row-gap: 14px;  grid-template-columns: repeat(auto-fill, minmax(400px, 400px));  margin-top: 50px;  justify-content: space-around;}input[type="text"] {  border: solid 1px var(--primary-color);  padding: 10px;  border-radius: 8px;  margin-right: 4px;  display: inline-block;  width: 30%;}button {  padding: 10px;  border: solid 1px var(--primary-color);  background: var(--primary-color);  color: white;  border-radius: 8px;}@media (min-width: 500px) and (max-width: 768px) {  .results {      grid-template-columns: repeat(2, 1fr);  }  input[type="text"] {      width: 70%;  }   }@media (max-width: 499px) {  .results {      grid-template-columns: 1fr;  }    }

    4. 确认应用构建没有错误。 你应该能在应用中找到过滤器查询框和按钮，并且它们应该被样式化。 在继续下一步之前，请纠正任何错误。

总结：在本课中，你为你的应用创建了一个新组件，并为其添加了一个过滤器编辑控件和按钮。
