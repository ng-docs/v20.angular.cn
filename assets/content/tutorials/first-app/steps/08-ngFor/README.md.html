在组件中使用 @for 列出对象

edit

本教程演示了如何在 Angular 模板中使用 @for 块来动态显示模板中重复的数据。

注意：此视频反映的是旧的语法，但主要概念仍然有效。

你将学到什么

  • 你将向此应用添加一个数据集
  • 你的应用将使用 @for 显示来自新数据集的元素列表

@for 的概念预览

在 Angular 中，@for 是一种特定的控制流块，用于在模板中动态重复数据。在纯 JavaScript 中，你会使用 for 循环 - @for 为 Angular 模板提供了类似的功能。

你可以利用 @for 来迭代数组甚至异步值。在本课中，你将添加一个新数据数组进行迭代。

有关更深入的解释，请参阅控制流指南。
