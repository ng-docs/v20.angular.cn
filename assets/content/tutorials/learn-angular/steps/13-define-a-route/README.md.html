定义路由

edit

既然你已经建立好应用来使用 Angular 路由，现在你需要定义路由。

注意：在深入指南中了解定义基本路由的更多信息。

在本活动中，你将学习如何在你的应用中添加和配置路由。

  1. 在 app.routes.ts 中定义路由

    在你的应用中，有两个页面要显示：（1）首页和（2）用户页面。

    要定义路由，请将一个路由对象添加到 app.routes.ts 中的 routes 数组，其中包含：

    • 路由的 path（它会自动从根路径（即 /）开始）
    • 你希望路由显示的 component
    import {Routes} from '@angular/router';import {Home} from './home/home';export const routes: Routes = [  {    path: '',    component: Home,  },];

    上面的代码是如何将 Home 添加为路由的示例。现在请继续在 playground 中实现此功能以及 User。

    将 User 的路径设置为 'user'。

  2. 向路由定义添加标题

    除了正确定义路由之外，Angular 路由还允许你通过向每个路由添加 title 属性，在用户导航时设置页面标题。

    在 app.routes.ts 中，将 title 属性添加到默认路由 (path: '') 和 user 路由。 这是一个例子：

    import {Routes} from '@angular/router';import {Home} from './home/home';export const routes: Routes = [  {    path: '',    title: 'App Home Page',    component: Home,  },];
