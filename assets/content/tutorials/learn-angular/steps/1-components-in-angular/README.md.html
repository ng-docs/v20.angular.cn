Angular 中的组件

edit

组件是任何 Angular 应用的基础构建块。每个组件都由三个部分组成：

  • TypeScript 类
  • HTML 模板
  • CSS 样式

注意：在基础指南中了解更多关于组件的信息。

在本活动中，你将学习如何更新组件的模板和样式。

这是一个开始使用 Angular 的绝佳机会。

  1. 更新组件模板

    更新 template 属性，使其读取 Hello Universe

    template: `  Hello Universe`,

    当你更改 HTML 模板后，预览会使用你的消息进行更新。 让我们更进一步：更改文本的颜色。

  2. 更新组件样式

    更新 styles 值并将 color 属性从 blue 更改为 #a144eb。

    styles: `  :host {    color: #a144eb;  }`,

    当你查看预览时，你会发现文本颜色将被更改。

在 Angular 中，你可以使用所有浏览器支持的 CSS 和 HTML。 如果你愿意，可以将模板和样式存储在单独的文件中。
