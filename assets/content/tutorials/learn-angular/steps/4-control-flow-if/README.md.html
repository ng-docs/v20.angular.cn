组件中的控制流 - @if

edit

决定在屏幕上为用户显示什么是应用开发中的常见任务。很多时候，这个决定是通过编程方式使用条件来做出的。

为了在模板中表达条件显示，Angular 使用 @if 模板语法。

注意：在基础指南中了解更多关于控制流的信息。

在本活动中，你将学习如何在模板中使用条件。

在模板中启用元素条件显示的语法是 @if。

下面是如何在组件中使用 @if 语法的示例：

@Component({  ...  template: `    @if (isLoggedIn) {      <p>Welcome back, Friend!</p>    }  `,})export class App {  isLoggedIn = true;}

有两件事需要注意：

  • if 有一个 @ 前缀，因为它是一种特殊的语法，称为 Angular 模板语法
  • 对于使用 v16 及更早版本的应用，请参阅 关于 NgIf 的 Angular 文档 以获取更多信息。

  1. 创建一个名为 isServerRunning 的属性

    在 App 类中，添加一个名为 isServerRunning 的 boolean 属性，并将初始值设置为 true。

  2. 在模板中使用 @if

    更新模板以显示消息 Yes, the server is running （如果 isServerRunning 的值为 true）。

  3. 在模板中使用 @else

    现在，Angular 支持使用原生模板语法通过 @else 语法定义 else 分支。更新模板以显示消息 No, the server is not running 作为 else 分支。

    这是一个范例：

    template: `  @if (isServerRunning) { ... }  @else { ... }`;

    添加你的代码以填充缺失的标记。

这种类型的功能称为条件控制流。接下来，你将学习如何在模板中重复条目。
