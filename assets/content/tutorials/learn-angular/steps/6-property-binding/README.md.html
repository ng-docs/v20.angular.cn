Angular 中的属性（Property）绑定

edit

Angular 中的属性（Property）绑定使你能够为 HTML 元素、Angular 组件等的属性（Property）设置值。

使用属性（Property）绑定来动态地为属性（Property）和 Attribute 设置值。你可以执行诸如切换按钮特性、以编程方式设置图片路径以及在组件之间共享值等操作。

注意：在基础指南中了解有关设置动态属性和特性的更多信息。

在本活动中，你将学习如何在模板中使用属性（Property）绑定。

要绑定到元素的 Attribute，请将 Attribute 名称放在方括号中。这是一个范例：

<img alt="photo" [src]="imageURL">

在本范例中，src Attribute 的值将绑定到类属性（Property） imageURL。无论 imageURL 具有什么值，都将设置为 img 标签的 src Attribute。

  1. 添加一个名为 isEditable 的属性

    通过向 App 类添加一个名为 isEditable 的属性（初始值设置为 true）来更新 app.ts 中的代码。

    export class App {  isEditable = true;}

  2. 绑定到 contentEditable

    接下来，通过使用 [] 语法，将 div 的 contentEditable Attribute 绑定到 isEditable 属性（Property）。

    @Component({  ...  template: `<div [contentEditable]="isEditable"></div>`,})

现在这个 div 是可编辑的了。干得漂亮 👍

属性（Property）绑定是 Angular 众多强大的特性之一。如果你想了解更多信息，请查看Angular 文档。
