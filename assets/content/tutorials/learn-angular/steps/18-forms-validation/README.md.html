验证表单

edit

使用表单时，另一个常见的场景是需要验证输入，以确保提交的数据正确。

注意：在深入指南中了解有关验证表单输入的更多信息。

在本活动中，你将学习如何使用响应式表单验证表单。

  1. 导入 Validators

    Angular 提供了一套验证工具。要使用它们，请首先更新组件以从@angular/forms导入Validators。

    import {ReactiveFormsModule, Validators} from '@angular/forms';@Component({...})export class App {}

  2. 为表单添加验证

    每个FormControl都可以传递你想要用于验证FormControl值的Validators。例如，如果你想让profileForm中的name字段成为必填项，则使用Validators.required。 对于我们 Angular 表单中的email字段，我们希望确保它不为空并且遵循有效的电子邮件地址结构。我们可以通过将Validators.required和Validators.email验证器组合在一个数组中来实现。 更新name和email的FormControl：

    profileForm = new FormGroup({  name: new FormControl('', Validators.required),  email: new FormControl('', [Validators.required, Validators.email]),});

  3. 在模板中检查表单验证

    要确定表单是否有效，FormGroup类有一个valid属性。 你可以使用此属性动态绑定属性。根据表单的有效性更新提交button以启用它。

    <button type="submit" [disabled]="!profileForm.valid">Submit</button>

现在你已经了解了关于验证如何与响应式表单一起工作的基础知识。

出色地学习了在 Angular 中使用表单的这些核心概念。 如果你想了解更多信息，请务必参考 Angular 表单文档。
