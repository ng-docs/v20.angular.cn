组件输出属性

edit

在与组件一起工作时，可能需要通知其他组件发生了某些事情。 也许是按钮被点击了，条目已从列表中添加/删除，或者发生了一些其他重要的更新。 在这种情况下，组件需要与父组件通信。

Angular 使用 output() 函数来实现这种行为。

注意：在输出指南中了解有关自定义事件的更多信息。

在此活动中，您将学习如何使用 output() 函数与组件进行通信。

要创建从子组件到父组件的通信路径，请使用 output 函数来初始化类属性。

child.ts

@Component({...})class Child {  incrementCountEvent = output<number>();}

现在，该组件可以生成可由父组件监听的事件。 通过调用 emit 方法触发事件：

child.ts

class Child {  ...  onClick() {    this.count++;    this.incrementCountEvent.emit(this.count);  }}

emit 函数将生成与 output 定义的类型相同的事件。

好的，轮到你来尝试一下了。 通过完成以下任务来完成代码：

  1. 添加一个 output() 属性

    通过添加一个名为 addItemEvent 的输出属性来更新 child.ts，请确保将输出类型设置为 string。

  2. 完成 addItem 方法

    在 child.ts 中更新 addItem 方法；使用以下代码作为逻辑：

    child.ts

    addItem() {  this.addItemEvent.emit('🐢');}

  3. 更新 App 模板

    在 app.ts 中通过添加以下代码来更新模板以监听发出的事件：

    <app-child (addItemEvent)="addItem($event)" />

    现在，“添加条目”按钮在每次单击时都会向列表中添加一个新条目。
