组合组件

edit

你已经学会了更新组件模板、组件逻辑和组件样式，但是如何在你的应用中使用组件呢？

组件配置的 selector 属性为你提供了一个名称，以便在另一个模板中引用该组件。你可以像使用 HTML 标签一样使用 selector，例如 app-user 在模板中会是 <app-user />。

注意：在基础指南中了解更多关于使用组件的信息。

在本活动中，你将学习如何组合组件。

在这个示例中，有两个组件 User 和 App。

  1. 添加对 User 的引用

    更新 App 模板，以包含对使用选择器 app-user 的 User 的引用。请确保将 User 添加到 App 的 imports 数组中，这样它就可以在 App 模板中使用。

    template: `<app-user />`,imports: [User]

    现在，该组件显示消息 Username: youngTech。你可以更新模板代码以包含更多标记。

  2. 添加更多标记

    因为你可以在模板中使用任何想要的 HTML 标记，所以尝试更新 App 的模板以包含更多 HTML 元素。此示例将在 <app-user> 元素之上添加一个 <section> 元素。

    template: `<section><app-user /></section>`,

你可以使用任意数量的 HTML 标记和组件，以将你的应用想法变为现实。你甚至可以在同一页面上拥有组件的多个副本。
