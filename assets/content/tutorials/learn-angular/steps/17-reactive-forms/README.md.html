响应式表单

edit

当你想以编程方式管理表单，而不是仅仅依赖模板时，响应式表单就是答案。

注意：在深入指南中了解有关响应式表单的更多信息。

在本次活动中，你将学习如何建立响应式表单。

  1. 导入 ReactiveForms 模块

    在app.ts中，从@angular/forms导入ReactiveFormsModule并将其添加到组件的imports数组中。

    import { ReactiveFormsModule } from '@angular/forms';@Component({  selector: 'app-root',  template: `    <form>      <label>Name        <input type="text" />      </label>      <label>Email        <input type="email" />      </label>      <button type="submit">Submit</button>    </form>  `,  imports: [ReactiveFormsModule],})

  2. 使用 FormControl 创建 FormGroup 对象

    响应式表单使用FormControl类来表示表单控件（例如，输入框）。Angular提供了FormGroup类，作为表单控件的集合，形成一个有用的对象，使开发人员能够更方便地处理大型表单。

    将FormControl和FormGroup添加到@angular/forms的导入中，以便您可以为每个表单创建一个FormGroup，并将name和email属性作为FormControls。

    import {ReactiveFormsModule, FormControl, FormGroup } from '@angular/forms';...export class App {  profileForm = new FormGroup({    name: new FormControl(''),    email: new FormControl(''),  });}

  3. 将 FormGroup 和 FormControl 链接到表单

    每个FormGroup都应使用[formGroup]指令附着到表单上。

    此外，每个FormControl都可以使用formControlName指令进行附着并赋值给相应的属性。使用以下表单代码更新模板：

    <form [formGroup]="profileForm">  <label>    Name    <input type="text" formControlName="name" />  </label>  <label>    Email    <input type="email" formControlName="email" />  </label>  <button type="submit">Submit</button></form>

  4. 处理表单更新

    当您想访问FormGroup中的数据时，可以通过访问FormGroup的值来完成。更新template以显示表单值：

    ...<h2>Profile Form</h2><p>Name: {{ profileForm.value.name }}</p><p>Email: {{ profileForm.value.email }}</p>

  5. 访问 FormGroup 值

    向组件类添加一个名为 handleSubmit 的新方法，稍后你将使用它来处理表单提交。 此方法将显示表单中的值，你可以从 FormGroup 访问这些值。

    在组件类中，添加 handleSubmit() 方法来处理表单提交。

    handleSubmit() {  alert(    this.profileForm.value.name + ' | ' + this.profileForm.value.email  );}

  6. 将 ngSubmit 添加到表单

    你已经可以访问表单值了，现在是时候处理提交事件并使用 handleSubmit 方法了。 Angular 为此特定目的提供了一个名为 ngSubmit 的事件处理程序。更新 form 元素，以便在提交表单时调用 handleSubmit 方法。

    <form  [formGroup]="profileForm"  (ngSubmit)="handleSubmit()">

就像这样，你就知道如何在 Angular 中使用响应式表单了。
