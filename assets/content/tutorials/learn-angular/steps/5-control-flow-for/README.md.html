组件中的控制流 - @for

edit

通常在构建 Web 应用时，你需要重复一段代码特定的次数 - 例如，给定一个名称数组，你可能希望在 <p> 标签中显示每个名称。

注意：在基础指南中了解有关控制流的更多信息。

在本活动中，你将学习如何使用 @for 在模板中重复条目。

在模板中启用条目重复的语法是 @for。

这是一个关于如何在组件中使用 @for 语法的范例：

@Component({  ...  template: `    @for (os of operatingSystems; track os.id) {      {{ os.name }}    }  `,})export class App {  operatingSystems = [{id: 'win', name: 'Windows'}, {id: 'osx', name: 'MacOS'}, {id: 'linux', name: 'Linux'}];}

注意看以下两点：

  • for 带有 @ 前缀，因为它是一种称为 Angular 模板语法 的特殊语法
  • 对于使用 v16 及更老版本的应用，请参考 关于 NgFor 的 Angular 文档

  1. 添加 users 属性（Property）

    在 App 类中，添加一个名为 users 的属性，其中包含用户及其姓名。

    [{id: 0, name: 'Sarah'}, {id: 1, name: 'Amy'}, {id: 2, name: 'Rachel'}, {id: 3, name: 'Jessica'}, {id: 4, name: 'Poornima'}]

  2. 更新模板

    更新模板以使用 @for 模板语法在 p 元素中显示每个用户名。

    @for (user of users; track user.id) {  <p>{{ user.name }}</p>}

    注意：必须使用 track，你可以使用 id 或其他唯一标识符。

这种类型的功能称为控制流。接下来，你将学习如何自定义组件以及与组件通信 - 顺便说一句，到目前为止你做得非常棒。
