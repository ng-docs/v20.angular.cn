更新组件类

edit

在 Angular 中，组件的逻辑和行为是在组件的 TypeScript 类中定义的。

注意：在基础指南中了解更多关于显示动态文本的信息。

在本活动中，你将学习如何更新组件类以及如何使用插值。

  1. 添加一个名为 city 的属性

    通过向 App 类添加一个名为 city 的属性来更新组件类。

    export class App {  city = 'San Francisco';}

    city 属性的类型是 string，但由于 TypeScript 中的类型推断，您可以省略该类型。city 属性可以在 App 类中使用，并可以在组件模板中引用。


    要在模板中使用类属性，你必须使用 {{ }} 语法。

  2. 更新组件模板

    更新 template 属性以匹配以下 HTML：

    template: `Hello {{ city }}`,

    这是插值的一个范例，也是 Angular 模板语法的一部分。它使你能够做更多的事情，而不仅仅是在模板中放置动态文本。你还可以使用此语法来调用函数、编写表达式等等。

  3. 更多插值实践

    试试这个 - 添加另一组 {{ }}，内容为 1 + 1：

    template: `Hello {{ city }}, {{ 1 + 1 }}`,

    Angular 会求值 {{ }} 的内容，并在模板中渲染输出。

这仅仅是 Angular 模板可能实现的冰山一角，继续学习以了解更多信息。
