创建一个自定义管道

edit

你可以在 Angular 中创建自定义管道，以满足你的数据转换需求。

注意：在深入指南中了解有关创建自定义管道的更多信息。

在这个活动中，你将创建一个自定义管道并在你的模板中使用它。

管道是一个带有@Pipe装饰器的 TypeScript 类。下面是一个示例：

import {Pipe, PipeTransform} from '@angular/core';@Pipe({  name: 'star',})export class StarPipe implements PipeTransform {  transform(value: string): string {    return `⭐️ ${value} ⭐️`;  }}

StarPipe 接受一个字符串值，并返回在该字符串前后加上星号的字符串。请注意：

  • @Pipe装饰器配置中的名称将在模板中使用
  • transform 函数是你放置逻辑的地方

好了，现在轮到你来尝试一下了——你将创建 ReversePipe：

  1. 创建 ReversePipe

    在reverse.pipe.ts中，将@Pipe装饰器添加到ReversePipe类并提供以下配置：

    @Pipe({  name: 'reverse'})

  2. 实现 transform 函数

    现在 ReversePipe 类是一个管道了。更新 transform 函数以添加反转逻辑：

    export class ReversePipe implements PipeTransform {  transform(value: string): string {    let reverse = '';    for (let i = value.length - 1; i >= 0; i--) {      reverse += value[i];    }    return reverse;  }}

  3. 在模板中使用 ReversePipe

    4. 在实现了管道逻辑之后，最后一步是在模板中使用它。在app.ts中，将管道包含在模板中并将其添加到组件导入中：

    @Component({  ...  template: `Reverse Machine: {{ word | reverse }}`  imports: [ReversePipe]})

这样你就完成了。祝贺你完成此活动。你现在知道如何使用管道，甚至知道如何实现你自己的自定义管道。
