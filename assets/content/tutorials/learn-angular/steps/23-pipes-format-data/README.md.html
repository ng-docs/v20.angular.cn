使用管道格式化数据

edit

你可以通过配置管道来进一步使用它们。可以通过向管道传递选项来配置管道。

注意：在深入指南中了解更多关于使用管道格式化数据的信息。

在本活动中，你将使用一些管道和管道参数。

要向管道传递参数，请使用 : 语法，后跟参数值。这是一个范例：

template: `{{ date | date:'medium' }}`;

输出是 Jun 15, 2015, 9:43:11 PM。

是时候自定义一些管道输出了：

  1. 使用 DecimalPipe 格式化数字

    在app.ts中，更新模板以包含decimal管道的参数。

    template: `  ...  <li>Number with "decimal" {{ num | number:"3.2-2" }}</li>`

    注意：这是什么格式？DecimalPipe的参数称为digitsInfo，此参数使用以下格式：{minIntegerDigits}.{minFractionDigits}-{maxFractionDigits}

  2. 使用 DatePipe 格式化日期

    现在，更新模板以使用 date 管道。

    template: `  ...  <li>Date with "date" {{ birthday | date: 'medium' }}</li>`

    为了增加乐趣，请尝试 date 的一些不同参数。更多信息可以在 Angular 文档 中找到。

  3. 使用 CurrencyPipe 格式化货币

    对于你的最后一个任务，更新模板以使用 currency 管道。

    template: `  ...  <li>Currency with "currency" {{ cost | currency }}</li>`

    你也可以尝试 currency 的不同参数。更多信息可以在 Angular 文档 中找到。

管道用得真棒。到目前为止，你已经取得了很大的进步。

还有更多内置管道可以在你的应用中使用。你可以在 Angular 文档 中找到列表。

如果内置管道不能满足你的需求，你还可以创建一个自定义管道。查看下一课以了解更多信息。
