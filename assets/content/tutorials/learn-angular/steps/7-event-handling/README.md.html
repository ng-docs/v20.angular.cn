事件处理

edit

事件处理让 Web 应用具备了交互特性。它使你作为开发人员能够响应用户的操作，例如按钮按下、表单提交等等。

注意：请在 基础指南 中了解更多关于处理用户交互的信息。

在本次活动中，你将学习如何添加事件处理器。

在 Angular 中，你要使用圆括号语法 () 绑定到事件。在给定的元素上，用圆括号包裹你要绑定的事件，并设置一个事件处理器。考虑一下这个 button 示例：

@Component({  ...  template: `<button (click)="greet()">`})export class App {  greet() {    console.log('Hello, there 👋');  }}

在这个例子中，每次单击按钮时，greet() 函数都会运行。请注意，greet() 语法包括了尾部的圆括号。

好了，轮到你来尝试一下了：

  1. 添加一个事件处理器

    在 App 类中添加 showSecretMessage() 事件处理函数。使用以下代码作为实现：

    showSecretMessage() {  this.message = 'Way to go 🚀';}

  2. 绑定到模板事件

    更新 app.ts 中的模板代码，以绑定到 section 元素的 mouseover 事件。

    <section (mouseover)="showSecretMessage()">

通过代码中的几个步骤，你已经在 Angular 中创建了你的第一个事件处理器。看来你在这方面做得相当不错，请继续保持。
