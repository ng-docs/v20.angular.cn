组件输入属性

edit

有时应用开发需要你将数据发送到组件中。这些数据可以用来定制组件，或者从父组件向子组件发送信息。

Angular 使用一个称为 input 的概念。这类似于其他框架中的 props。要创建 input 属性，请使用 input() 函数。

注意：在输入属性指南中了解更多关于使用输入属性接受数据的信息。

在此活动中，您将学习如何使用 input() 函数将信息发送到组件。

要创建 input 属性，请添加 input() 函数来初始化组件类的属性：

user.ts

class User {  occupation = input<string>();}

当您准备好通过 input 传入值时，可以使用属性语法在模板中设置值。这是一个示例：

app.ts

@Component({  ...  template: `<app-user occupation="Angular Developer"></app-user>`})export class App {}

input 函数返回一个 InputSignal。您可以通过调用该信号来读取其值。

user.ts

@Component({  ...  template: `<p>The user's occupation is {{occupation()}}</p>`})

  1. 定义一个 input() 属性

    更新 user.ts 中的代码，为 User 定义一个名为 name 的 input 属性，并指定 string 类型。目前，不要设置初始值，并调用不带参数的 input()。请务必更新模板，调用并插值显示 name 属性在句子末尾。

  2. 向 input 属性传递值

    更新 app.ts 中的代码，将 name 属性的值设置为 "Simran"。

    当代码成功更新后，应用将显示 The user's name is Simran。
