表单概览

edit

表单是许多应用的重要组成部分，因为它们让你的应用能够接受用户输入。让我们来学习一下 Angular 中是如何处理表单的。

在 Angular 中，有两种类型的表单：模板驱动型和响应式。在接下来的几个活动中，你将学习这两种表单。

注意：在深入指南中了解更多关于Angular 中的表单。

在这个活动中，你将学习如何使用模板驱动型方法建立表单。

  1. 创建一个输入字段

    在 user.ts 中，通过添加一个文本输入框来更新模板，将其 id 设置为 framework，类型设置为 text。

    <label for="framework">  Favorite Framework:  <input id="framework" type="text" /></label>

  2. 导入 FormsModule

    为了让此表单使用支持表单数据绑定的 Angular 功能，你需要导入 FormsModule。

    从 @angular/forms 导入 FormsModule，并将其添加到 User 的 imports 数组中。

    import {Component} from '@angular/core';import {FormsModule} from '@angular/forms';@Component({  ...  imports: [FormsModule],})export class User {}

  3. 为输入框的值添加绑定

    FormsModule 有一个名为 ngModel 的指令，它将输入的值绑定到你类中的一个属性。

    更新输入框以使用 ngModel 指令，特别是使用以下语法 [(ngModel)]="favoriteFramework" 来绑定到 favoriteFramework 属性（Property）。

    <label for="framework">  Favorite Framework:  <input id="framework" type="text" [(ngModel)]="favoriteFramework" /></label>

    在你做出更改后，尝试在输入字段中输入一个值。注意它如何在屏幕上更新（是的，非常酷）。

    注意：[()] 语法被称为“香蕉入盒”，但它代表双向绑定：属性（Property）绑定和事件绑定。在关于双向数据绑定的 Angular 文档 中了解更多信息。

你现在已经朝着使用 Angular 构建表单迈出了重要的第一步。

做得好。让我们保持势头！
