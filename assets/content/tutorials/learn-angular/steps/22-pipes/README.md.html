管道

edit

管道是在模板中转换数据时使用的函数。通常，管道是不会引起副作用的“纯”函数。Angular 有许多有用的内置管道，你可以导入并在组件中使用。你也可以创建自定义管道。

注意：在深入指南中了解有关管道的更多信息。

在本活动中，你将导入一个管道并在模板中使用它。

要在模板中使用管道，请将其包含在插值表达式中。请查看以下范例：

import {UpperCasePipe} from '@angular/common';@Component({  ...  template: `{{ loudMessage | uppercase }}`,  imports: [UpperCasePipe],})export class App {  loudMessage = 'we think you are doing great!'}

现在，轮到你来尝试一下了：

  1. 导入 LowerCase 管道

    首先，通过从@angular/common添加文件级别导入LowerCasePipe来更新app.ts。

    import { LowerCasePipe } from '@angular/common';

  2. 将管道添加到模板导入

    接下来，更新@Component()装饰器的imports以包含对LowerCasePipe的引用

    @Component({  ...  imports: [LowerCasePipe]})

  3. 将管道添加到模板

    最后，在app.ts中更新模板以包含lowercase管道：

    template: `{{username | lowercase }}`

管道也可以接受参数，这些参数可用于配置其输出。在下一个活动中了解更多信息。
