检查组件树

edit

调试你的应用

组件选项卡可让你探索应用的结构。 你可以可视化 DOM 中的组件和指令实例，并检查或修改它们的状态。

探索应用结构

组件树显示了应用中组件和指令的层级关系。

单击组件浏览器中的各个组件或指令以选择它们并预览其属性。 Angular DevTools 在组件树的右侧显示属性和元数据。

要按名称查找组件或指令，请使用组件树上方的搜索框。

导航到宿主节点

要转到特定组件或指令的宿主元素，请在组件浏览器中双击它。 Angular DevTools 将在 Chrome 中打开 Elements 选项卡，或在 Firefox 中打开 Inspector 选项卡，并选择关联的 DOM 节点。

导航到源码

对于组件，Angular DevTools 可让你在 Sources 选项卡（Chrome）和 Debugger 选项卡（Firefox）中导航到组件定义。 选择特定组件后，单击属性视图右上角的图标：

更新属性值

与浏览器的开发者工具类似，属性视图允许你编辑输入、输出或其他属性的值。 右键单击属性值，如果此值类型可用编辑功能，则会出现一个文本输入框。 键入新值并按 Enter 将此值应用于属性。

在控制台中访问选定的组件或指令

作为控制台中的快捷方式，Angular DevTools 提供对最近选定的组件或指令实例的访问。 键入 $ng0 以获取当前选定组件或指令实例的引用，键入 $ng1 获取先前选定的实例，键入 $ng2 获取在此之前的选定实例，依此类推。

选择指令或组件

与浏览器的开发者工具类似，你可以检查页面以选择特定的组件或指令。 单击 Angular DevTools 内左上角的检查元素图标，并将鼠标悬停在页面上的 DOM 元素上。 该扩展程序会识别关联的指令和/或组件，并允许你在组件树中选择相应的元素。

检查你的注入器

注意：Injector Tree 可用于使用 17 或更高版本构建的 Angular 应用。

查看应用的注入器层级

注入器树选项卡可让你探索应用中配置的注入器结构。在这里，你将看到两个树，代表你应用的注入器层级。一个树是你的环境层级，另一个是你的元素层级。

可视化解析路径

当选中特定注入器时，Angular 的依赖注入算法从该注入器遍历到根的路径将被高亮显示。对于元素注入器，这包括高亮显示当依赖项在元素层级中无法解析时，依赖注入算法跳转到的环境注入器。

有关 Angular 如何解析解析路径的更多详细信息，请参阅解析规则。

查看注入器提供者

单击已配置提供者的注入器将在注入器树视图的右侧以列表形式显示这些提供者。在这里你可以查看提供的令牌及其类型。
