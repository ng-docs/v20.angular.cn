CLI 参考

edit
命令 别名 描述
add 为你的项目添加对外部库的支持。
analytics 配置 Angular CLI 使用情况指标的收集。
build b 将 Angular 应用或库编译到给定输出路径的名为 dist/ 的输出目录中。
cache 配置持久磁盘缓存并检索缓存统计信息。
completion 为你的终端建立 Angular CLI 自动补全。
config 检索或设置工作区 angular.json 文件中的 Angular 配置值。
deploy 为指定项目或工作区中的默认项目调用部署构建器。
e2e e 构建并为 Angular 应用启动服务器，然后运行端到端测试。
extract-i18n 从源码中提取 i18n 消息。
generate g 基于原理图生成和/或修改文件。
lint 在给定的项目文件夹中对 Angular 应用代码运行 lint 工具。
new n 创建一个新的 Angular 工作区。
run 使用在你的项目中定义的可选自定义构建器配置来运行 Architect target。
serve s、dev 构建你的应用并为其启动服务器，并在文件更改时重新构建。
test t 在一个项目中运行单元测试。
update 更新你的工作区及其依赖项。请参阅 https://angular.dev/update-guide/。
version v 输出 Angular CLI 版本。
