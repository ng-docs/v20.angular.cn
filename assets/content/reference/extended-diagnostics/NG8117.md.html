函数应在文本插值中调用。

edit

此诊断检测文本插值中未调用的函数。

import {Component} from '@angular/core';@Component({  template: `{{ getValue }}`,})class MyComponent {  getValue() {    return 'value';  }}

有什么问题吗？

文本插值中的函数应被调用以返回值。 如果函数未被调用，它将不返回任何值，插值将无法按预期工作。

我应该怎么做？

确保在文本插值中使用函数时调用它以返回值。

import {Component} from '@angular/core';@Component({  template: `{{ getValue() }}`,})class MyComponent {  getValue() {    return 'value';  }}

配置要求

必须启用 strictTemplates 才能发出任何扩展诊断。 uninvokedFunctionInTextInterpolation 除了 strictTemplates 之外没有其他要求。

如果无法避免该怎么办？

可以通过编辑项目的 tsconfig.json 文件来禁用此诊断：

{  "angularCompilerOptions": {    "extendedDiagnostics": {      "checks": {        "uninvokedFunctionInTextInterpolation": "suppress"      }    }  }}

有关更多信息，请参阅 扩展诊断配置。
