缺少引用目标

edit

Angular 找不到带有 {{ PLACEHOLDER }} 导出名称的指令。 这通常是由于缺少导入或指令上缺少 exportAs 导致的。

提示： 这相当于编译器中常见的运行时错误 NG0301：找不到导出。

调试错误

使用找不到的导出的字符串名称来追踪使用此导出的模板或模块。

确保所有依赖项都已正确导入并在我们的模块中声明。 例如，如果未找到的导出是 ngForm，我们需要导入 FormsModule 并在 *.module.ts 的导入列表中声明它，以解决缺失导出错误。

import { FormsModule } from '@angular/forms';@NgModule({  …  imports: [    FormsModule,    …

如果您最近添加了导入，则需要重启服务器才能看到这些更改。
