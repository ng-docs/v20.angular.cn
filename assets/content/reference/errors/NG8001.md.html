无效元素

edit

在编译期间，一个或多个元素无法被求解，原因是该元素未由 HTML 规范定义，或者没有组件或指令具有这样的元素选择器。

提示： 这相当于编译器中常见的运行时错误 NG0304: '${tagName}' 不是已知元素：...。

调试错误

使用错误信息中的元素名称来查找正在使用该元素的文件。

检查名称和选择器是否正确。

通过检查组件是否出现在 imports 字段中，确保该组件已正确导入到你的 NgModule 或独立组件中。如果该组件是在 NgModule 中声明的（意味着它不是独立的），请通过检查组件是否出现在 exports 字段中，确保它已从中正确导出。

使用自定义元素或 Web 组件时，请确保将 CUSTOM_ELEMENTS_SCHEMA 添加到应用程序模块。

如果这样做未能求解此错误，请检查导入的库，看看你正在使用的导出和属性是否有任何最新更改，并重启你的服务器。
