参数不是字面量

edit

为了加快 Angular 编译器中元数据提取的速度，@NgModule、@Pipe、@Component、@Directive 和 @Injectable 装饰器只接受对象字面量作为参数。

这是 Ivy 中的一项有意变更，它对装饰器的参数要求比 View Engine 更严格。 Ivy 需要这种方法，因为它会通过将表达式移动到类输出中的其他位置来编译装饰器。

调试错误

将所有对象声明移到：

const moduleDefinition = {…}@NgModule(moduleDefinition)export class AppModule {    constructor() {}}

装饰器中：

@NgModule({…})export class AppModule {    constructor() {}}
