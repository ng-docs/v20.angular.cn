edit

引导组件在 @NgModule 装饰器的 bootstrap 属性中定义，或者在独立组件的 bootstrapApplication 的第一个参数中定义。

当 Angular 尝试引导这些组件之一，但在 DOM 中找不到其对应的节点时，会发生此错误。

调试错误

当选择器与标签不匹配时，会发生此问题

@Component({  selector: 'my-app',  ...})export class AppComponent {}
<html>    <app-root></app-root> <!-- Doesn't match the selector --></html>

用正确的标签替换该标签：

<html>    <my-app></my-app> <!-- OK --></html>
