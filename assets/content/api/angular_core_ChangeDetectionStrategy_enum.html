ChangeDetectionStrategy

枚举
stable
代码

默认变更检测器用来检测变更的策略。 当设置后，将在下次变更检测触发时生效。

变更检测用法跳过组件子树

API

    
      enum ChangeDetectionStrategy {}

OnPush

使用 CheckOnce 策略，意味着自动变更检测将被停用，直到通过将策略设置为 Default (CheckAlways) 重新激活。 变更检测仍然可以被显式调用。 此策略适用于所有子指令，并且不能被重写。

默认

使用默认的 CheckAlways 策略，在该策略中，变更检测是自动的，直到被显式停用。
