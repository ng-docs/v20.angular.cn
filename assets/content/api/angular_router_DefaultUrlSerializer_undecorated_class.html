描述

对 UrlSerializer 的默认实现。

URL 范例：

/inbox/33(popup:compose)/inbox/33;open=true/messages/44

DefaultUrlSerializer 使用圆括号来序列化二级分段（例如，popup:compose），使用冒号语法来指定出口，并使用 ';parameter=value' 语法（例如，open=true）来指定路由特定的参数。
