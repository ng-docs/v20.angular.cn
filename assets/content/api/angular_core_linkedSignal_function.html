创建一个可写信号，其值由关联的响应式计算初始化和重置。 这是一个高级 API 形式，其中计算可以访问信号的先前值和计算结果。

注意：此计算是响应式的，这意味着每当计算中使用的任何信号发生更改时，关联的信号都会自动更新。
