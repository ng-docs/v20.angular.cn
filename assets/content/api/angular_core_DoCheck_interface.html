描述

一个生命周期钩子，除了默认变更检测器执行的检查之外，它还为指令调用自定义变更检测函数。

默认的变更检测算法通过跨变更检测运行按引用比较绑定属性值来查找差异。你可以使用此钩子通过其他方式检查和响应更改。

当默认变更检测器检测到更改时，它会调用 ngOnChanges()（如果已提供），而无论你是否执行了额外的变更检测。 通常，你不应同时使用 DoCheck 和 OnChanges 来响应同一输入的更改。

使用须知

以下代码片段展示了一个组件如何实现此接口以调用其自身的变更检测周期。

@Component({        selector: 'my-cmp',        template: `...`,        standalone: false,      })      class MyComponent implements DoCheck {        ngDoCheck() {          // ...        }      }
