I18nSelectPipe

pipe
stable
代码

通用选择器，用于显示与当前值匹配的字符串。

管道用法

{{ value_expression | i18nSelect:mapping }}

API

    
      class I18nSelectPipe implements PipeTransform {}

transform

string
@paramvaluestring | null | undefined

要国际化的字符串。

@parammapping{ [key: string]: string; }

一个对象，指示对于所提供的 value 的不同值应显示的文本。

@returnsstring

描述

通用选择器，用于显示与当前值匹配的字符串。

如果 mapping 的任何键都不与 value 匹配，则当 other 键存在时，返回其内容；否则返回空字符串。

用法说明

范例 

@Component({  selector: 'i18n-select-pipe',  template: `<div>{{ gender | i18nSelect: inviteMap }}</div>`,  standalone: false,})export class I18nSelectPipeComponent {  gender: string = 'male';  inviteMap: any = {'male': 'Invite him.', 'female': 'Invite her.', 'other': 'Invite them.'};}
