The InjectionToken and @Injectable classes for resolvers are deprecated in favor of plain JavaScript functions instead. Dependency injection can still be achieved using the inject function from @angular/core and an injectable class can be used as a functional guard using inject: myResolvedData: () => inject(MyResolver).resolve().
