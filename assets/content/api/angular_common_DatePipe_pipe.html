说明

根据语言环境规则格式化日期值。

DatePipe 仅在检测到输入值的纯更改时执行。 纯更改是指原始输入值（例如 String、Number、Boolean 或 Symbol）的更改， 或对象引用的更改（例如 Date、Array、Function 或 Object）。

请注意，修改 Date 对象不会导致管道再次渲染。 要确保管道被执行，你必须创建一个新的 Date 对象。

只有 en-US 语言环境数据随 Angular 一起提供。要在另一种语言中本地化日期，你必须导入相应的语言环境数据。 有关更多信息，请参阅 I18n 指南。

格式化值的时区可以通过将其作为管道的第二个参数传入，或通过设置 DATE_PIPE_DEFAULT_OPTIONS 令牌来指定默认值。 作为第二个参数传入的值优先于通过令牌定义的值。

用法说明

当输入发生变化时，此管道的结果不会重新求值。为了避免在每次变更检测周期都重新格式化日期，请将日期视为不可变对象，并在管道需要再次运行时更改引用。

格式范例

这些范例将日期转换为各种格式，假设 dateObj 是一个 JavaScript Date 对象，其值为：年份：2015，月份：6，日期：15，小时：21，分钟：43，秒：11，并且给定的时间为 en-US 语言环境的本地时间。

{{ dateObj | date }}               // output is 'Jun 15, 2015'{{ dateObj | date:'medium' }}      // output is 'Jun 15, 2015, 9:43:11 PM'{{ dateObj | date:'shortTime' }}   // output is '9:43 PM'{{ dateObj | date:'mm:ss' }}       // output is '43:11'{{ dateObj | date:"MMM dd, yyyy 'at' hh:mm a" }}  // output is 'Jun 15, 2015 at 09:43 PM'

用法范例

以下组件使用日期管道以不同的格式显示当前日期。

@Component({ selector: 'date-pipe', template: `<div>   <p>Today is {{today | date}}</p>   <p>Or if you prefer, {{today | date:'fullDate'}}</p>   <p>The time is {{today | date:'h:mm a z'}}</p> </div>`})// Get the current date and time as a date-time value.export class DatePipeComponent {  today: number = Date.now();}
