defaultUrlMatcher

UrlMatchResult | null

将路由配置 (route) 与实际 URL (segments) 进行匹配。

当 Route 上未定义匹配器时，这是 Router 默认使用的匹配器。

@paramsegmentsUrlSegment[]

当前导航中剩余的未匹配的段

@paramsegmentGroupUrlSegmentGroup

当前正在匹配的段组

@paramrouteRoute

要匹配的 Route。

@returnsUrlMatchResult | null

说明

将路由配置 (route) 与实际 URL (segments) 进行匹配。

当 Route 上未定义匹配器时，这是 Router 默认使用的匹配器。
