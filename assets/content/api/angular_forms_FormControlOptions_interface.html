      interface FormControlOptions extends AbstractControlOptions {}

nonNullable

boolean | undefined

是否使用用于构造 FormControl 的初始值作为其默认值。 如果此选项为 false 或未提供，则 FormControl 的默认值为 null。 当 FormControl 在没有显式值的情况下被重置时，其值将恢复为其默认值。

initialValueIsDefault

boolean | undefined
@deprecated

请改用 nonNullable。

validators

ValidatorFn | ValidatorFn[] | null | undefined

应用于控件的验证器列表。

asyncValidators

AsyncValidatorFn | AsyncValidatorFn[] | null | undefined

应用于控件的异步验证器列表。

updateOn

"change" | "blur" | "submit" | undefined

控件更新时所依据的事件名称。

描述

提供给 FormControl 的选项接口。

此接口扩展了 AbstractControlOptions 中的所有选项，并增加了一些 FormControl 特有的选项。
