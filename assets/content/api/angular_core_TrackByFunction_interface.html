描述

一个可选的函数，传入 NgForOf 指令，用于自定义 NgForOf 如何唯一标识可迭代对象中的条目。

NgForOf 需要唯一标识可迭代对象中的条目，以便在可迭代对象中的条目重新排序、添加新条目或删除现有条目时，正确执行 DOM 更新。

在所有这些场景中，通常都希望仅更新与受更改影响的条目关联的 DOM 元素。此行为对于以下方面非常重要：

  • 当可迭代对象被修改时，保留任何特定于 DOM 的 UI 状态（如光标位置、焦点、文本选择）
  • 启用条目添加、删除和可迭代对象重新排序的动画
  • 当使用 NgForOf 动态填充嵌套的 <option> 元素并且绑定的可迭代对象被更新时，保留 <select> 元素的值

自定义 trackBy 函数的一个常见用途是当 NgForOf 迭代的模型包含具有唯一标识符的属性（Property）时。例如，给定一个模型：

class User {  id: number;  name: string;  ...}

自定义 trackBy 函数可能如下所示：

function userTrackBy(index, user) {  return user.id;}

自定义 trackBy 函数必须具有以下几个属性：

  • 具有 幂等性（无副作用，并且对于给定的输入始终返回相同的值）
  • 为所有唯一输入返回唯一值
  • 速度快
