用户在拖动可拖动条目时发出的事件。

API

    
      interface CdkDragMove<T = any> {}

source

CdkDrag<T>

正在被拖动的条目。

pointerPosition

{ x: number; y: number; }

用户指针在页面上的位置。

event

MouseEvent | TouchEvent

导致拖动的原生事件。

distance

{ x: number; y: number; }

自拖动序列开始以来，用户拖动的像素距离。

delta

{ x: 0 | 1 | -1; y: 0 | 1 | -1; }

指示用户沿每个轴拖动元素的方向。 1 表示位置在增加（例如，用户向右或向下移动）， 而 -1 表示位置在减小（他们向左或向上移动）。0 表示 位置未改变。
