说明

一个可注入的服务，用于在 Angular Zone 内部或外部执行工作。

此服务最常见的用途是在启动一项工作时优化性能，这项工作由一个或多个异步任务组成，这些任务不需要由 Angular 处理 UI 更新或错误处理。此类任务可以通过 runOutsideAngular 启动，如果需要，这些任务可以通过 run 重新进入 Angular Zone。
