这是一个生命周期钩子，当 Angular 初始化完指令的所有数据绑定属性之后调用。 定义 ngOnInit() 方法以处理任何额外的初始化任务。

ngOnInit

void

这是一个回调方法，在默认变更检测器首次检查完指令的数据绑定属性之后，并且在检查任何视图或内容子项之前立即调用。 它仅在指令实例化时调用一次。
