InitialNavigation

类型别名
stable
代码

在 ExtraOptions 对象中允许的值，用于配置路由器执行初始导航操作的时间。

forRoot

API

    
      type InitialNavigation = 'disabled' | 'enabledBlocking' | 'enabledNonBlocking'

描述

在 ExtraOptions 对象中允许的值，用于配置路由器执行初始导航操作的时间。

  • 'enabledNonBlocking' - （默认值）初始导航在根组件创建之后启动。启动过程不会因初始导航完成而被阻塞。
  • 'enabledBlocking' - 初始导航在根组件创建之前启动。启动过程会被阻塞，直到初始导航完成。如果你使用服务器端渲染，但没有为你的应用启用水合，则应设置此值。
  • 'disabled' - 不执行初始导航。位置监听器在根组件创建之前建立。如果由于某些复杂的初始化逻辑，需要更好地控制路由器何时启动其初始导航，请使用此选项。
