组

AnimationGroupMetadata

定义要并行运行的动画步骤列表。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramstepsAnimationMetadata[]

动画步骤对象数组。

  • 当步骤由 style() 或 animate() 函数调用定义时，组内的每个调用都会立即执行。
  • 要指定稍后应用的偏移样式，请使用 keyframes() 定义步骤，或使用带有延迟值的 animate() 调用。 例如：
group([animate("1s", style({ background: "black" })),animate("2s", style({ color: "white" }))])
@paramoptionsAnimationOptions | null

一个 options 对象，其中包含延迟和开发者定义的参数，这些参数提供样式默认值，并且可以在调用时被重写。

@returnsAnimationGroupMetadata

用法说明

当一系列样式必须在不同的开始时间启动动画并在不同的结束时间结束时，分组动画非常有用。

当在 sequence() 或 transition() 调用中调用时，在所有内部动画步骤完成之前，不会继续到下一个指令。
