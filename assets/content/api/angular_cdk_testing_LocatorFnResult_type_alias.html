  • 如果其中一个查询是针对 ComponentHarnessConstructor<C1> 的，则表示结果可能是 C1 类型的测试工具。

描述

使用特定查询列表进行搜索时获得的结果类型。此类型取决于正在查询的特定条目。

  • 如果其中一个查询是针对 ComponentHarnessConstructor<C1> 的，则表示结果可能是 C1 类型的测试工具。
  • 如果其中一个查询是针对 HarnessPredicate<C2> 的，则表示结果可能是 C2 类型的测试工具。
  • 如果其中一个查询是针对 string 的，则表示结果可能是 TestElement。

由于我们不确定哪个查询会匹配，因此结果类型是所有可能结果类型的联合。

使用说明

示例

类型：

LocatorFnResult<[  ComponentHarnessConstructor<MyHarness>,  HarnessPredicate<MyOtherHarness>,  string]>

等同于：

MyHarness | MyOtherHarness | TestElement
