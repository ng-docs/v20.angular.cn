@angular/platform-browser/animations

BrowserAnimationsModule

NgModule
已弃用自 v20.2 起
代码

导出带有额外的依赖注入提供者的 BrowserModule，以用于动画。请参阅动画。

弃用警告

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

API

    
      class BrowserAnimationsModule {}

withConfig

ModuleWithProviders<BrowserAnimationsModule>

基于指定的对象配置此模块。

@paramconfigBrowserAnimationsModuleConfig

用于配置 BrowserAnimationsModule 行为的对象。

@returnsModuleWithProviders<BrowserAnimationsModule>
用法说明

注册 BrowserAnimationsModule 时，可以使用 withConfig 函数，如下所示：

@NgModule({  imports: [BrowserAnimationsModule.withConfig(config)]})class MyNgModule {}
