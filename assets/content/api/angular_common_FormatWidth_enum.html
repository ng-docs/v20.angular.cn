FormatWidth

枚举
已弃用自 v18.0 起
代码

可用于日期时间格式的字符串宽度。 具体的字符宽度是特定于语言环境的。 给出的示例是针对 en-US 的。

getLocaleDateFormatgetLocaleTimeFormatgetLocaleDateTimeFormat国际化 (i18n) 指南

弃用警告

日期语言环境数据获取器已弃用

API

    
      enum FormatWidth {}

短

对于 en-US，'M/d/yy, h:mm a' （范例：6/15/15, 9:03 AM）

中等

对于 en-US，'MMM d, y, h:mm:ss a' （范例：Jun 15, 2015, 9:03:01 AM）

长

对于 en-US，'MMMM d, y, h:mm:ss a z' （范例：June 15, 2015 at 9:03:01 AM GMT+1）

完整

对于 en-US，'EEEE, MMMM d, y, h:mm:ss a zzzz' （范例：Monday, June 15, 2015 at 9:03:01 AM GMT+01:00）
