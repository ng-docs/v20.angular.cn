function resource<T, R>(options: ResourceOptions<T, R> & { defaultValue: NoInfer<T>; }): ResourceRef<T>;

构建一个 Resource，它将一个响应式请求投影到一个由加载器函数定义的异步操作，该函数通过信号公开加载操作的结果。

请注意，resource 用于读取操作，而不是执行变异的操作。 当 resource 被销毁或新的请求对象可用时，它将通过 AbortSignal 取消进行中的加载，这可能会过早地中止变异。
