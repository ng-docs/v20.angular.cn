描述

一个不可修改的项目列表，当应用程序状态发生变化时，Angular 会保持其更新。

ViewChildren、ContentChildren 和 QueryList 提供的对象类型。

实现了可迭代接口，因此既可以在 ES6 JavaScript 的 for (var i of items) 循环中使用，也可以在 Angular 模板中使用 @for(i of myList; track $index)。

可以通过订阅 changes Observable 来观察更改。 *
