query

AnimationQueryMetadata

在序列中查找当前正在动画的元素内部的一个或多个元素。与 animate() 一起使用。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramselectorstring

要查询的元素，或包含 Angular 特有特征的一组元素，可以使用以下一个或多个标记来指定。

  • query(":enter") 或 query(":leave")：查询新插入/移除的元素（并非所有元素都可以通过这些令牌查询，请参阅 进入和离开元素）
  • query(":animating")：查询所有当前正在动画的元素。
  • query("@triggerName")：查询包含动画触发器的元素。
  • query("@*")：查询所有包含动画触发器的元素。
  • query(":self")：将当前元素包含在动画序列中。
@paramanimationAnimationMetadata | AnimationMetadata[]

要应用于查询到的一个或多个元素的动画步骤。数组被视为动画序列。

@paramoptionsAnimationQueryOptions | null

一个选项对象。使用 'limit' 字段来限制要收集的条目总数。

@returnsAnimationQueryMetadata

用法说明

多个标记

标记可以合并到组合的查询选择器字符串中。例如：

 query(':self, .record:enter, .record:leave,
