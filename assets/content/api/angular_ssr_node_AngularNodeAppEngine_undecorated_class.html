@angular/ssr/node

AngularNodeAppEngine

类

Angular 服务端应用引擎。 管理 Angular 服务端应用（包括本地化的应用），处理渲染请求， 并可以选择在渲染前转换索引 HTML。

API

    
      class AngularNodeAppEngine {}

handle

Promise<Response | null>

通过提供预渲染的内容、执行服务端渲染， 或基于 RenderMode 设置为客户端渲染路由交付静态文件，来处理传入的 HTTP 请求。

此方法将 Node.js 的 IncomingMessage 或 Http2ServerRequest 适配为与 AngularAppEngine 兼容的格式，并将处理逻辑委托给它。

@paramrequestany
  • 传入的 HTTP 请求 (IncomingMessage 或 Http2ServerRequest)。
@paramrequestContextunknown
  • 用于渲染的可选上下文，例如与请求关联的元数据。
@returnsPromise<Response | null>
用法说明

对 https://www.example.com/page/index.html 的请求将服务或渲染与 https://www.example.com/page 相对应的 Angular 路由。

用法说明

这个类应该只实例化一次，并在整个服务端 应用中用作单例，以确保渲染请求和资源管理的一致处理。
