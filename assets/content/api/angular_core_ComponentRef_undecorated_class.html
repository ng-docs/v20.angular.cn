@angular/core

ComponentRef

类
stable
代码

表示由 ComponentFactory 创建的组件。 提供对组件实例及相关对象的访问权限， 并提供销毁实例的方法。

API

    
      abstract class ComponentRef<C> {}

setInput

void

将指定的输入名称更新为新值。使用此方法将正确地标记使用 OnPush 变更检测策略的组件以进行检查。 它还将确保在动态创建的组件进行变更检测时，OnChanges 生命周期钩子会运行。

@paramnamestring

输入的名称。

@paramvalueunknown

输入的新值。

@returnsvoid
