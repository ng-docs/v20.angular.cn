getLocaleNumberFormat

string

检索给定语言环境的数字格式。

数字使用模式格式化，例如 #,###.00。例如，模式 #,###.00 当用于格式化数字 12345.678 时，可能会产生 "12'345,678"。如果你的语言的分组分隔符是撇号，而小数分隔符是逗号，则会发生这种情况。

重要： 字符 . , 0 #（以及下面的其他字符）是特殊的占位符 代表小数点等，它们不是真实字符。 您绝对不能“翻译”占位符。例如，不要将 . 改为 ,，即使在 您的语言中小数点是用逗号书写的。符号应替换为 本地等效字符，使用适合您语言的 NumberSymbol。

以下是数字模式中使用的特殊字符：

符号 含义
. 由用于小数点的字符自动替换。
, 由“分组”（千位）分隔符替换。
0 由数字替换（如果数字不足，则为零）。
# 由数字替换（如果数字不足，则不替换）。
¤ 由货币符号替换，例如 $ 或 USD。
% 标记百分比格式。% 符号可能会更改位置，但必须保留。
E 标记科学计数法格式。E 符号可能会更改位置，但必须保留。
' 用作字面量字符的特殊字符用 ASCII 单引号引起来。
