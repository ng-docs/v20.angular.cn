使用 UpgradeAdapterRef 来控制混合式的 AngularJS / Angular 应用。

自 v5 起已弃用。请改用 upgrade/static，它也支持 预先编译。

注册一个回调函数，它会在混合的 AngularJS / Angular 应用成功启动后被通知。

ready 回调函数是在 Angular Zone 内部调用的，因此不需要调用 $apply()。
