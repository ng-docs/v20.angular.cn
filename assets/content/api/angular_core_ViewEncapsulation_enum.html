ViewEncapsulation

枚举
stable
代码

定义了 Component 装饰器的 encapsulation 选项的 CSS 样式封装策略。

API

    
      enum ViewEncapsulation {}

已模拟

通过给组件的宿主元素添加一个特定的 Attribute，并将相同的 Attribute 应用于通过 styles 或 styleUrls 提供的所有 CSS 选择器，来模拟原生的 Shadow DOM 封装行为。

这是默认选项。

无

不提供任何类型的 CSS 样式封装，这意味着通过 styles 或 styleUrls 提供的所有样式都适用于应用的任何 HTML 元素，而与其宿主组件无关。

ShadowDom

使用浏览器原生的 Shadow DOM API 来封装 CSS 样式，这意味着它会为组件的宿主元素创建一个 ShadowRoot，然后用它来封装组件的所有样式。

描述

定义了 Component 装饰器的 encapsulation 选项的 CSS 样式封装策略。
