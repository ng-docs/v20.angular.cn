MissingTranslationStrategy

枚举
stable
代码

在启动时使用此枚举作为 bootstrapModule 的选项，以定义编译器在缺少翻译时应使用的策略：

  • 错误：如果你有缺失的翻译，则抛出错误。
  • 警告（默认）：在控制台和/或 Shell 中显示警告。
  • 忽略：什么也不做。
