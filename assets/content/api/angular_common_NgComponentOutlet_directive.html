描述

实例化一个 Component 类型，并将其宿主视图插入到当前视图中。 NgComponentOutlet 为动态组件创建提供了一种声明式方法。

NgComponentOutlet 需要一个组件类型，如果设置了假值，视图将清除并且 任何现有组件都将被销毁。

用法说明

微调控制

你可以通过使用以下可选属性来控制组件创建过程：

  • ngComponentOutletInputs：可选的组件输入对象，它将被绑定到组件。

  • ngComponentOutletInjector：可选的自定义 Injector，它将用作组件的父注入器。默认为当前视图容器的注入器。

  • ngComponentOutletEnvironmentInjector：可选的自定义 EnvironmentInjector，它将 提供组件的环境。

  • ngComponentOutletContent：可投影节点的可选列表，以插入到组件的内容部分（如果存在）。

  • ngComponentOutletNgModule：可选的 NgModule 类引用，以允许动态加载另一个模块，然后从该模块加载组件。

  • ngComponentOutletNgModuleFactory：已弃用的配置选项，该选项允许提供可选的 NgModule 工厂，以允许动态加载另一个模块，然后从该模块加载组件。请改用 ngComponentOutletNgModule。

语法

简单

<ng-container *ngComponentOutlet="componentTypeExpression"></ng-container>

带输入

<ng-container *ngComponentOutlet="componentTypeExpression;                                  inputs: inputsExpression;"></ng-container>

自定义注入器/内容

<ng-container *ngComponentOutlet="componentTypeExpression;                                  injector: injectorExpression;                                  content: contentNodesExpression;"></ng-container>

自定义 NgModule 引用

<ng-container *ngComponentOutlet="componentTypeExpression;                                  ngModule: ngModuleClass;"></ng-container>
