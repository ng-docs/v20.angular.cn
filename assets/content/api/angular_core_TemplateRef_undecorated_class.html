ElementRef<any>

此内嵌视图在父视图中的锚点元素。

从此 TemplateRef 创建的嵌入式视图的数据绑定和注入上下文继承自此位置的上下文。

通常，新的内嵌视图会附着到此位置的视图容器，但在高级用例中，视图可以附着到不同的容器，同时保留来自原始位置的数据绑定和注入上下文。

描述

表示一个可用于实例化嵌入式视图的嵌入式模板。 要根据模板实例化嵌入式视图，请使用 ViewContainerRef 的 createEmbeddedView() 方法。

通过将指令放置在 <ng-template> 元素（或以 * 为前缀的指令）上来访问 TemplateRef 实例。 嵌入式视图的 TemplateRef 使用 TemplateRef 令牌注入到指令的构造函数中。

您还可以使用 Query 来查找与组件或指令关联的 TemplateRef。
