animate

AnimationAnimateMetadata

定义动画步骤，将样式信息与时序信息相结合。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramtimingsstring | number

为父动画设置 AnimateTimings。 格式为 "duration [delay] [easing]" 的字符串。

  • 持续时间和延迟表示为一个数字和可选的时间单位，例如 "1s" 或 "10ms" 分别表示一秒和 10 毫秒。 默认单位是毫秒。
  • 缓动值控制动画在其运行时如何加速和减速。值是 ease、ease-in、ease-out、ease-in-out 或 cubic-bezier() 函数调用之一。 如果未提供，则不应用缓动。

例如，字符串 "1s 100ms ease-out" 指定持续时间为 1000 毫秒，延迟为 100 毫秒，以及 "ease-out" 缓动样式， 该样式在持续时间接近结束时减速。

@paramstylesAnimationStyleMetadata | AnimationKeyframesSequenceMetadata | null

为父动画设置 AnimationStyles。 调用 style() 或 keyframes() 函数，该函数返回一组要应用于父动画的 CSS 样式条目。 当为 null 时，使用目标状态的样式。 这在描述将完成动画的动画步骤时很有用； 请参阅 transitions() 中的“动画到最终状态”。

@returnsAnimationAnimateMetadata

用法说明

在动画 sequence()、group() 或 transition() 调用中调用，以指定一个动画步骤， 该步骤在给定时间内将给定的样式数据应用于父动画。

语法范例

时序范例

以下范例展示了各种 timings 规范。

  • animate(500) : 持续时间为 500 毫秒。
  • animate("1s") : 持续时间为 1000 毫秒。
  • animate("100ms 0.5s")：持续时间为 100 毫秒，延迟为 500 毫秒。
  • animate("5s ease-in") : 持续时间为 5000 毫秒，缓动为 ease-in。
  • animate("5s 10ms cubic-bezier(.17,.67,.88,.1)")：持续时间为 5000 毫秒，延迟为 10 毫秒，根据贝塞尔曲线缓动。

样式范例

以下示例调用 style() 来设置单个 CSS 样式。

animate(500, style({ background: "red" }))

以下示例调用 keyframes() 来为连续的关键帧设置不同的 CSS 样式值。

animate(500, keyframes( [  style({ background: "blue" }),  style({ background: "red" }) ])
