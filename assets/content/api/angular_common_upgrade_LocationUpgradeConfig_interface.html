@angular/common/upgrade

LocationUpgradeConfig

interface
stable
代码

LocationUpgrade 的配置选项。

API

    
      interface LocationUpgradeConfig {}

useHash

boolean | undefined

配置位置升级模块是否应该使用 HashLocationStrategy 或 PathLocationStrategy

hashPrefix

string | undefined

配置当使用 HashLocationStrategy 时，URL 中使用的哈希前缀

urlCodec

typeof UrlCodec | undefined

配置用于编码和解码 URL 的 URL 编解码器。默认为 AngularJSCodec

serverBaseHref

string | undefined

配置在服务器端渲染的应用中使用时的 base href

appBaseHref

string | undefined

配置在客户端渲染的应用中使用时的 base href
