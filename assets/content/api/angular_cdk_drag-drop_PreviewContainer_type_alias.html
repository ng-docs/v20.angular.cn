PreviewContainer

类型别名
代码

拖动项的预览可以插入的可能位置。

  • global - 预览将插入到 <body> 的底部。优点是您不必担心 overflow: hidden 或 z-index，但该项将不会保留其继承的样式。
  • parent - 预览将插入到拖动项的父级中。优点是继承的样式将得以保留，但它可能会被 overflow: hidden 裁剪，或由于 z-index 而不可见。此外，预览将对诸如 :nth-child 之类的选择器以及某些 flexbox 配置产生影响。
  • ElementRef<HTMLElement> | HTMLElement - 预览将插入到特定元素中。 与 parent 具有相同的优点和缺点。

API

    
      type PreviewContainer = 'global' | 'parent' | ElementRef<HTMLElement> | HTMLElement
