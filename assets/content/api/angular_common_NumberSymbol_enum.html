可用于替换数字模式中占位符的符号。 范例基于 en-US 值。

getLocaleNumberSymbol国际化 (i18n) 指南

弃用警告

getLocaleNumberSymbol 已弃用

API

    
      enum NumberSymbol {}

小数

小数分隔符。 对于 en-US，为点字符。 范例：2,345.67

分组

分组分隔符，通常用于千位分隔。 对于 en-US，为逗号字符。 范例：2,345.67

列表

列表条目分隔符。 范例："one, two, and three"

PercentSign

百分号（百分之一百）。 范例：23.4%

PlusSign

正数符号。 范例：+23

MinusSign

负数符号。 范例：-23

指数

指数值的计算机表示法（n 乘以 10 的幂）。 范例：1.2E3

SuperscriptingExponent

指数的易读格式。 范例：1.2x103

PerMille

千分号（千分之一千）。 范例：23.4‰

无穷大

无穷大，可以与加号和减号一起使用。 范例：∞、+∞、-∞

NaN

非数字。 范例：NaN

TimeSeparator

时间单位之间使用的符号。 范例：10:52

CurrencyDecimal

货币值的小数分隔符（回退到 Decimal）。 范例：$2,345.67

CurrencyGroup

货币值的分组分隔符（回退到 Group）。 范例：$2,345.67
