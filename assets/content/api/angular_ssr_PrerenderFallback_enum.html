PrerenderFallback

枚举

定义了当预渲染路径不可用时，静态站点生成 (SSG) 路由的回退策略。 这对于带有参数化 URL 的路由尤其重要，因为某些路径可能在构建时未进行预渲染。

ServerRoutePrerenderWithParams

API

    
      enum PrerenderFallback {}

服务器端

如果预渲染路径不可用，则回退到服务器端渲染 (SSR)。 此策略在请求时于服务器上动态生成页面。

客户端

如果预渲染路径不可用，则回退到客户端渲染 (CSR)。 此策略允许在客户端渲染页面。

无

没有回退；如果路径未预渲染，服务器将不会处理该请求。 这意味着对于未预渲染的路径，应用将不会提供任何响应。
