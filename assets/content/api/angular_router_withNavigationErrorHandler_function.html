withNavigationErrorHandler

NavigationErrorHandlerFeature

提供了一个在导航错误发生时被调用的函数。

此函数在应用的注入上下文中运行，因此您可以使用inject函数。

此函数可以返回一个RedirectCommand，将错误转换为重定向，类似于从守卫返回UrlTree或RedirectCommand。这还将阻止Router发出NavigationError；它将改为发出带有代码NavigationCancellationCode.Redirect的NavigationCancel。 除RedirectCommand以外的返回值将被忽略，并且不会改变Router处理错误的方式。

描述

提供了一个在导航错误发生时被调用的函数。

此函数在应用的注入上下文中运行，因此您可以使用inject函数。

此函数可以返回一个RedirectCommand，将错误转换为重定向，类似于从守卫返回UrlTree或RedirectCommand。这还将阻止Router发出NavigationError；它将改为发出带有代码NavigationCancellationCode.Redirect的NavigationCancel。 除RedirectCommand以外的返回值将被忽略，并且不会改变Router处理错误的方式。
