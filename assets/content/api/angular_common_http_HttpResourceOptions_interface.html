实验性v19.2 起

创建 httpResource 的选项。

API

    
      interface HttpResourceOptions<TResult, TRaw> {}

解析

((value: TRaw) => TResult) | undefined

在 HTTP 请求的结果传递给资源之前对其进行转换。

parse 以其原始类型（例如，对于 JSON 数据，为 unknown）接收来自 HTTP 层的值。 它可用于验证或转换资源的类型，并返回更具体的类型。这对于使用运行时模式验证 库（如 Zod）验证后端响应也很有用。

defaultValue

NoInfer<TResult> | undefined

资源在空闲或加载状态时将采用的值。

如果未设置，资源将使用 undefined 作为其默认值。

injector

Injector | undefined

用于创建 httpResource 的 Injector。

如果未提供此项，则将改用当前的 注入上下文（通过 inject）。

equal

ValueEqualityFn<NoInfer<TResult>> | undefined

一个比较函数，用于定义响应值的相等性。
