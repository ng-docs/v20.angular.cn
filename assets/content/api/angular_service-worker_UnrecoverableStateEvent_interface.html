API

    
      interface UnrecoverableStateEvent {}

类型

"UNRECOVERABLE_STATE"

reason

string

说明

当 Service Worker 用来为该客户端提供服务的应用版本处于无法恢复的损坏状态，并且需要完全重新加载页面时，会发出此事件。

例如，Service Worker 可能无法从缓存或服务器检索到必需的资源。如果新版本已部署到服务器，并且浏览器已部分清理了 Service Worker 缓存，从而删除了先前应用版本的一些文件但并非全部文件，则可能会发生这种情况。
