描述

用于在运行时运行 angular 编译器的底层服务，以创建 ComponentFactory，稍后可用于创建和渲染 Component 实例。

每个 @NgModule 都向其注入器提供自己的 Compiler，该编译器将使用该 ng 模块的指令/管道来编译组件。
