provideAnimationsAsync

EnvironmentProviders

返回一组依赖注入提供者，以在应用中启用动画。请参阅动画指南，以了解有关 Angular 动画的更多信息。

当你使用此函数而不是急性的 provideAnimations() 时，动画将不会被渲染，直到渲染器加载完毕。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramtype"animations" | "noop"

传入 'noop' 作为参数以禁用动画。

@returnsEnvironmentProviders

描述

返回一组依赖注入提供者，以在应用中启用动画。请参阅动画指南，以了解有关 Angular 动画的更多信息。

当你使用此函数而不是急性的 provideAnimations() 时，动画将不会被渲染，直到渲染器加载完毕。

用法说明

当你想在使用 bootstrapApplication 函数引导的应用中启用动画时，此函数非常有用。在这种情况下，完全不需要导入 BrowserAnimationsModule NgModule，只需将此函数返回的提供者添加到 providers 列表中，如下所示。

bootstrapApplication(RootComponent, {  providers: [    provideAnimationsAsync()  ]});
