pendingUntilEvent

MonoTypeOperatorFunction<T>

该操作符会使应用变得不稳定，直到可观察对象发出值、完成、报错或被取消订阅。

在那些订阅对于渲染很重要，并且应该包含在 SSR 序列化中的可观察者中使用此操作符。
