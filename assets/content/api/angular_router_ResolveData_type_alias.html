Route#resolve

API

    
      type ResolveData = {  [key: string | symbol]: ResolveFn<unknown> | DeprecatedResolve;}

说明

表示与特定路由关联的已求解数据。

返回一个 RedirectCommand 会指示路由器取消当前导航，并重定向到 RedirectCommand 中提供的位置。 请注意，解析器执行时没有排序保证。 如果多个解析器返回一个 RedirectCommand，则只使用第一个返回的解析器。
