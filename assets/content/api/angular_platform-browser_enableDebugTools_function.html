enableDebugTools

ComponentRef<T>

启用可通过浏览器开发者控制台访问的 Angular 调试工具。

用法：

  1. 打开开发者控制台（例如 Chrome 浏览器中按 Ctrl + Shift + j）
  2. 输入 ng.（通常控制台会显示自动完成建议）
  3. 尝试变更检测分析器 ng.profiler.timeChangeDetection()，然后按 Enter 键。
