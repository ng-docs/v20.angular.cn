value

Signal<T>

当前 Resource 的值，如果资源处于错误状态，则抛出错误。

status

Signal<ResourceStatus>

当前 Resource 的状态，描述了资源当前正在做什么以及可以对其 value 预期什么。

error

Signal<Error | undefined>

处于 error 状态时，此函数返回 Resource 的最后一个已知错误。

isLoading

Signal<boolean>

此资源是否正在加载新值（或重新加载现有值）。

hasValue

this is Resource<Exclude<T, undefined>>

此资源是否具有有效的当前值。

此函数是响应式的。

描述

Resource 是一种异步依赖项（例如，API 调用的结果），它通过信号进行管理和传递。

创建 Resource 的常用方法是通过 resource 函数，但各种其他 API 也可能提供 Resource 实例来描述它们自己的概念。
