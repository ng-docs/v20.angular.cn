捕获 Resource 状态的字符串值。

API

    
      type ResourceStatus = 'idle' | 'error' | 'loading' | 'reloading' | 'resolved' | 'local'

描述

捕获 Resource 状态的字符串值。

可能的状体有：

idle - 该资源没有有效请求，也不会执行任何加载。value() 将为 undefined。

loading - 该资源正在加载新值，这是由于其响应式依赖项发生更改所致。value() 将为 undefined。

reloading - 该资源正在为相同的响应式依赖项重新加载新值。在重新加载操作期间，value() 将继续返回先前获取的值。

error - 加载因错误而失败。value() 将为 undefined。

resolved - 加载已完成，资源具有从加载器返回的值。

local - 该资源的值是通过 .set() 或 .update() 在本地设置的。
