API

    
      type ImportedNgModuleProviders = EnvironmentProviders

说明

通过 importProvidersFrom 函数从 NgModules 导入的提供者。

这些提供者旨在用于应用注入器（或其他环境注入器），不应在组件注入器中使用。

此类型不能直接实现。它由 importProvidersFrom 函数返回，用于防止提取的 NgModule 提供者在错误的上下文中被使用。
