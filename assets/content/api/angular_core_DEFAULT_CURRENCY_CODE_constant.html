@angular/core

DEFAULT_CURRENCY_CODE

常量
stable
代码

提供此令牌以设置你的应用在没有传入货币代码时 CurrencyPipe 使用的默认货币代码。这仅供 CurrencyPipe 使用，与语言环境货币无关。如果未配置，则默认为 USD。

API

    
      const DEFAULT_CURRENCY_CODE: InjectionToken<string>;

描述

提供此令牌以设置你的应用在没有传入货币代码时 CurrencyPipe 使用的默认货币代码。这仅供 CurrencyPipe 使用，与语言环境货币无关。如果未配置，则默认为 USD。

有关更多信息，请参阅 i18n 指南。

默认货币代码目前始终为 USD。

如果需要之前的行为，请通过在应用程序的 NgModule 中创建 DEFAULT_CURRENCY_CODE 提供者来设置它：

{provide: DEFAULT_CURRENCY_CODE, useValue: 'USD'}

使用须知

范例

在独立应用中：

import { LOCALE_ID, ApplicationConfig } from '@angular/core';const appConfig: ApplicationConfig = {  providers: [{provide: DEFAULT_CURRENCY_CODE, useValue: 'EUR' }]};

在基于模块的应用中：

import { platformBrowser } from '@angular/platform-browser';import { AppModule } from './app/app.module';platformBrowser().bootstrapModule(AppModule, {  providers: [{provide: DEFAULT_CURRENCY_CODE, useValue: 'EUR' }]});
