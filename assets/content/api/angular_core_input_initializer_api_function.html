@angular/core

input

初始化器 API
代码

input 函数允许在指令和组件中声明 Angular 输入属性。

可以声明以下两种变体的输入属性（Property）：

  1. 带有初始值的可选输入属性（Property）。
  2. 使用者需要设置的必要输入属性（Property）。

默认情况下，input 函数将声明具有始终存在的初始值的可选输入。可以使用 input.required() 函数声明必需输入。

输入属性是信号。输入属性的值会暴露为一个 Signal。该信号始终保存从父组件绑定过来的输入属性的最新值。

使用须知

要使用基于信号的输入属性，请从 @angular/core 导入 input。

import {input} from '@angular/core';

在组件内部，引入一个新的类成员，并使用对 input 或 input.required 的调用来初始化它。

@Component({  ...})export class UserProfileComponent {  firstName = input<string>();             // Signal<string|undefined>  lastName  = input.required<string>();    // Signal<string>  age       = input(0)                     // Signal<number>}

在你的组件模板内部，你可以通过调用信号来显示输入属性（Property）的值。

<span>{{firstName()}}</span>
