说明

表示错误或失败的响应，可能来自不成功的 HTTP 状态、执行请求时出错或在解析响应期间发生的其他一些故障。

返回到 Observable 响应流中的任何错误都将被包装在 HttpErrorResponse 中，以提供有关发生错误时 HTTP 层的状态的附加上下文。error 属性将包含一个包装后的 Error 对象或从服务器返回的错误响应。
