描述

SecurityContext 标记了一个位置，该位置具有危险的安全隐患，例如，像 innerHTML 这样的 DOM 属性，当处理不当时，可能会导致跨站脚本 (XSS) 安全漏洞。

有关更多详细信息，请参阅 DomSanitizer，关于 Angular 应用中的安全性。
