API

function provideZonelessChangeDetection(): EnvironmentProviders;

provideZonelessChangeDetection

EnvironmentProviders

为使用 bootstrapApplication 引导的应用提供不带 ZoneJS 的变更检测。

此函数允许你配置应用不使用 ZoneJS 的状态/状态更改来调度应用中的变更检测。当页面上根本不存在 ZoneJS 时，或者当它存在是因为其他东西在使用它时（无论是另一个使用 ZoneJS 进行调度的 Angular 应用，还是依赖 ZoneJS 的其他库），此功能都将生效。

也可以将此项添加到 TestBed 的提供者中，以使测试环境更接近生产行为。这将有助于提高组件与无区域变更检测兼容性的信心。

ZoneJS 使用浏览器事件来触发变更检测。使用此提供者时，Angular 将改用 Angular API 来调度变更检测。这些 API 包括：

  • ChangeDetectorRef.markForCheck
  • ComponentRef.setInput
  • 更新模板中读取的信号
  • 绑定宿主或模板监听器被触发时
  • 附着一个被上述任一方式标记为脏的视图
  • 移除视图
  • 注册渲染钩子（仅当渲染钩子执行上述任一操作时，模板才会刷新）
@returnsEnvironmentProviders
