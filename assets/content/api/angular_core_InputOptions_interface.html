      interface InputOptions<T, TransformT> {}

alias

string | undefined

输入的可选公共名称。默认情况下，使用类字段名称。

transform

((v: TransformT) => T) | undefined

可选的转换，每当绑定新值时运行。可用于在更新输入之前转换输入值。

转换函数可以拓宽输入的类型。例如，考虑 disabled 的输入。实际上，作为组件作者，你只想处理布尔值，但如果用户仅使用属性形式通过 <my-dir input> 绑定到输入，他们可能希望绑定字符串。然后，转换可以处理此类字符串值并将其转换为 boolean。请参阅：booleanAttribute。

debugName

string | undefined

输入信号的调试名称。在 Angular DevTools 中用于标识信号。
