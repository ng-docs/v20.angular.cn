animateChild

API

    
      interface AnimateChildOptions extends AnimationOptions {}

duration

string | number | undefined

延迟

string | number | undefined

为启动动画动作设置时间延迟。 一个数字以及可选的时间单位，例如“1 秒”或“10 毫秒”分别表示一秒和十毫秒。默认单位是毫秒。 默认值为 0，表示没有延迟。

参数

{ [name: string]: any; } | undefined

一组由开发者定义的参数，用于在动画动作开始时修改样式和时序。一个键值对数组，其中提供的值被用作默认值。
