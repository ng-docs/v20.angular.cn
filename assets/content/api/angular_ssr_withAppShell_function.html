withAppShell

ServerRenderingFeature<ServerRenderingFeatureKind.AppShell>

配置应用程序的 shell。

应用 shell 是一个最小化的静态 HTML 页面，会立即提供服务，而完整的 Angular 应用程序则在后台加载。通过向用户提供即时反馈，可以提高感知性能。

此函数配置应用 shell 路由，该路由为与任何已定义服务器路由都不匹配的请求提供所提供的组件。

@paramcomponentType<unknown> | (() => Promise<Type<unknown> | DefaultExport<Type<unknown>>>)
  • 用于渲染应用 shell 的 Angular 组件。可以是直接的组件类型，也可以是动态导入函数。
@returnsServerRenderingFeature<ServerRenderingFeatureKind.AppShell>
