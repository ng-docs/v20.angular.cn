boolean | undefined

当组件的模板中存在未知元素时，是否应该抛出 NG0304 运行时错误。默认为 false，此时只会记录错误。如果设置为 true，则会抛出错误。

errorOnUnknownProperties

boolean | undefined

当组件的模板中存在未知属性（Property）时，是否应该抛出错误。默认为 false，此时只会记录错误。如果设置为 true，则会抛出错误。

rethrowApplicationErrors

boolean | undefined

是否应该重新抛出在应用变更检测期间发生的错误。

当为 true 时，在应用变更检测期间捕获的错误将报告给 ErrorHandler 并重新抛出，以防止它们在测试中被忽略。

当为 false 时，错误仅转发到 ErrorHandler，默认情况下，它只会将错误记录到控制台。

默认为 true。
