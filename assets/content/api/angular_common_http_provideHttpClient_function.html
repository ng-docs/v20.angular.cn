provideHttpClient

EnvironmentProviders

配置 Angular 的 HttpClient 服务以供注入。

默认情况下，HttpClient 将使用其默认的 XSRF 保护传出请求的选项进行配置以供注入。可以通过将功能函数传递给 provideHttpClient 来提供额外的配置选项。 例如，可以使用 withInterceptors(...) 功能添加 HTTP 拦截器。

强烈建议为使用 服务器端渲染 (Server-Side Rendering) 的应用启用 fetch 以获得更好的性能和兼容性。要启用 fetch，请在应用的根目录下的 provideHttpClient() 调用中添加 withFetch() 功能：

provideHttpClient(withFetch());
