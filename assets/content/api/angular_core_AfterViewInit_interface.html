代码

一个生命周期钩子，在 Angular 完全初始化完组件的视图之后调用。 定义 ngAfterViewInit() 方法来处理任何额外的初始化任务。

OnInitAfterContentInit生命周期钩子指南

API

    
      interface AfterViewInit {}

ngAfterViewInit

void

当 Angular 完成组件视图的初始化之后，会立即调用此回调方法。 它只在视图被实例化时调用一次。

@returnsvoid

使用须知

以下代码片段展示了一个组件如何实现此接口来定义自己的视图初始化方法。

@Component({        selector: 'my-cmp',        template: `...`,        standalone: false,      })      class MyComponent implements AfterViewInit {        ngAfterViewInit() {          // ...        }      }
