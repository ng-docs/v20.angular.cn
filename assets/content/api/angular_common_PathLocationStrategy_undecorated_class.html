描述

一个用于配置 Location 服务以在浏览器 URL 的 路径中表示其状态的 LocationStrategy。

如果你正在使用 PathLocationStrategy，你可以提供一个 APP_BASE_HREF 或向文档添加一个 <base href> 元素来覆盖默认值。

例如，如果你提供一个 '/my/app/' 的 APP_BASE_HREF 并调用 location.go('/foo')，浏览器的 URL 将变为 example.com/my/app/foo。为了确保所有相对 URI 都解析正确， <base href> 和/或 APP_BASE_HREF 应该以 / 结尾。

类似地，如果你在文档中添加 <base href='/my/app/'/> 并调用 location.go('/foo')，浏览器 URL 将会变成 example.com/my/app/foo。

请注意，在使用 PathLocationStrategy 时，<base href> 中的查询或片段都不会被保留，如 RFC 所述。
