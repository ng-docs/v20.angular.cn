state

AnimationStateMetadata

在一个附着到元素上的触发器中声明一个动画状态。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramnamestring

以逗号分隔的字符串形式表示的已定义状态的一个或多个名称。 以下保留的状态名称可用于为特定用例定义样式：

  • void 你可以将样式与此名称关联，以便在该元素从应用中分离时使用。例如，当 ngIf 求值 结果为 false 时，关联元素的状态为 void。
  • * （星号）表示默认状态。你可以将样式与此名称关联，以便在动画状态未在触发器中声明时用作回退。
@paramstylesAnimationStyleMetadata

一组与此状态关联的 CSS 样式，使用 style() 函数创建。 一旦达到该状态，这组样式就会保留在元素上。

@paramoptions{ params: { [name: string]: any; }; } | undefined

当状态被调用时，可以传递给状态的参数。 0 个或多个键值对。

@returnsAnimationStateMetadata

用法说明

使用 trigger() 函数将状态注册到动画触发器。 使用 transition() 函数在状态之间进行动画处理。 当组件中的某个状态处于活动状态时，其关联的样式会保留在元素上，即使动画结束。
