Angular 平台是 Angular 在网页上的入口点。 每页只有一个平台。在页面上运行的每个 Angular 应用共有的服务（例如反射） 都绑定在其作用域内。 当使用 PlatformBrowser 等平台工厂创建平台时，页面的平台会被隐式初始化，或者通过调用 createPlatform() 函数显式初始化。
