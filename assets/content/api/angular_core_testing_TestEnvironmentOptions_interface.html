teardown

ModuleTeardownOptions | undefined

配置 TestBed 中的测试模块拆卸行为。

errorOnUnknownElements

boolean | undefined

当组件模板中存在未知元素时，是否应该抛出错误。 默认为 false，此时只会记录错误。 如果设置为 true，则会抛出错误。

errorOnUnknownProperties

boolean | undefined

当组件模板中存在未知属性时，是否应该抛出错误。 默认为 false，此时只会记录错误。 如果设置为 true，则会抛出错误。
