@angular/animations

AnimationStateMetadata

interface
已弃用自 v20.2 起
代码

通过将状态名称与一组 CSS 样式相关联来封装动画状态。 由 state() 函数实例化并返回。

弃用警告

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

API

    
      interface AnimationStateMetadata extends AnimationMetadata {}

name

string

状态名称，在组件内是唯一的。

styles

AnimationStyleMetadata

与此状态关联的 CSS 样式。

options

{ params: { [name: string]: any; }; } | undefined

一个选项对象，其中包含开发者定义的参数，这些参数提供样式默认值，并且可以在调用时被重写。

类型

AnimationMetadataType
