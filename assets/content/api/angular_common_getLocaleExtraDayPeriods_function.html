getLocaleExtraDayPeriods

函数
已弃用v18 起
代码

检索特定于语言环境的日时段，这些时段大致指示了一天在不同语言中是如何划分的。 例如，对于 en-US，时段为上午、中午、下午、傍晚和午夜。

getLocaleExtraDayPeriodRules国际化 (i18n) 指南

弃用警告

Angular 建议依赖 Intl API 进行 i18n。 要提取一天中的时段，请改用带有 dayPeriod 选项的 Intl.DateTimeFormat。

API

function getLocaleExtraDayPeriods(
  locale: string,
  formStyle: FormStyle,
  width: TranslationWidth,
): string[];

getLocaleExtraDayPeriods

string[]

检索特定于语言环境的日时段，这些时段大致指示了一天在不同语言中是如何划分的。 例如，对于 en-US，时段为上午、中午、下午、傍晚和午夜。

仅当您加载了完整的语言环境数据后，此功能才可用。 请参阅“I18n 指南”。
