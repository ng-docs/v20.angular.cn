      interface AfterRenderOptions {}

injector

Injector | undefined

创建期间要使用的 Injector。

如果未提供此项，则将改用当前的注入上下文（通过 inject）。

manualCleanup

boolean | undefined

此 Hook 是否应要求手动清理。

如果此值为 false（默认值），则该钩子将自动注册以与当前的 DestroyRef 一起清理。
