用于写入范围值和监听范围输入更改的 ControlValueAccessor。 此值访问器由 FormControlDirective、FormControlName 和 NgModel 指令使用。

导出自

用法说明

将 range 输入与响应式表单一起使用

以下范例展示了如何将 range 输入与响应式表单一起使用。

const ageControl = new FormControl();
<input type="range" [formControl]="ageControl">
