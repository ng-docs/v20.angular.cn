provideRouter

EnvironmentProviders

设置启用应用程序Router功能所需的提供者。 允许配置一组路由以及应启用的额外功能。

@paramroutesRoutes

用于应用程序路由表的Route集合。

@paramfeaturesRouterFeatures[]

用于配置其他路由器行为的可选特性。

@returnsEnvironmentProviders

用法说明

关于如何向你的应用添加 Router 的基本范例：

const appRoutes: Routes = [];bootstrapApplication(AppComponent, {  providers: [provideRouter(appRoutes)]});

你还可以通过添加来自RouterFeatures类型的函数来启用路由器的可选功能：

const appRoutes: Routes = [];bootstrapApplication(AppComponent,  {    providers: [      provideRouter(appRoutes,        withDebugTracing(),        withRouterConfig({paramsInheritanceStrategy: 'always'}))    ]  });
