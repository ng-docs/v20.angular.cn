effect

EffectRef

注册一个“副作用”，每当它读取的信号发生变化时，该副作用将被调度和执行。

Angular 有两种不同类型的副作用：组件副作用和根副作用。组件副作用是在组件、指令或组件/指令的服务中调用 effect() 时创建的。根副作用是在组件树外部调用 effect() 时创建的，例如在根服务中。

这两种副作用类型在时序上有所不同。组件副作用在 Angular 的同步（变更检测）过程中作为组件生命周期事件运行，并且可以安全地读取输入信号或创建/销毁依赖于组件状态的视图。根副作用作为微任务运行，并且与组件树或变更检测无关。

effect() 必须在注入上下文中运行，除非手动指定了 injector 选项。
