代码

返回带有指定 id 的 NgModule 类（使用 @NgModule.id 字段 指定），如果它存在并且已被加载。对于未指定 id 的 NgModule 类，无法检索。如果找不到 NgModule，则会抛出错误。

API

function getNgModuleById<T>(id: string): Type<T>;
