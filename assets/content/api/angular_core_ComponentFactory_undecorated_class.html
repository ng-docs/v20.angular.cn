ComponentFactory

类
deprecated
代码

用于动态创建组件的工厂的基类。 使用 resolveComponentFactory() 为给定的组件类型实例化一个工厂。 使用生成的 ComponentFactory.create() 方法来创建该类型的组件。

弃用警告

Angular 不再需要 Component 工厂。请使用其他可以直接使用 Component 类的地方。
