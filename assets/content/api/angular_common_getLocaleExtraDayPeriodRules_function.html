API

function getLocaleExtraDayPeriodRules(locale: string): (Time | [Time, Time])[];

getLocaleExtraDayPeriodRules

(Time | [Time, Time])[]

检索用于确定在为语言环境定义了多个周期时，要使用哪个日周期的特定于语言环境的规则。

每个定义的日周期都有一条规则。第一条规则应用于第一个日周期，依此类推。 当没有规则可用时，回退到 AM/PM。

规则可以将周期指定为时间范围，或指定为单个时间值。

仅当加载完整的语言环境数据时，此功能才可用。 请参阅“I18n 指南”。
