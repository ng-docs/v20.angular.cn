initTestEnvironment

使用编译器工厂、PlatformRef 和 Angular 模块初始化测试环境。这些是套件中每个测试通用的。

此方法只能调用一次，以在当前平台上为当前测试套件建立通用的提供者。如果确实需要更改提供者，请先使用 resetTestEnvironment。

各个平台的测试模块和平台可从 '@angular//testing' 获得。
