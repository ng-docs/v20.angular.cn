function getOwningComponent<T>(elementOrDir: {} | Element): T | null;

getOwningComponent

T | null

检索视图中包含 DOM 元素的组件实例。

例如，如果 <child-comp> 在 <app-comp> 的模板中使用（即 <app-comp> 的 ViewChild），在 <child-comp> 上调用 getOwningComponent 将返回 <app-comp>。
