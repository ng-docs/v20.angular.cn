`Request` 在 MDN 上

API

    
      const REQUEST: InjectionToken<Request | null>;

描述

表示当前 HTTP 请求对象的注入令牌。

在处理服务端渲染 (SSR) 时，使用此令牌来访问当前请求。

用法说明

在以下场景中，此令牌可能为 null：

  • 在构建过程中。
  • 当应用在浏览器中渲染时（客户端渲染）。
  • 当执行静态站点生成 (SSG) 时。
  • 在开发期间的路由提取过程中（在请求时）。
