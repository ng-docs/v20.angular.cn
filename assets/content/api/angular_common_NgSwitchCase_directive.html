描述

提供一个 switch case 表达式，用于匹配包含的 ngSwitch 表达式。 当表达式匹配时，将渲染给定的 NgSwitchCase 模板。 如果多个匹配表达式匹配 switch 表达式的值，则会显示所有匹配项。

导出自

  • CommonModule

用法说明

在 switch 容器中，*ngSwitchCase 语句将匹配表达式指定为属性。将 *ngSwitchDefault 作为最终 case 包含进来。

<container-element [ngSwitch]="switch_expression">  <some-element *ngSwitchCase="match_expression_1">...</some-element>  ...  <some-element *ngSwitchDefault>...</some-element></container-element>

每个 switch-case 语句都包含一个内联 HTML 模板或模板引用，用于定义如果匹配表达式的值与 switch 表达式的值匹配时要选择的子树。

从 Angular v17 开始，NgSwitch 指令使用严格相等比较 (===) 而不是宽松相等 (==) 来匹配不同的 case。
