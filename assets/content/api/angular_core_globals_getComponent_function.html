getComponent

T | null

检索与给定的 DOM 元素关联的组件实例。

@paramelementElement

要从中检索组件的 DOM 元素。

@returnsT | null

使用须知

给定以下 DOM 结构：

<app-root>  <div>    <child-comp></child-comp>  </div></app-root>

在 <child-comp> 上调用 getComponent 将返回与此 DOM 元素关联的 ChildComponent 实例。

在 <app-root> 上调用此函数将返回 MyApp 实例。
