ComponentMirror

interface
stable
代码

一个描述组件元数据的子集的接口，该子集可以通过 reflectComponentType 函数检索。

API

    
      interface ComponentMirror<C> {}

selector

string

组件的 HTML 选择器。

类型

Type<C>

工厂将要创建的组件类型。

inputs

readonly { readonly propName: string; readonly templateName: string; readonly transform?: ((value: any) => any) | undefined; readonly isSignal: boolean; }[]

组件的输入属性（Property）。

outputs

readonly { readonly propName: string; readonly templateName: string; }[]

组件的输出属性（Property）。

ngContentSelectors

readonly string[]

所有组件中的元素的选择器。

isStandalone

boolean

该组件是否被标记为独立组件。 注意：这是一个额外的标志，在 ComponentFactory 中不存在。
