描述

该服务可以用来获取和设置当前 HTML 文档的标题。

由于 Angular 应用无法在整个 HTML 文档（<html> 标签）上启动，因此无法绑定到代表 <title> 标签的 HTMLTitleElement 元素的 text 属性（Property）。但是，可以使用此服务来设置和获取当前的标题值。
