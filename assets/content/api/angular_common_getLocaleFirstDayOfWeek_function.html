检索给定语言环境的每周的第一天。

国际化 (i18n) 指南

弃用警告

Angular 建议依赖 Intl API 进行国际化。 Intl 的 getWeekInfo 支持不完整（Chromium M99 和 Safari 17）。 您可能需要依赖以下替代方案：

  • 像 Luxon 这样的库依赖于 Intl，但在 getWeekInfo 不受支持时会回退到 ISO 8601 定义（星期一）。
  • 像 date-fns、day.js 或 weekstart 库这样的其他库为每周的第一天提供了自己的基于语言环境的数据。

API

function getLocaleFirstDayOfWeek(locale: string): WeekDay;
