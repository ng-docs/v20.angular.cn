@angular/forms

PatternValidator

指令
stable
代码

一个指令，用于为带有 pattern 属性的控件添加正则表达式模式验证。正则表达式必须匹配整个控件值。 该指令通过 NG_VALIDATORS 多提供者列表提供。

表单验证

API

    
      class PatternValidator extends AbstractValidatorDirective {}

pattern

string | RegExp

跟踪绑定到此指令的 pattern 的更改。

enabled

boolean

确定此验证器是否应基于输入处于活动状态。基类实现检查是否定义了输入（如果该值与 null 和 undefined 不同）。扩展此基类的验证器类可以使用特定于特定验证器指令的逻辑来重写此函数。

用法说明

添加模式验证器

以下示例展示了如何将模式验证器添加到附着到 ngModel 绑定的输入。

<input name="firstName" ngModel pattern="[a-zA-Z ]*">
