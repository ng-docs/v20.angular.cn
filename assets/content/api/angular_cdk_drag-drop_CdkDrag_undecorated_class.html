将用于确定根可拖动元素的选择器，从 cdkDrag 元素开始向上遍历 DOM。传递备用根元素在尝试启用您可能无法访问的元素的拖动时很有用。

将用于确定可拖动元素位置约束的节点或选择器。如果传入字符串，它将用作选择器，从元素的父级开始向上遍历 DOM，直到找到匹配项。

设置位于放置容器外部的 CdkDrag 的位置。 可用于为返回的用户恢复元素的位置。

是否禁用此元素的拖动开始。

disabled

boolean

constrainPosition

DragConstrainPosition | undefined

可用于自定义拖动项在拖动时位置限制逻辑的函数。调用时会传入一个包含用户指针在页面上的当前位置、被拖动项的引用及其尺寸的点。 应返回一个描述项应渲染位置的点。

previewClass

string | string[]

要添加到预览元素中的类。

previewContainer

any

配置项的预览将被插入的位置。可以通过 CDK_DROP_LIST 全局配置。可能的值：

  • global - 预览将插入到 <body> 的底部。优点是您不必担心 overflow: hidden 或 z-index，但该项不会保留其继承的样式。
  • parent - 预览将插入到拖动项的父级中。优点是可以保留继承的样式，但它可能会被 overflow: hidden 裁剪，或者由于 z-index 而不可见。此外，预览将对 :nth-child 等选择器以及某些 flexbox 配置产生影响。
  • ElementRef<HTMLElement> | HTMLElement - 预览将插入到特定元素中。 与 parent 具有相同的优缺点。

scale

number

如果被拖动元素的父级具有 scale 变换，则在用户开始拖动时可能会影响定位。使用此输入通知 CDK 缩放比例。

started

EventEmitter<CdkDragStart<any>>

用户开始拖动项时发出。

released

EventEmitter<CdkDragRelease<any>>

用户释放拖动项（在任何动画开始之前）时发出。

ended

EventEmitter<CdkDragEnd<any>>

用户在容器中停止拖动项时发出。

entered

EventEmitter<CdkDragEnter<any, any>>

用户将项移动到新容器时发出。

exited

EventEmitter<CdkDragExit<any, any>>

用户通过将项拖动到另一个容器中而将其从其容器中移除时发出。

dropped

EventEmitter<CdkDragDrop<any, any, any>>

用户将项放置在容器内时发出。

moved

Observable<CdkDragMove<T>>

在用户拖动项时发出。请谨慎使用，因为此事件会在用户拖动的每一像素处触发。
