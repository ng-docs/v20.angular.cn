Route#runGuardsAndResolvers

API

    
      type RunGuardsAndResolvers = | 'pathParamsChange'  | 'pathParamsOrQueryParamsChange'  | 'paramsChange'  | 'paramsOrQueryParamsChange'  | 'always'  | ((from: ActivatedRouteSnapshot, to: ActivatedRouteSnapshot) => boolean)

描述

在路由上何时运行守卫和求解器的策略。

当路由被激活或停用时，守卫和/或求解器将始终运行。当路由未更改时，默认行为与 paramsChange 相同。

paramsChange：当路径或路径参数更改时，重新运行守卫和求解器。这不包括查询参数。此选项为默认选项。

  • always：每次执行时都运行。
  • pathParamsChange：当路径参数更改时，重新运行守卫和求解器。这不比较矩阵参数或查询参数。
  • paramsOrQueryParamsChange：当路径、矩阵或查询参数更改时运行。
  • pathParamsOrQueryParamsChange：当路径参数更改或查询参数已更改时，重新运行守卫和求解器。这不包括矩阵参数。
