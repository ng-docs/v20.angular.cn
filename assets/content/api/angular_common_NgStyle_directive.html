一个属性型指令，用于更新包含 HTML 元素的样式。 设置一个或多个样式属性（Property），以冒号分隔的键值对形式指定。 键是样式名，带有可选的 .<unit> 后缀 （例如 'top.px'、'font-style.em'）。 值是要进行求值的表达式。 求值结果为非 null 值时，会以给定的单位表示，并赋值给给定的样式属性（Property）。 如果求值结果为 null，则会移除相应的样式。

描述

一个属性型指令，用于更新包含 HTML 元素的样式。 设置一个或多个样式属性（Property），以冒号分隔的键值对形式指定。 键是样式名，带有可选的 .<unit> 后缀 （例如 'top.px'、'font-style.em'）。 值是要进行求值的表达式。 求值结果为非 null 值时，会以给定的单位表示，并赋值给给定的样式属性（Property）。 如果求值结果为 null，则会移除相应的样式。

用法说明

将包含元素的宽度设置为表达式返回的像素值。

<some-element [ngStyle]="{'max-width.px': widthExp}">...</some-element>

使用返回键值对的表达式设置样式值的集合。

<some-element [ngStyle]="objExp">...</some-element>

对于更简单的用例，你可以直接使用样式绑定。 它不需要导入指令。

将包含元素的字体设置为表达式的结果。

<some-element [style]="{'font-style': styleExp}">...</some-element>
