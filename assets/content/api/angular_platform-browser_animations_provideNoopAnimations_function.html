function provideNoopAnimations(): Provider[];

provideNoopAnimations

Provider[]

返回用于在应用中禁用动画的依赖注入提供者集合。请参阅动画指南以了解关于 Angular 动画的更多信息。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@returnsProvider[]

用法说明

当你想使用 bootstrapApplication 函数引导应用，但需要禁用动画时（例如，当运行测试时），此函数很有用。

bootstrapApplication(RootComponent, {  providers: [    provideNoopAnimations()  ]});
