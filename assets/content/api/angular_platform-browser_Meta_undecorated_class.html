描述

一个用于管理 HTML <meta> 标签的服务。

MetaDefinition 对象的属性与 HTML <meta> 标记的属性匹配。这些标记定义了文档元数据，这些元数据对于配置内容安全策略、定义浏览器兼容性和安全设置、设置 HTTP 标头、为社交分享定义丰富内容以及搜索引擎优化 (SEO) 等非常重要。

要在文档中识别特定的 <meta> 标签，请使用格式为 "tag_attribute='value string'" 的属性选择字符串。例如，attrSelector 值为 "name='description'" 的选择器会匹配 name 属性（Attribute）的值为 "description" 的标签。选择器与 querySelector() Document 方法一起使用，格式为 meta[{attrSelector}]。
