restoredState

{ [k: string]: any; navigationId: number; } | null | undefined

先前提供给 pushState 调用的导航状态， 当导航由 popstate 事件触发时。否则为 null。

状态对象由 NavigationExtras 定义，并包含任何 开发者定义的状态值，以及路由器分配给每个路由器转换/导航的唯一 ID。

从路由器的角度来看，路由器永远不会“后退”。 当用户单击浏览器中的后退按钮时， 会创建一个新的导航 ID。

使用此先前状态对象中的 ID 来区分新创建的状态和由 popstate 事件返回的状态，以便您可以恢复某些 记住的状态，例如滚动位置。
