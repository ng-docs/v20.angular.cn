根据语言环境规则格式化日期。

@paramvaluestring | number | Date

要格式化的日期，可以是 Date 对象、数字（自 UTC 纪元以来的毫秒数）或 ISO 日期时间字符串。

@paramformatstring

要包含的日期时间组件。有关详细信息，请参阅 DatePipe。

@paramlocalestring

用于语言环境格式规则的语言环境代码。

@paramtimezonestring | undefined

时区。 与 GMT 的时区偏移量（例如 '+0430'）。 如果未指定，则使用宿主系统设置。

@returnsstring
