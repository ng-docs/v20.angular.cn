检索给定货币代码的货币符号。

国际化 (i18n) 指南

弃用警告

Angular 建议依赖 Intl API 进行 i18n。 你可以使用 Intl.NumberFormat().formatToParts() 来提取货币符号。 例如：Intl.NumberFormat('en', {style:'currency', currency: 'USD'}).formatToParts().find(part => part.type === 'currency').value 在 en 语言环境中为 USD 货币代码返回 $。 注意：US$ 是 en-ca 语言环境的货币符号，但不是 en-us 语言环境的货币符号。

API

function getCurrencySymbol(
  code: string,
  format: 'wide' | 'narrow',
  locale?: string,
): string;

getCurrencySymbol

string

检索给定货币代码的货币符号。

例如，对于默认的 en-US 语言环境，代码 USD 可以用窄符号 $ 或宽符号 US$ 表示。
