描述

扩展此基类以实现自定义渲染。默认情况下，Angular 会将模板渲染到 DOM 中。你可以使用自定义渲染来拦截渲染调用，或者渲染到 DOM 以外的其他内容。

请注意，在访问 DOM 元素时使用 `Renderer2` 不提供额外的安全性，这使其等同于 安全漏洞。

使用 RendererFactory2 创建自定义渲染器。

使用自定义渲染器可以绕过 Angular 的模板，并进行无法以声明方式表达的自定义 UI 更改。例如，如果你需要设置一个名字不是静态已知的属性（Property）或属性（Attribute），请使用 setProperty() 或 setAttribute() 方法。
