描述

将一个 HttpRequest 转换成一个 HttpEvent 流，其中一个事件很可能是 HttpResponse。

HttpHandler 是可注入的。注入时，处理程序实例将请求分派给链中的第一个拦截器，该拦截器再分派给第二个，依此类推，最终到达 HttpBackend。

在 HttpInterceptor 中，HttpHandler 参数是链中的下一个拦截器。
