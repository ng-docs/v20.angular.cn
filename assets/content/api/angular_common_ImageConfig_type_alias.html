@angular/common

ImageConfig

类型别名
stable
代码

用于图像相关选项的配置对象。包含：

  • breakpoints：用于为响应式图像生成 srcset 的整数断点数组。
  • disableImageSizeWarning：一个布尔值。设置为 true 将禁用关于超大图像的控制台警告。
  • disableImageLazyLoadWarning：一个布尔值。设置为 true 将禁用关于配置了 loading="lazy" 的 LCP 图像的控制台警告。 在 NgOptimizedImage 指南中了解有关响应式图像配置的更多信息。 在 相关的错误页面中了解有关图像警告选项的更多信息。

API

    
      type ImageConfig = {  breakpoints?: number[];  placeholderResolution?: number;  disableImageSizeWarning?: boolean;  disableImageLazyLoadWarning?: boolean;}
