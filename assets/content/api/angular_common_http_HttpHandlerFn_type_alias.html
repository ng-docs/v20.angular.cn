      type HttpHandlerFn = (req: HttpRequest<unknown>) => Observable<HttpEvent<unknown>>

描述

表示拦截器链中的下一个拦截器；如果没有更多拦截器，则表示真正的后端。

大多数拦截器将委托给此函数，并在发出的请求或响应到达时对其进行修改。但是，在当前请求的范围内，可以为任意数量的下游请求多次调用此函数。这些下游请求不必与当前请求具有相同的 URL，甚至不必与当前请求具有相同的来源。完全不调用下游处理程序，而在拦截器中完全处理当前请求也是有效的。

此函数应仅在当前正在被拦截的请求范围内调用。一旦该请求完成，就不应再调用此下游处理程序函数。
