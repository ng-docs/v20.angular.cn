stagger

AnimationStaggerMetadata

在动画的 query() 调用中使用，以便在每个查询到的条目被动画化之后发出一个时序间隔。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramtimingsstring | number

延迟值。

@paramanimationAnimationMetadata | AnimationMetadata[]

一个或多个动画步骤。

@returnsAnimationStaggerMetadata

用法说明

在下面的示例中，一个容器元素包装了由 @for 块生成的条目列表。该容器元素包含一个动画触发器，该触发器稍后将被设置为查询每个内部条目。

每次添加条目时，都会运行不透明度淡入动画，并且每个移除的条目都会淡出。 当这些动画中的任何一个发生时，会在每个条目的动画开始后应用交错效果。
