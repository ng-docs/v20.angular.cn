outputFromObservable

OutputRef<T>

声明一个 Angular 输出，它使用 RxJS 可观察者作为事件源，这些事件将被分发给父订阅者。

作为事件源的可观察者的行为定义如下：

  1. 新值会被转发到 Angular 输出（下一个通知）。
  2. 错误通知不会被 Angular 处理。你需要手动处理这些错误。例如，通过使用 catchError。
  3. 完成通知会阻止输出发出新值。

描述

声明一个 Angular 输出，它使用 RxJS 可观察者作为事件源，这些事件将被分发给父订阅者。

作为事件源的可观察者的行为定义如下：

  1. 新值会被转发到 Angular 输出（下一个通知）。
  2. 错误通知不会被 Angular 处理。你需要手动处理这些错误。例如，通过使用 catchError。
  3. 完成通知会阻止输出发出新值。

用法说明

通过声明一个类字段并使用 outputFromObservable() 函数对其进行初始化，来初始化指令中的输出。

@Directive({..})export class MyDir {  nameChange$ = <some-observable>;  nameChange = outputFromObservable(this.nameChange$);}
