@angular/core

ValueProvider

interface
stable
代码

配置 Injector 以返回令牌的值。

API

    
      interface ValueProvider extends ValueSansProvider {}

provide

any

一个注入令牌。通常是 Type 或 InjectionToken 的实例，但也可以是 any。

multi

boolean | undefined

为 true 时，注入器返回一个实例数组。这对于允许多个提供者分散在多个文件中，以便为通用令牌提供配置信息非常有用。

useValue

any

要注入的值。

用法说明

范例 

const injector = Injector.create({providers: [{provide: String, useValue: 'Hello'}]});        expect(injector.get(String)).toEqual('Hello');

多值范例 

const locale = new InjectionToken<string[]>('locale');        const injector = Injector.create({          providers: [            {provide: locale, multi: true, useValue: 'en'},            {provide: locale, multi: true, useValue: 'sk'},          ],        });        const locales: string[] = injector.get(locale);        expect(locales).toEqual(['en', 'sk']);
