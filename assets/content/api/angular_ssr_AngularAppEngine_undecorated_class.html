@angular/ssr

AngularAppEngine

类

Angular 服务端应用引擎。 管理 Angular 服务端应用（包括本地化的应用），处理渲染请求， 并可选择在渲染前转换索引 HTML。

API

    
      class AngularAppEngine {}

handle

Promise<Response | null>

通过提供预渲染内容、执行服务器端渲染或根据 RenderMode 设置为客户端渲染路由提供静态文件来处理传入的 HTTP 请求。

@paramrequestRequest
  • 要处理的 HTTP 请求。
@paramrequestContextunknown
  • 用于渲染的可选上下文，例如与请求关联的元数据。
@returnsPromise<Response | null>
用法说明

对 https://www.example.com/page/index.html 的请求将为 https://www.example.com/page 对应的 Angular 路由启动服务器或进行渲染。

用法说明

此类应实例化一次，并在服务端应用中用作单例，以确保渲染请求和资源管理的一致处理。
