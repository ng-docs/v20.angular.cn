在水合逻辑完成之前，支持重放页面上发生的用户事件（例如 click）。一旦应用完成水合，所有捕获的事件都将被重放，并且相关的事件监听器将被执行。

withEventReplay

HydrationFeature<HydrationFeatureKind.EventReplay>

在水合逻辑完成之前，支持重放页面上发生的用户事件（例如 click）。一旦应用完成水合，所有捕获的事件都将被重放，并且相关的事件监听器将被执行。

@returnsHydrationFeature<HydrationFeatureKind.EventReplay>

用法说明

当使用 bootstrapApplication 函数时，在应用中启用事件重放的基本示例：

bootstrapApplication(AppComponent, {  providers: [provideClientHydration(withEventReplay())]});
