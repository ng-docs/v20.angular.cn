描述

使 Location 服务能够从浏览器的 URL 读取路由状态。 Angular 提供了两种策略：HashLocationStrategy 和 PathLocationStrategy。

应用程序应使用 Router 或 Location 服务来 与应用程序路由状态进行交互。

例如，HashLocationStrategy 生成的 URL 类似 http://example.com/#/foo， 而 PathLocationStrategy 生成 http://example.com/foo 作为等效 URL。

参阅这两个类以了解更多。
