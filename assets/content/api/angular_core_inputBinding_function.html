inputBinding

Binding

创建一个输入绑定。

@parampublicNamestring

要绑定的输入的公共名称。

@paramvalue() => unknown

返回绑定当前值的回调函数。可以是信号（signal）或普通的 getter 函数。

用法示例

在此示例中，我们创建了一个 MyButton 组件的实例，并将 isDisabled 信号的值绑定到其 disabled 输入。

const isDisabled = signal(false);createComponent(MyButton, {bindings: [inputBinding('disabled', isDisabled)]});
