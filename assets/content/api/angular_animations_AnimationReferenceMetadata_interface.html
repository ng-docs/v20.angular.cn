@angular/animations

AnimationReferenceMetadata

interface
已弃用自 v20.2 起
代码

封装一个可复用的动画，它是一系列单独的动画步骤的集合。 由 animation() 函数实例化并返回，并 传递给 useAnimation() 函数。

弃用警告

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

API

    
      interface AnimationReferenceMetadata extends AnimationMetadata {}

animation

AnimationMetadata | AnimationMetadata[]

一个或多个动画步骤对象。

options

AnimationOptions | null

一个 options 对象，其中包含延迟和开发者定义的参数，这些参数提供样式默认值，并且可以在调用时被重写。默认延迟为 0。

类型

AnimationMetadataType
