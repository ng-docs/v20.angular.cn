platformServer

PlatformRef

创建 Angular 平台的服务器端实例。

在执行 Angular 应用的服务器端渲染时，应使用此平台。 可以使用来自 @angular/platform-browser 的 bootstrapApplication 函数在服务器上引导独立应用程序。使用 bootstrapApplication 时，应首先创建 platformServer，并通过 BootstrapContext 传递给引导函数。
