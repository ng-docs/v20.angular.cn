provideEnvironmentInitializer

EnvironmentProviders

此函数用于提供初始化函数，这些函数将在构造环境注入器时执行。

请注意，所提供的初始化器是在注入上下文中运行的。

以前，这是通过使用现已弃用的 ENVIRONMENT_INITIALIZER 令牌实现的。

用法说明

以下示例说明了如何使用 provideEnvironmentInitializer() 配置初始化函数

createEnvironmentInjector(  [    provideEnvironmentInitializer(() => {      console.log('environment initialized');    }),  ],  parentInjector);
