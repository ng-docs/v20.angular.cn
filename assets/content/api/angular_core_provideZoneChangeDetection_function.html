provideZoneChangeDetection

EnvironmentProviders

为使用 bootstrapApplication 引导的应用提供基于 NgZone 的变更检测。

NgZone 默认已在应用中提供。此提供者允许你在 NgZone 中配置诸如 eventCoalescing 之类的选项。 此提供者不适用于 platformBrowser().bootstrapModule，因为它改用 BootstrapOptions。

@paramoptionsNgZoneOptions | undefined
@returnsEnvironmentProviders

描述

为使用 bootstrapApplication 引导的应用提供基于 NgZone 的变更检测。

NgZone 默认已在应用中提供。此提供者允许你在 NgZone 中配置诸如 eventCoalescing 之类的选项。 此提供者不适用于 platformBrowser().bootstrapModule，因为它改用 BootstrapOptions。

用法说明

bootstrapApplication(MyApp, {providers: [  provideZoneChangeDetection({eventCoalescing: true}),]});
