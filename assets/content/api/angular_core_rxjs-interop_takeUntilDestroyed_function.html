takeUntilDestroyed

MonoTypeOperatorFunction<T>

操作符，当调用上下文（组件、指令、服务等）被销毁时，它会完成可观察对象。

@paramdestroyRefDestroyRef | undefined

可选的，代表当前上下文的 DestroyRef。可以在 注入上下文 之外显式传递以使用 takeUntilDestroyed。否则，将注入当前的 DestroyRef。

@returnsMonoTypeOperatorFunction<T>
