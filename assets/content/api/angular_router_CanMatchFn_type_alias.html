      type CanMatchFn = (route: Route, segments: UrlSegment[]) => MaybeAsync<GuardResult>

描述

用作 Route 上 canMatch 守卫的函数的签名。

如果所有守卫都返回 true，导航将继续，Router 将在激活期间使用 Route。如果任何守卫返回 false，则 Route 将在匹配时被跳过，而是处理其他 Route 配置。

以下示例实现并使用了一个 CanMatchFn，该函数检查当前用户是否有权访问团队页面。

const canMatchTeam: CanMatchFn = (route: Route, segments: UrlSegment[]) => {  return inject(PermissionsService).canMatch(inject(UserToken));};bootstrapApplication(App, {  providers: [    provideRouter([      {        path: 'team/:id',        component: TeamComponent,        canMatch: [canMatchTeam],      },    ]),  ],});
