说明

Testability 服务提供了可以从浏览器访问的测试钩子。

通过 @NgModule.bootstrap 字段引导的 Angular 应用（在开发和生产模式下）默认也会实例化可测试性。

对于使用 bootstrapApplication 函数引导的应用，默认情况下不包含 Testability。你可以通过使用 provideProtractorTestingSupport() 函数获取必要的提供者列表并将它们添加到 options.providers 数组中，将其包含到你的应用中。例如：

import {provideProtractorTestingSupport} from '@angular/platform-browser';await bootstrapApplication(RootComponent, providers: [provideProtractorTestingSupport()]);
