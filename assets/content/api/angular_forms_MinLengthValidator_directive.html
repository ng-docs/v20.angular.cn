MinLengthValidator

指令
stable
代码

一个指令，用于向带有 minlength 属性标记的控件添加最小长度验证。该指令通过 NG_VALIDATORS 多提供者列表提供。

表单验证

API

    
      class MinLengthValidator extends AbstractValidatorDirective {}

minlength

string | number | null

追踪绑定到此指令的最小长度的变化。

enabled

boolean

根据输入确定此验证器是否应处于活动状态。 基类实现检查是否已定义输入（如果该值与 null 和 undefined 不同）。扩展此基类的验证器类可以使用特定于特定验证器指令的逻辑来重写此函数。

@paraminputunknown
@returnsboolean

说明

一个指令，用于向带有 minlength 属性标记的控件添加最小长度验证。该指令通过 NG_VALIDATORS 多提供者列表提供。

用法说明

添加最小长度验证器

以下范例展示了如何将最小长度验证器添加到附着在 ngModel 绑定上的输入。

<input name="firstName" ngModel minlength="4">
