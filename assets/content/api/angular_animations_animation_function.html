animation

AnimationReferenceMetadata

生成一个可复用的动画，可以通过调用 useAnimation() 函数在另一个动画或序列中调用。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramstepsAnimationMetadata | AnimationMetadata[]

一个或多个动画对象，由 animate() 或 sequence() 函数返回，它们构成了一个状态到另一个状态的转换。 当你传递一个数组时，默认使用序列。

@paramoptionsAnimationOptions | null

一个选项对象，可以包含动画启动的延迟值，以及其他由开发者定义的参数。 为附加参数提供的值将用作默认值，并且重写值可以在调用时传递给调用者。

@returnsAnimationReferenceMetadata

用法说明

以下范例定义了一个可复用的动画，并提供了一些默认参数值。

var fadeAnimation = animation([  style({ opacity: '{{ start }}' }),  animate('{{ time }}',  style({ opacity: '{{ end }}'}))  ],  { params: { time: '1000ms', start: 0, end: 1 }});

以下调用通过调用 useAnimation() 并传入覆盖参数值来调用已定义的动画。

useAnimation(fadeAnimation, {  params: {    time: '2s',    start: 1,    end: 0  }})

如果此调用中缺少任何传入的参数值，则使用默认值。如果在动画某个步骤之前缺少一个或多个参数值，useAnimation() 将抛出错误。
