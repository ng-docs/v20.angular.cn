@let 允许你定义一个局部变量并在整个模板中复用它。

语法 

@let name = user.name;@let data = data$ | async;

描述

@let 声明类似于 JavaScript 的 let， 它们的值可以是任何有效的 Angular 表达式。每次模板执行时，这些表达式都会被重新求值。

与 JavaScript 变量类似，用 @let 声明的变量可以用于：

  • 在它们被声明之后
  • 在相同作用域和嵌套作用域的范围内
@let user = user$ | async;@if (user) {  <h1>Hello, {{user.name}}</h1>  <user-avatar [photo]="user.photo"/>  <ul>    @for (snack of user.favoriteSnacks; track snack.id) {      <li>{{snack.name}}</li>    }  </ul>  <button (click)="update(user)">Update profile</button>}

语法定义

@let 语法的正式定义为：

  • @let 关键字。
  • 后面跟着一个或多个空格，不包括换行符。
  • 后面跟着一个有效的 JavaScript 名称和零个或多个空格。
  • 后面跟着 = 符号和零个或多个空格。
  • 后面跟着一个 Angular 表达式，它可以是多行的。
  • 以 ; 符号结尾。
