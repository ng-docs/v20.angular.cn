描述

预加载器会乐观地加载所有路由器配置，以加快导航到应用的惰性加载部分的速度。

预加载器在后台运行。当路由器启动时，预加载器开始监听所有导航事件。在每个此类事件之后，预加载器将检查是否有任何配置可以惰性加载。

如果某个路由受到 canLoad 守卫的保护，则预加载器将不会加载它。
