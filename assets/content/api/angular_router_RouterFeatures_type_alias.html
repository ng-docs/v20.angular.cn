RouterFeatures

类型别名
stable
代码

一个类型别名，代表所有可与 provideRouter 一同使用的 Router 特性。 可以通过向 provideRouter 调用添加特殊函数来启用特性。 请参阅每个符号的文档以查找对应的函数名。另请参阅 provideRouter 文档，了解如何使用这些函数。
