用于指定此容器在所属视图中的位置的锚点元素。 每个视图容器只能有一个锚点元素，并且每个锚点元素只能有一个视图容器。

通过在元素上放置一个注入了 ViewContainerRef 的 Directive，或者使用 ViewChild 查询来访问元素的 ViewContainerRef。

描述

表示一个容器，其中一个或多个视图可以附着到一个组件。

可以包含宿主视图（通过使用 createComponent() 方法实例化组件创建），以及嵌入式视图（通过使用 createEmbeddedView() 方法实例化 TemplateRef 创建）。

视图容器实例可以包含其他视图容器，从而创建视图分层结构。

用法说明

下面的示例演示了如何使用 createComponent 函数动态创建 ComponentRef 实例并将其附加到 ApplicationRef，以便将其包含在更改检测周期中。

注意：该范例使用独立组件，但该函数也可用于非独立组件（在 NgModule 中声明）。
