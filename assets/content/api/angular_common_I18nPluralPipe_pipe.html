根据语言环境规则将值映射到对值进行复数化的字符串。

管道用法

{{ value_expression | i18nPlural:pluralMap:locale }}

API

    
      class I18nPluralPipe implements PipeTransform {}

@paramvaluenumber | null | undefined

@parampluralMap{ [count: string]: string; }

一个模仿 ICU 格式的对象，请参阅 https://unicode-org.github.io/icu/userguide/format_parse/messages/。

@paramlocalestring | undefined

一个 string，用于定义要使用的语言环境（默认情况下使用当前的 LOCALE_ID）。

@returnsstring

说明

根据语言环境规则将值映射到对值进行复数化的字符串。
