AnimationOptions

interface
已弃用自 v20.2 起
代码

用于控制动画样式和时序的选项。

弃用警告

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

API

    
      interface AnimationOptions {}

延迟

string | number | undefined

为启动动画动作设置时间延迟。 一个数字以及可选的时间单位，例如 "1s" 或 "10ms" 分别表示一秒和 10 毫秒。默认单位为毫秒。 默认值为 0，表示没有延迟。

参数

{ [name: string]: any; } | undefined

一组由开发人员定义的参数，用于在动画动作开始时修改样式和时序。一个键值对数组，其中提供的值用作默认值。

描述

用于控制动画样式和时序的选项。

以下动画函数接受 AnimationOptions 数据：

  • transition()
  • sequence()
  • group()
  • query()
  • animation()
  • useAnimation()
  • animateChild()

使用 AnimationBuilder 服务构建的程序化动画也 会使用 AnimationOptions。
