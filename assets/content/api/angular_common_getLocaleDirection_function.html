国际化 (i18n) 指南

弃用警告

Angular 建议依赖 Intl API 进行国际化。 对于日期和数字，请让 Intl.DateTimeFormat() 和 Intl.NumberFormat() 来确定书写方向。 Intl 的替代方案 getTextInfo。 仅有部分支持 (Chromium M99 & Safari 17)。 第三方替代方案，如 rtl-detect，可以解决此问题。

function getLocaleDirection(locale: string): 'ltr' | 'rtl';
