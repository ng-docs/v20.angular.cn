shouldProcessUrl

boolean

告知路由器是否应该处理此 URL。

当它返回 true 时，路由器将执行常规导航。 当它返回 false 时，路由器会将路由器状态设置为空状态。 因此，所有活动组件都将被销毁。

@paramurlUrlTree
@returnsboolean
