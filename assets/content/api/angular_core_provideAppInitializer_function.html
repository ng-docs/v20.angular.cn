APP_INITIALIZER

API

function provideAppInitializer(initializerFn: () => any): EnvironmentProviders;

provideAppInitializer

EnvironmentProviders

所提供的函数会在应用启动时被注入，并在应用初始化期间执行。如果该函数返回一个 Promise 或一个 Observable，则初始化不会完成，直到 Promise 被求解或 Observable 完成。

例如，你可以创建一个加载语言数据或外部配置的函数，并使用 provideAppInitializer() 提供该函数。该函数在应用程序引导过程中执行，所需数据在启动时即可获得。

请注意，所提供的初始化器是在注入上下文中运行的。

以前，这是通过现已弃用的 APP_INITIALIZER 令牌实现的。
