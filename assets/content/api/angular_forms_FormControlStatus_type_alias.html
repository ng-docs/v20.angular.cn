FormControlStatus

类型别名
stable
代码

表单可以有几种不同的状态。每种可能的状态都以字符串字面量的形式返回。

API

    
      type FormControlStatus = 'VALID' | 'INVALID' | 'PENDING' | 'DISABLED'

描述

表单可以有几种不同的状态。每种可能的状态都以字符串字面量的形式返回。

  • VALID：报告控件是有效的，意味着输入值中不存在错误。
  • INVALID：报告控件是无效的，意味着输入值中存在错误。
  • PENDING：报告控件处于待定状态，意味着正在进行异步验证，并且输入值的错误尚不可用。
  • DISABLED：报告控件已禁用，意味着该控件免于祖先对有效性或值的计算。
