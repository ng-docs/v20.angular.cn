API

    
      class MaxValidator extends AbstractValidatorDirective {}

max

string | number | null

追踪绑定到本指令的 max 边界的变更。

enabled

boolean

根据输入来确定是否应激活此验证器。 基类实现会检查是否已定义输入（如果该值与 null 和 undefined 不同）。扩展此基类的验证器类可以使用特定于特定验证器指令的逻辑来重写此函数。
