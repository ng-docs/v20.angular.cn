loadTranslations

void

如果执行运行时翻译，则加载翻译以供 $localize 使用。

如果 $localize 标记的字符串不会在编译时被替换，则可以加载一组翻译，这些翻译将在运行时在浏览器中应用于 $localize 标记的字符串。

如果新加载的翻译与之前的翻译具有相同的 MessageId，则会覆盖之前的翻译。

请注意，$localize 消息仅在首次遇到标记的字符串时处理一次，并且不提供在不刷新浏览器的情况下动态更改语言的功能。在应用生命周期后期加载新的翻译不会更改已翻译消息的翻译文本。

消息 ID 和翻译的格式与提取消息时渲染到“简单 JSON”翻译文件中的格式相同。特别是，消息中的占位符使用 {$PLACEHOLDER_NAME} 语法渲染。例如，来自以下模板的消息：

<div i18n>pre<span>inner-pre<b>bold</b>inner-post</span>post</div>

在 translations 映射表中将具有以下形式：

{  "2932901491976224757":     "pre{$START_TAG_SPAN}inner-pre{$START_BOLD_TEXT}bold{$CLOSE_BOLD_TEXT}inner-post{$CLOSE_TAG_SPAN}post"}
@paramtranslationsRecord<string, string>

从消息 ID 到翻译后消息的映射表。

这些消息会根据其 MessageId 进行处理并添加到查找表中。
