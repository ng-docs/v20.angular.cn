描述

一个定义了为 Router 开发组件 outlet 的契约的接口。

出口充当占位符，Angular 会根据当前的路由器状态动态地填充它。

路由器 outlet 应通过 ChildrenOutletContexts#onChildOutletCreated 向 Router 注册自身，并通过 ChildrenOutletContexts#onChildOutletDestroyed 注销自身。当 Router 识别到匹配的 Route 时，它会在 ChildrenOutletContexts 中查找已注册的 outlet 并激活它。
