描述

一个指令，它将 email 验证器添加到带有 email 属性标记的控件上。该指令通过 NG_VALIDATORS 多提供者列表提供。

电子邮件验证基于 WHATWG HTML 规范，并进行了一些增强以纳入更多 RFC 规则。更多信息可以在 Validators.email 页面上找到。
