@angular/common/http

httpResource

初始化器 API
实验性自 v19.2 起
代码

httpResource 发起一个响应式 HTTP 请求，并将请求状态和响应值公开为一个 WritableResource。默认情况下，它假定后端将返回 JSON 数据。要发起一个期望不同类型数据的请求，可以使用 httpResource 的子构造函数，例如 httpResource.text。

API

    
      
    
    
function httpResource<TResult = unknown>(url: () => string | undefined, options: HttpResourceOptions<TResult, unknown> & { defaultValue: NoInfer<TResult>; }): HttpResourceRef<TResult>;

创建一个 Resource，它通过对给定 URL 的 HTTP GET 请求来获取数据。

当 URL 通过信号（signals）发生变化时，该资源将更新。

使用 HttpClient 来发起请求，并支持拦截器、测试以及 HttpClient API 的其他特性。默认情况下，数据被解析为 JSON —— 使用 httpResource 的子函数，例如 httpResource.text()，来以不同方式解析响应。
