描述

根据一个数值来添加/移除 DOM 子树。专为复数化而设计。

显示与 switch 表达式值匹配的 DOM 子树；如果匹配失败，则显示与 switch 表达式的复数化类别匹配的 DOM 子树。

要使用此指令，你必须提供一个容器元素，并将 [ngPlural] 属性设置为一个 switch 表达式。带有 [ngPluralCase] 的内部元素将根据其表达式显示：

  • 如果 [ngPluralCase] 被设置为以 = 开头的值，则仅当该值与 switch 表达式完全匹配时才会显示。
  • 否则，该视图将被视为“类别匹配”，并且仅当未找到完全值匹配且该值映射到其为已定义的语言环境的类别时才会显示。

请参阅 http://cldr.unicode.org/index/cldr-spec/plural-rules
