ComponentFactoryResolver

描述

一个简单的注册表，它将 Components 映射到生成的 ComponentFactory 类，这些类可用于创建组件的实例。 使用它来获取给定组件类型的工厂，然后使用工厂的 create() 方法来创建该类型的组件实例。

注意：自 v13 起，通过 ViewContainerRef.createComponent 进行动态组件创建不需要解析组件工厂：可以直接使用组件类。
