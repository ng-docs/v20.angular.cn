@angular/animations

AnimationQueryOptions

interface
已弃用自 v20.2 起
代码

封装动画查询选项。传递给 query() 函数。

弃用警告

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

API

    
      interface AnimationQueryOptions extends AnimationOptions {}

optional

boolean | undefined

如果此查询是可选的，则为 True；如果此查询是必要的，则为 false。默认为 false。 如果执行查询时未检索到任何元素，则必要的查询会抛出错误。可选查询则不会。

limit

number | undefined

要从查询返回的最大结果总数。 如果为负数，则结果将从查询列表的末尾向开头方向限制。 默认情况下，结果不受限制。

延迟

string | number | undefined

设置启动动画操作的延迟时间。 一个数字和一个可选的时间单位，例如“1 秒”为“1s”，10 毫秒为“10ms”。默认单位为毫秒。 默认值为 0，表示没有延迟。

参数

{ [name: string]: any; } | undefined

一组由开发人员定义的参数，用于在动画操作开始时修改样式和时序。 一个键值对数组，其中提供的值用作默认值。
