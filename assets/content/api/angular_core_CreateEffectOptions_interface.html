CreateEffectOptions

interface
稳定版v20.0 起
代码

传递给 effect 函数的选项。

API

    
      interface CreateEffectOptions {}

injector

Injector | undefined

创建此 effect 的 Injector。

如果未提供此项，则将使用当前的 注入上下文（通过 inject）来代替。

manualCleanup

boolean | undefined

effect 是否应要求手动清理。

如果此值为 false（默认值），则 effect 会自动向当前的 DestroyRef 注册以进行清理。

如果此值为 true 并且您想在注入上下文之外使用 effect，您仍然需要向 effect 提供一个 Injector。

allowSignalWrites

boolean | undefined
@deprecated

不再需要，默认情况下允许信号写入。

debugName

string | undefined

副作用的调试名称。在 Angular DevTools 中用于标识副作用。
