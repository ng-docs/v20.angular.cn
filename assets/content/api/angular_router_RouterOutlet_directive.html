说明

充当占位符，Angular 会根据当前路由器状态动态填充它。

每个出口都可以有一个唯一的名称，由可选的 name 属性确定。 名称不能动态设置或更改。如果未设置，则默认值为“primary”。

<router-outlet></router-outlet><router-outlet name='left'></router-outlet><router-outlet name='right'></router-outlet>

命名出口可以是辅助路由的目标。 辅助路由的 Route 对象有一个 outlet 属性来标识目标出口：

{path: <base-path>, component: <component>, outlet: <target_outlet_name>}

使用命名出口和辅助路由，您可以在同一个 RouterLink 指令中定位多个出口。

路由器会跟踪导航树中每个命名出口的独立分支，并在 URL 中生成该树的表示。 辅助路由的 URL 使用以下语法来同时指定主路由和辅助路由：

http://base-path/primary-route-path(outlet-name:route-path)

路由器出口在实例化新组件时发出激活事件，在组件销毁时发出停用事件。 当 RouteReuseStrategy 指示出口重新附加子树时，会发出附加事件；当 RouteReuseStrategy 指示出口分离子树时，会发出分离事件。

<router-outlet  (activate)='onActivate($event)'  (deactivate)='onDeactivate($event)'  (attach)='onAttach($event)'  (detach)='onDetach($event)'></router-outlet>
