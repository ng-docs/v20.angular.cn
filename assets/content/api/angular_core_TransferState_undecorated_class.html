描述

一个键值存储，用于从服务器端的应用传输到客户端的应用。

该 TransferState 可作为可注入的令牌使用。 在客户端，只需使用 DI 注入此令牌并使用它，它将惰性初始化。 在服务器上，如果使用了 renderApplication 函数，它已包含在内。否则，导入 ServerTransferStateModule 模块以使 TransferState 可用。

存储中的值使用 JSON.stringify/JSON.parse 进行序列化/反序列化。因此，只有布尔值、数字、字符串、null 和非类对象会以无损的方式进行序列化和反序列化。
