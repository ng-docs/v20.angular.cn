描述

一个类可以实现的接口，用于决定子路由是否可以被加载。 如果所有守卫都返回 true，导航将继续。如果任何守卫返回 false，导航将被取消。 如果任何守卫返回一个 UrlTree，则当前导航将被取消，并开始导航到守卫返回的 UrlTree。

下面的示例实现了一个 CanLoad 函数，该函数决定当前用户是否有权限加载请求的子路由。
