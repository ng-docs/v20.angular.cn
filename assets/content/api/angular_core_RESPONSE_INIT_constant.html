描述

用于响应初始化选项的令牌。

使用此令牌可以为配置或初始化服务器端渲染或 API 端点中的 HTTP 响应提供响应选项。

用法说明

在以下情况下，此令牌可能为 null：

  • 在构建过程中。
  • 当应用在浏览器中渲染时（客户端渲染）。
  • 当执行静态站点生成 (SSG) 时。
  • 在开发期间提取路由时（在请求时）。
