跟踪绑定到该指令的 FormArray 的名称。该名称对应于父 FormGroup 或 FormArray 中的一个键。 接受字符串或数字作为名称。 字符串形式的名称对于单个表单很有用，而数字形式允许在 FormArray 中迭代数组时将表单数组绑定到索引。

@returnsboolean
用法说明

例如，对于以下 FormGroup：

form = new FormGroup({  address: new FormGroup({ street: new FormControl() })});

从根表单到“street”控件的路径将是“address”->“street”。

可以通过以下两种格式之一将其提供给此方法：

  1. 字符串控件名称的数组，例如 ['address', 'street']
  2. 一个以句点分隔的控制名称列表，在一个字符串中，例如 'address.street'

如果未提供路径，则此方法检查当前控件上的错误。

描述

将嵌套的 FormArray 同步到 DOM 元素。

此指令旨在与父 FormGroupDirective（选择器：[formGroup]）一起使用。

它接受您要链接的嵌套 FormArray 的字符串名称，并将在您传递给 FormGroupDirective 的父 FormGroup 实例中查找具有该名称注册的 FormArray。
