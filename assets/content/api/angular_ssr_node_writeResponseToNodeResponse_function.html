writeResponseToNodeResponse

Promise<void>

将 Web 标准的 Response 流式传输到 Node.js 的 ServerResponse 或 Http2ServerResponse 中。

此函数调整 web Response 对象以将其内容写入 Node.js 响应对象，同时处理 HTTP/1.1 和 HTTP/2。

@paramsourceResponse
  • 要从中流式传输的 Web 标准 Response 对象。
@paramdestinationany
  • 要流式传输到其中的 Node.js 响应对象（ServerResponse 或 Http2ServerResponse）。
@returnsPromise<void>

说明

将 Web 标准的 Response 流式传输到 Node.js 的 ServerResponse 或 Http2ServerResponse 中。
