sequence

AnimationSequenceMetadata

定义要按顺序逐个运行的动画步骤列表。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramstepsAnimationMetadata[]

动画步骤对象数组。

  • 通过 style() 调用定义的步骤会立即应用样式数据。
  • 通过 animate() 调用定义的步骤会根据指定的时序数据随时间应用样式数据。
sequence([style({ opacity: 0 }),animate("1s", style({ opacity: 1 }))])
@paramoptionsAnimationOptions | null

一个 options 对象，其中包含延迟和开发者定义的参数，这些参数提供样式默认值，并且可以在调用时被重写。

@returnsAnimationSequenceMetadata

用法说明

当你将一个步骤数组传递给 transition() 调用时，这些步骤默认会按顺序运行。 与运行动画步骤的 group() 调用并行运行形成对比。

当序列在 group() 或 transition() 调用中使用时，只有在每个内部动画步骤都完成后，执行才会继续到下一个指令。
