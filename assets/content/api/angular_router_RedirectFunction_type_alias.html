Route#redirectTo

API

    
      type RedirectFunction = (  redirectData: Pick<    ActivatedRouteSnapshot,    'routeConfig' | 'url' | 'params' | 'queryParams' | 'fragment' | 'data' | 'outlet' | 'title'  >,) => MaybeAsync<string | UrlTree>

描述

当路径匹配 Route 配置时，用于处理重定向的函数类型。

RedirectFunction 不能访问完整的ActivatedRouteSnapshot 接口。在路由匹配阶段，某些数据无法准确获知。例如，解析器（resolvers）要到稍后才会运行，因此任何已解析的标题都不会被填充。惰性加载的组件也是如此。对于直到根节点的所有快照也是如此，因此包含父级（root、parent、pathFromRoot）的属性也被排除在外。当然，完整的路由匹配尚未发生，因此 firstChild 和 children 也不可用。
