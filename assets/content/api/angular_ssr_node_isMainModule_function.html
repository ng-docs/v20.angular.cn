isMainModule

函数

确定提供的 URL 是否表示主入口点模块。

API

function isMainModule(url: string): boolean;

isMainModule

boolean

确定提供的 URL 是否表示主入口点模块。

此函数检查提供的 URL 是否对应于直接执行的主 ESM 模块。 它对于有条件地执行仅应在模块是入口点时运行的代码很有用，例如启动服务器或初始化应用。

它执行两个关键检查：

  1. 验证 URL 是否以 'file:' 开头，确保它是一个本地文件。
  2. 将 URL 的已求解文件路径与第一个命令行参数 (process.argv[1]) 进行比较， 该参数指向正在执行的文件。
@paramurlstring

要检查的模块的 URL。这通常应该是 import.meta.url。
