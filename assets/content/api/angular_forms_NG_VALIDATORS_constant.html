代码

一个用于注册附加同步验证器的InjectionToken，这些验证器与AbstractControl一起使用。

NG_ASYNC_VALIDATORS

API

    
      const NG_VALIDATORS: InjectionToken<readonly (Function | Validator)[]>;

用法说明

提供自定义验证器

以下范例注册了一个自定义验证器指令。 要将验证器添加到现有的验证器集合中，需要 multi: true 选项。

@Directive({  selector: '[customValidator]',  providers: [{provide: NG_VALIDATORS, useExisting: forwardRef(() => CustomValidatorDirective), multi: true}]})class CustomValidatorDirective implements Validator {  validate(control: AbstractControl): ValidationErrors | null {    return { 'custom': true };  }}
