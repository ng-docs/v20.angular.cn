弃用警告

请改用 provideRouter 或 RouterModule/RouterModule.forRoot。 此模块以前用于提供一组有用的测试模拟， 最值得注意的是 Location 和 LocationStrategy。 这些通常不再需要，因为 MockPlatformLocation 默认在 TestBed 中提供。 但是，你可以直接使用 provideLocationMocks 来使用它们。
