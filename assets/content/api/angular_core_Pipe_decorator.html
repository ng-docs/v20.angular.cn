将类标记为管道并提供配置元数据的装饰器。

风格指南：管道名称

API

@Pipe({
  name: string;
  pure?: boolean | undefined;
  standalone?: boolean | undefined;
})

描述

将类标记为管道并提供配置元数据的装饰器。

管道类必须实现 PipeTransform 接口。 例如，如果名称是 "myPipe"，请使用如下的模板绑定表达式：

{{ exp | myPipe }}

表达式的结果会传递给管道的 transform() 方法。

管道必须属于一个 NgModule 才能在模板中可用。 要使其成为 NgModule 的成员，请将其列在 NgModule 元数据的 declarations 字段中。
