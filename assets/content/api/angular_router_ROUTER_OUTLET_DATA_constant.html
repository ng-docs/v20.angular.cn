@angular/router

ROUTER_OUTLET_DATA

常量
stable
代码

由 RouterOutlet 提供的 InjectionToken，可以使用 routerOutletData 输入进行设置。

API

    
      const ROUTER_OUTLET_DATA: InjectionToken<Signal<unknown>>;

说明

由 RouterOutlet 提供的 InjectionToken，可以使用 routerOutletData 输入进行设置。

如果未设置，则此值默认为 null。

用法说明

要从带有 router-outlet 的组件模板中设置数据，请执行以下操作：

<router-outlet [routerOutletData]="{name: 'Angular'}" />

要在已路由的组件中读取数据，请执行以下操作：

data = inject(ROUTER_OUTLET_DATA) as Signal<{name: string}>;
