@switch 控制块的灵感来源于 JavaScript 的 switch 语句：

语法 

@switch (condition) {  @case (caseA) {    Case A.  }  @case (caseB) {    Case B.  }  @default {    Default case.  }}

描述

@switch 控制块显示的内容是由与条件表达式匹配的 case 语句选择的。条件表达式的值会使用 === 操作符与 case 表达式进行比较。

@default 控制块是可选的，可以省略。如果没有 @case 匹配表达式，并且没有 @default 控制块，则不会显示任何内容。

@switch 没有 fallthrough 行为，因此你不需要等效于 break 或 return 语句的写法。
