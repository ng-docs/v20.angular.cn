useAnimation

AnimationAnimateRefMetadata

启动一个可复用的动画，该动画是使用 animation() 函数创建的。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramanimationAnimationReferenceMetadata

要启动的可复用动画。

@paramoptionsAnimationOptions | null

一个 options 对象，可以包含动画启动的延迟值，以及开发者定义的参数的额外重写值。

@returnsAnimationAnimateRefMetadata
