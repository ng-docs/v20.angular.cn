代码

用于创建资源的 resource 函数的选项。

API

    
      interface PromiseResourceOptions<T, R> extends BaseResourceOptions<T, R> {}

loader

ResourceLoader<T, R>

加载函数，它为给定的请求返回资源值的 Promise。

stream

undefined

不能同时指定 stream 和 loader。

参数

(() => R) | undefined

一个响应式函数，用于确定要发出的请求。每当请求更改时，将触发加载器以获取资源的新值。

如果未提供 params 函数，则除非重新加载资源，否则加载器不会重新运行。

defaultValue

NoInfer<T> | undefined

当服务器值不可用时（例如，资源仍在加载时）将从资源返回的值。

equal

ValueEqualityFn<T> | undefined

用于比较加载器返回值的相等性函数。

injector

Injector | undefined

重写由 resource 使用的 Injector。
