描述

封装的Provider，仅在创建EnvironmentInjector（例如在NgModule中）时接受。

使用此包装器类型可防止仅为应用程序/环境注入器设计的提供者意外包含在@Component.providers中，并最终进入组件注入器。

此包装器类型可防止访问内部的Provider。
