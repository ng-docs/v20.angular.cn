@angular/common

DATE_PIPE_DEFAULT_OPTIONS

常量
代码

允许为应用程序中的 DatePipe 实例提供默认配置的 DI 令牌。该值是一个对象，可以包含以下字段：

  • dateFormat：配置默认日期格式。如果未提供，DatePipe 将使用 'mediumDate' 作为值。
  • timezone：配置默认时区。如果未提供，DatePipe 将使用最终用户的本地系统时区。
DatePipeConfig

API

    
      const DATE_PIPE_DEFAULT_OPTIONS: InjectionToken<DatePipeConfig>;

用法说明

可以通过提供带有此接口值的令牌来覆盖各种日期管道的默认值。

例如：

通过使用此令牌提供一个值来覆盖默认日期格式：

providers: [  {provide: DATE_PIPE_DEFAULT_OPTIONS, useValue: {dateFormat: 'shortDate'}}]

通过使用此令牌提供一个值来覆盖默认时区：

providers: [  {provide: DATE_PIPE_DEFAULT_OPTIONS, useValue: {timezone: '-1200'}}]
