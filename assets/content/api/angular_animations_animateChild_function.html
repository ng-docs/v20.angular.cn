animateChild

AnimationAnimateChildMetadata

在动画序列中执行查询到的内部动画元素。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramoptionsAnimateChildOptions | null

一个 options 对象，可以包含动画启动的延迟值，以及开发者定义的参数的额外重写值。

@returnsAnimationAnimateChildMetadata

用法说明

每次在 Angular 中触发动画时，父动画都具有优先权，并且任何子动画都会被阻止。为了使子动画运行，父动画必须查询每个包含子动画的元素，并使用此函数运行它们。

请注意，此功能旨在与 query() 结合使用，并且仅适用于使用 Angular 动画库指定的动画。此 API 不处理 CSS 关键帧和过渡。
