使用说明

例如，对于以下 FormGroup：

form = new FormGroup({  address: new FormGroup({ street: new FormControl() })});

从根表单到 'street' 控件的路径将是 'address' -> 'street'。

可以通过以下两种格式之一提供给此方法：

  1. 字符串控件名称的数组，例如 ['address', 'street']
  2. 一个字符串中以句点分隔的控件名称列表，例如 'address.street'

如果未提供路径，则此方法检查当前控件上的错误。

描述

创建 FormGroup 实例并将其绑定到 DOM 元素。

此指令只能用作 NgForm 的子项（在 <form> 标签内）。

使用此指令可将表单的子组与表单的其余部分分开验证，或者如果域模型中的某些值在嵌套对象中一起使用更有意义。

为子组提供一个名称，它将成为表单完整值中子组的键。如果需要直接访问，请使用 ngModelGroup 将指令导出到本地模板变量中（例如：#myGroup="ngModelGroup"）。
