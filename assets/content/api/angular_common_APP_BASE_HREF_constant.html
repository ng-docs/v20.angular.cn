@angular/common

APP_BASE_HREF

常量
stable
代码

一个预定义的 DI 令牌，用于与 PathLocationStrategy 一起使用的基础 href。 基础 href 是在生成和识别 URL 时应保留的 URL 前缀。

API

    
      const APP_BASE_HREF: InjectionToken<string>;

用法说明

以下范例展示了如何使用此令牌配置带有 base href 值的根应用注入器，以便 DI 框架可以在应用中的任何位置提供依赖项。

import {NgModule} from '@angular/core';import {APP_BASE_HREF} from '@angular/common';@NgModule({  providers: [{provide: APP_BASE_HREF, useValue: '/my/app'}]})class AppModule {}
