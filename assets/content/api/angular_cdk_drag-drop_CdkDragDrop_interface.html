@angular/cdk/drag-drop

CdkDragDrop

interface
代码

当用户将一个可拖拽的条目放置在放置容器内时发出的事件。

API

    
      interface CdkDragDrop<T, O = T, I = any> {}

previousIndex

number

拾取条目时的索引。

currentIndex

number

条目的当前索引。

item

CdkDrag<I>

正在被放置的条目。

container

CdkDropList<T>

条目被放置在其中的容器。

previousContainer

CdkDropList<O>

拾取条目时的容器。可以与 container 相同。

isPointerOverContainer

boolean

用户在放置条目时，指针是否位于容器上方。

distance

{ x: number; y: number; }

自拖拽序列开始以来，用户拖拽的像素距离。

dropPoint

{ x: number; y: number; }

放置条目时指针所在的位置

event

MouseEvent | TouchEvent

导致放置事件的原生事件。
