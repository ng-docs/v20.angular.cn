stable
代码

如何在路由器链接中处理查询参数。 以下之一：

  • "merge"：将新参数与当前参数合并。
  • "preserve"：保留当前参数。
  • "replace"：用新参数替换当前参数。这是默认行为。
  • ""：由于历史原因，与 'replace' 相同。

API

    
      type QueryParamsHandling = 'merge' | 'preserve' | 'replace' | ''
