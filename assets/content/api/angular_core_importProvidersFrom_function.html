importProvidersFrom

EnvironmentProviders

从所有 NgModule 和独立组件（包括传递导入的组件）中收集提供者。

通过 importProvidersFrom 提取的提供者只能在应用程序注入器或另一个环境注入器（例如路由注入器）中使用。它们不应用于组件提供者。

关于独立组件的更多信息，请参见本指南。

使用须知

通过 importProvidersFrom 调用的结果可以在 bootstrapApplication 调用中使用：

await bootstrapApplication(RootComponent, {  providers: [    importProvidersFrom(NgModuleOne, NgModuleTwo)  ]});

当使用独立组件时，您还可以将 importProvidersFrom 的结果用于路由的 providers 字段中：

export const ROUTES: Route[] = [  {    path: 'foo',    providers: [      importProvidersFrom(NgModuleOne, NgModuleTwo)    ],    component: YourStandaloneComponent  }];
