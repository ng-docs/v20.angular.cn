verify

void

验证没有未匹配的待处理请求。

如果有任何待处理的请求，则会失败，并显示一条错误消息，指示哪些请求未处理。

如果未设置 ignoreCancelled（默认值），则如果取消的请求未显式匹配，verify() 也会失败。

@paramopts{ ignoreCancelled?: boolean | undefined; } | undefined
@returnsvoid
