formatCurrency

string

使用语言环境规则将数字格式化为货币。

@paramvaluenumber

要格式化的数字。

@paramlocalestring

用于指定语言环境格式规则的语言环境代码。

@paramcurrencystring

包含货币符号或其名称的字符串，例如“$”或“Canadian Dollar”。用于输出字符串，但不影响函数的操作。

@paramcurrencyCodestring | undefined

用于确定小数部分的数字位数的 ISO 4217 货币代码，例如美元的 USD 和欧元的 EUR。

@paramdigitsInfostring | undefined

小数表示选项，格式为字符串：{minIntegerDigits}.{minFractionDigits}-{maxFractionDigits}。有关更多详细信息，请参阅 DecimalPipe。

@returnsstring
