style

AnimationStyleMetadata

声明一个键/值对象，其中包含 CSS 属性/样式，这些属性/样式随后可用于动画 state、动画 sequence 或用于调用 animate() 和 keyframes() 的样式数据。

@deprecated

请改用 animate.enter 或 animate.leave。计划在 v23 中移除

@paramtokens"*" | { [key: string]: string | number; } | ("*" | { [key: string]: string | number; })[]

与动画状态关联的一组 CSS 样式或 HTML 样式。 该值可以是以下任何一种：

  • 将 CSS 属性与值关联的键值样式对。
  • 键值样式对的数组。
  • 星号 (*)，用于使用自动样式，其中样式源自正在动画化的元素，并在动画开始时应用于动画。

自动样式可用于定义依赖于布局或其他环境因素的状态。

@returnsAnimationStyleMetadata

用法说明

以下示例创建动画样式，这些样式收集一组 CSS 属性值：

// string values for CSS propertiesstyle({ background: "red", color: "blue" })// numerical pixel valuesstyle({ width: 100, height: 0 })

以下示例使用自动样式以允许元素从高度 0 动画化到其完整高度：

style({ height: 0 }),animate("1s", style({ height: "*" }))
