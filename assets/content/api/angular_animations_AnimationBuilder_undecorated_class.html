用法说明

要使用此服务，请将其作为依赖项添加到你的组件或指令中。 该服务会随你的组件一起实例化。

应用通常不需要创建自己的动画播放器，但是如果你 需要，请按照以下步骤操作：

  1. 使用 AnimationBuilder.build() 方法 以编程方式创建动画。该方法返回一个 AnimationFactory 实例。

  2. 使用工厂对象创建一个 AnimationPlayer 并将其附着到 DOM 元素。

  3. 使用播放器对象以编程方式控制动画。

例如：

// import the service from BrowserAnimationsModuleimport {AnimationBuilder} from '@angular/animations';// require the service as a dependencyclass MyCmp {  constructor(private _builder: AnimationBuilder) {}  makeAnimation(element: any) {    // first define a reusable animation    const myAnimation = this._builder.build([      style({ width: 0 }),      animate(1000, style({ width: '100px' }))    ]);    // use the returned factory object to create a player    const player = myAnimation.create(element);    player.play();  }}
