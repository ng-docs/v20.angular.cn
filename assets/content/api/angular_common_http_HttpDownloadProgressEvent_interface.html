HttpDownloadProgressEvent

interface
stable
代码

一个下载进度事件。

API

    
      interface HttpDownloadProgressEvent extends HttpProgressEvent {}

类型

HttpEventType.DownloadProgress

partialText

string | undefined

到目前为止已下载的部分响应体。

仅当 responseType 为 text 时才存在。

loaded

number

已上传或下载的字节数。

total

number | undefined

要上传或下载的总字节数。根据请求或响应，这可能无法计算，因此可能不存在。
