控件名称的列表，用于指定如何从当前控件移动到应查询错误的控件。

@returnsboolean
用法说明

例如，对于以下 FormGroup：

form = new FormGroup({  address: new FormGroup({ street: new FormControl() })});

从根表单到“street”控件的路径将是“address”->“street”。

可以通过以下两种格式之一将其提供给此方法：

  1. 字符串控件名称的数组，例如 ['address', 'street']
  2. 一个字符串中以句点分隔的控件名称列表，例如 'address.street'

如果未提供路径，则此方法检查当前控件上是否存在错误。

描述

将独立的 FormControl 实例同步到表单控件元素。

请注意，在 Angular v6 中，已弃用对将 ngModel 输入属性和 ngModelChange 事件与响应式表单指令一起使用的支持，并计划在 Angular 的未来版本中将其移除。
