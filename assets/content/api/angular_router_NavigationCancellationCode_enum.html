NavigationCancellationCode

枚举
stable
代码

一个用于 Router 的 NavigationCancel 事件的编码，用于指示导航失败的原因。

API

    
      enum NavigationCancellationCode {}

重定向

导航失败，因为守卫返回了一个 UrlTree 以进行重定向。

SupersededByNewNavigation

导航失败，因为启动了一个更新的导航。

NoDataFromResolver

导航失败，因为其中一个求解器在完成时没有发出值。

GuardRejected

导航失败，因为守卫返回了 false。

已中止

导航被 Navigation.abort 函数中止了。
