getListeners

stable
代码

检索与 DOM 元素关联的事件监听器列表。该列表包含宿主监听器，但不包含在 Angular 上下文之外定义的事件监听器（例如，通过 addEventListener）。

API

function getListeners(element: Element): Listener[];

getListeners

Listener[]

检索与 DOM 元素关联的事件监听器列表。该列表包含宿主监听器，但不包含在 Angular 上下文之外定义的事件监听器（例如，通过 addEventListener）。

@paramelementElement

应该在其中检索 DOM 监听器的元素。

@returnsListener[]

使用须知

假定有以下 DOM 结构：

<app-root>  <div (click)="doSomething()"></div></app-root>

在 <div> 上调用 getListeners 将返回一个如下所示的对象：

{  name: 'click',  element: <div>,  callback: () => doSomething(),  useCapture: false}
