CLI

serve

命令
ng serve
ng dev
ng s

构建并为你的应用启动服务器，并在文件更改时重新构建。

参数

project

要构建的项目名称。可以是一个应用或一个库。

值类型string

选项

allowed-hosts

开发服务器将响应的主机。此选项设置同名的 Vite 选项。更多详情：https://vite.dev/config/server-options.html#server-allowedhosts

值类型boolean
build-target

要启动服务器的构建器目标，格式为 project:target[:configuration]。你也可以传入多个配置名称，以逗号分隔的列表形式。范例：project:target:production,staging。

值类型string
configuration
别名c

一个或多个已命名的构建器配置，以逗号分隔的列表形式，在 angular.json 的 "configurations" 部分中指定。 构建器使用已命名的配置来运行给定的目标。 欲知详情，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

值类型string
headers

要添加到所有响应的自定义 HTTP 标头。

值类型array
help

在控制台中显示此命令的帮助信息。

值类型boolean
hmr

启用热模块替换。默认为 'liveReload' 的值。目前仅支持全局和组件样式表。

值类型boolean
host

要监听的地址。

值类型string默认值localhost
inspect

激活调试检查器。此选项仅在启用“SSR”或“SSG”时有效。

值类型string
live-reload

是否在更改时重新加载页面，使用动态重新加载。

值类型boolean默认值true
open
别名o

在默认浏览器中打开 URL。

值类型boolean默认值false
poll

启用并定义文件监视轮询时间段，以毫秒为单位。

值类型number
port

要监听的端口。

值类型number默认值4200
prebundle

启用并控制基于 Vite 的开发服务器的预捆绑功能。要启用预捆绑，还必须启用 Angular CLI 缓存。

值类型boolean默认值true
proxy-config

代理配置文件。欲知详情，请参阅 https://angular.dev/tools/cli/serve#proxying-to-a-backend-server。

值类型string
serve-path

应用将被伺服的路径名。

值类型string
ssl

使用 HTTPS 伺服。

值类型boolean默认值false
ssl-cert

用于使用 HTTPS 伺服的 SSL 证书。

值类型string
ssl-key

用于使用 HTTPS 伺服的 SSL 密钥。

值类型string
verbose

向输出日志添加更多详细信息。

值类型boolean
watch

更改时重新构建。

值类型boolean默认值true
