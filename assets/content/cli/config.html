CLI

配置

命令
ng config

工作区在顶层有一个 CLI 配置文件 angular.json。
projects 对象包含工作区中每个项目的配置对象。

你可以直接在代码编辑器中编辑配置，
也可以使用此命令在命令行中间接编辑。

可配置的属性名称与命令选项名称匹配，
不同之处在于配置文件中，所有名称都必须使用 camelCase，
而在命令行选项中可以使用 dash-case。

更多详情，请参阅工作区配置。

有关 CLI 使用情况分析的配置，请参阅 ng analytics。

参数

json-path

要设置或查询的配置键，以 JSON 路径格式表示。例如：“a[3].foo.bar[2]”。如果未提供新值，则返回此键的当前值。

值类型string
value

如果提供，则为给定配置键的新值。

值类型string

选项

global
别名g

访问调用者主目录中的全局配置。

值类型boolean默认值false
help

在控制台中显示此命令的帮助消息。

值类型boolean
