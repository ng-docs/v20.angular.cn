CLI

extract-i18n

命令
ng extract-i18n

从源码中提取 i18n 消息。

参数

project

要构建的项目名称。可以是一个应用或一个库。

值类型string

选项

build-target

用于提取 i18n 消息的构建器目标，格式为 project:target[:configuration]。你也可以传入多个配置名称，以逗号分隔的列表形式。范例：project:target:production,staging。

值类型string
configuration
别名c

一个或多个命名的构建器配置，以逗号分隔的列表形式，在 angular.json 的 "configurations" 部分中指定。 构建器使用这些命名的配置来运行给定的目标。 更多信息，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

值类型string
format

生成文件的输出格式。

值类型string允许值arb, json, legacy-migrate, xlf, xlf2, xlif, xliff, xliff2, xmb默认值xlf
help

在控制台中显示此命令的帮助消息。

值类型boolean
i18n-duplicate-translation

如何处理重复的翻译。

值类型string允许值error, ignore, warning
out-file

要输出的文件名。

值类型string
output-path

输出文件将放置的路径。

值类型string
progress

将进度记录到控制台。

值类型boolean默认值true
