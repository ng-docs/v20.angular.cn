CLI

run

命令
ng run

Architect 是 CLI 使用的工具，用于根据提供的配置执行诸如编译等复杂任务。 CLI 命令运行 Architect 目标，例如 build、serve、test 和 lint。 每个具名目标都有一个默认配置（由 options 对象指定），以及 configurations 对象中一组可选的具名备用配置。

例如，新生成的应用的 serve 目标具有一个名为 production 的预定义备用配置。

你可以在 angular.json 文件的 architect 部分定义新的目标及其配置选项，然后可以使用 ng run 命令从命令行运行它们。

参数

target

要运行的 Architect 目标，以下列格式提供：project:target[:configuration]。

值类型string

选项

help

在控制台中显示此命令的帮助消息。

值类型boolean
