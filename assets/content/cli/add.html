CLI

add

命令
ng add

将已发布库的 npm 包添加到你的工作区，并配置当前工作目录中的项目以使用该库，具体配置由该库的原理图指定。 例如，添加 @angular/pwa 会将你的项目配置为支持 PWA：

ng add @angular/pwa

参数

collection

要添加的包。

值类型string

选项

defaults

禁用带默认值的选项的交互式输入提示。

值类型boolean默认值false
dry-run
别名d

贯穿运行并报告活动，但不写出结果。

值类型boolean默认值false
force

强制覆盖现有文件。

值类型boolean默认值false
help

在控制台中显示此命令的帮助消息。

值类型boolean
interactive

启用交互式输入提示。

值类型boolean默认值true
registry

要使用的 NPM 注册表。

值类型string
skip-confirmation

跳过在安装和执行包之前显示确认提示。请确保包名称在使用此选项之前是正确的。

值类型boolean默认值false
verbose

显示有关执行期间内部操作的其他详细信息。

值类型boolean默认值false
