CLI

deploy

命令
ng deploy

本命令接受一个可选的项目名称，正如 angular.json 工作区配置文件的 projects 部分所指定的。 当未提供项目名称时，则为默认项目执行 deploy 构建器。

要使用 ng deploy 命令，请使用 ng add 添加一个软件包，以便为你的首选平台实现部署功能。 添加软件包会自动更新你的工作区配置，并添加一个部署CLI 构建器。 例如：

"projects": {  "my-project": {    ...    "architect": {      ...      "deploy": {        "builder": "@angular/fire:deploy",        "options": {}      }    }  }}

参数

project

要构建的项目的名称。可以是一个应用或一个库。

值类型string

选项

configuration
别名c

一个或多个命名构建器配置，以逗号分隔的列表形式，在 angular.json 中的 "configurations" 部分中指定。 构建器使用这些命名配置来运行给定的目标。 更多信息，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

值类型string
help

在控制台中显示此命令的帮助信息。

值类型boolean
