CLI

lint

命令
ng lint

该命令接受一个可选的项目名称，如 angular.json 工作区配置文件的 projects 部分所指定。 当未提供项目名称时，将为所有项目执行 lint 构建器。

要使用 ng lint 命令，请使用 ng add 添加一个实现了 lint 功能的包。添加该包会自动更新你的工作区配置，并添加一个 lint CLI 构建器。 例如：

"projects": {  "my-project": {    ...    "architect": {      ...      "lint": {        "builder": "@angular-eslint/builder:lint",        "options": {}      }    }  }}

参数

project

要构建的项目的名称。可以是应用或库。

值类型string

选项

configuration
别名c

一个或多个命名的构建器配置，以逗号分隔的列表形式在 angular.json 的 "configurations" 部分中指定。 构建器使用命名的配置来运行给定的目标。 更多信息，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

值类型string
help

在控制台中显示此命令的帮助消息。

值类型boolean
