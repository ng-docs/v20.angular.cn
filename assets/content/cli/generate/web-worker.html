CLI

生成 web-worker

命令
ng generate web-worker

在你的项目中创建一个新的 web worker。Web worker 允许你在后台运行 JavaScript 代码，通过卸载计算密集型任务来提高应用的性能和响应性。这个原理图为新的 web worker 生成必要的文件，并提供一个可选的代码片段来演示其用法。

参数

name

新 web worker 的名称。这将用于创建 worker 文件（例如，my-worker.worker.ts）。

值类型string

选项

project

应该在其中创建 web worker 的项目名称。如果未指定，CLI 将从当前目录确定项目。

值类型string
snippet

生成一个代码片段，演示如何创建和使用新的 web worker。

值类型boolean默认值true
