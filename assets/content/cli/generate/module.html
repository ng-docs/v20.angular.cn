CLI

generate module

命令
ng generate module
ng generate m

在给定的项目中创建一个新的通用 NgModule 定义。

参数

name

NgModule 的名称。

值类型string

选项

flat

在当前项目根目录的顶层创建新文件。

值类型boolean默认值false
module
别名m

声明它的 NgModule。

值类型string
project

项目的名称。

值类型string
route

惰性加载模块的路由路径。当提供时，将在新模块中创建一个组件，并将该组件的路由添加到 --module 选项中提供的模块里声明的 Routes 数组中。

值类型string
routing

创建一个路由模块。

值类型boolean默认值false
routing-scope

新路由模块的作用域。

值类型string允许值Child, Root默认值Child
type-separator

在生成的文件名中使用在类型之前的分隔符字符。例如，如果将选项设置为 .，则文件名将是 example.module.ts。

值类型string允许值-, .默认值-
