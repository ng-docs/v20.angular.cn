CLI

生成类

命令
ng generate class
ng generate cl

在你的项目中创建一个新类。类是 TypeScript 中面向对象编程的基础构建块。它们为创建具有属性和方法的对象提供了一个蓝图。这个原理图可以帮助你生成一个具有基本结构和可选测试文件的新类。

参数

name

新类的名称。这将用于创建类文件（例如，my-class.ts），如果启用，还将创建相应的测试文件 my-class.spec.ts。

值类型string

选项

project

应该在其中添加此类的项目的名称。如果未指定，CLI 将从当前目录确定项目。

值类型string
skip-tests

跳过为新类生成单元测试文件 spec.ts。

值类型boolean默认值false
type

向文件名添加自定义类型，允许你创建更具描述性的类名。例如，如果将类型设置为 helper，则文件名将为 my-class.helper.ts。

值类型string
