CLI

生成 interface

命令
ng generate interface
ng generate i

在你的项目中创建一个新的 interface。Interface 定义了 TypeScript 中对象的结构，确保了类型安全和代码清晰度。这个原理图会生成一个具有指定名称和类型的新 interface。

参数

name

新 interface 的名称。这将用于创建 interface 文件 (例如，my-interface.interface.ts)。

值类型string
type

向文件名添加自定义类型，允许你创建更具描述性的 interface 名称。例如，如果你将类型设置为 data，则文件名将为 my-interface.data.ts。

值类型string

选项

prefix

要添加到 interface 名称的前缀。这通常不用于 interface，因为它们没有像组件或指令那样的选择器。

值类型string
project

要在其中创建 interface 的项目名称。如果未指定，CLI 将从当前目录确定项目。

值类型string
