CLI

生成枚举

命令
ng generate enum
ng generate e

在你的项目中创建一个新的枚举。枚举（enumerations）是一种定义一组命名常量的方式，使你的代码更具可读性和可维护性。此原理图使用指定的名称和类型生成一个新的枚举。

参数

name

新枚举的名称。这将用于创建枚举文件（例如，my-enum.enum.ts）。

值类型string

选项

project

要在其中创建枚举的项目名称。如果未指定，CLI 将从当前目录确定项目。

值类型string
type

向文件名添加自定义类型，允许你创建更具描述性的枚举名称。例如，如果将类型设置为 status，则文件名将为 my-enum.status.ts。

值类型string
