什么是 Angular？

edit

Angular 是一个 Web 框架，旨在帮助开发者构建快速、可靠的应用。

Angular 由 Google 的专业团队维护，提供了一整套广泛的工具、API 和库，以简化和优化你的开发工作流程。Angular 为你提供了一个坚实的平台，使你能够构建快速、可靠的应用，并能随着你的团队规模和代码库规模的扩大而扩展。

想看一些代码示例吗？ 请跳转到我们的 入门指南 以快速了解 Angular 的使用方式，或者如果你喜欢按照步骤操作，请从教程开始。

为你的开发赋能的特性

使用具有统一规范的组件模型和灵活的依赖注入体系，保持代码库的井井有条

Angular 组件使你可以轻松地将代码拆分为良好封装的部分。

通用的依赖注入有助于你保持代码的模块化、松耦合和可测试性。

组件入门

通过基于 Signal 的细粒度响应式系统，获得快速的状态更新

我们细粒度的响应式模型，结合编译时优化，简化了开发，并有助于默认构建更快的应用。

细粒度地跟踪状态在整个应用中的使用方式和位置，使框架能够通过高度优化的指令渲染快速更新。

探索 Angular Signal

通过 SSR、SSG、水合和下一代延迟加载，实现你的性能目标

Angular 同时支持服务端渲染 (SSR) 和静态站点生成 (SSG) 以及完整的 DOM 水合。模板中的 @defer 控制块可以轻松地将模板声明式地划分为可惰性加载的部分。

阅读关于 SSR 的内容

使用 Angular 的表单、路由等第一方模块，保证一切协同工作

Angular 的路由器 提供了一个功能丰富的导航工具包，包括对路由守卫、数据求解、惰性加载等的支持。

Angular 的表单模块 提供了一个用于表单参与和验证的标准化系统。

以超乎以往的速度开发应用

使用 Angular CLI，轻松构建、启动服务器、测试和部署

Angular CLI 可以在不到一分钟的时间内启动你的项目，并提供你所需的命令，以便将其发展为已部署的生产应用。

Angular CLI

使用 Angular DevTools 浏览器扩展，以可视化方式调试、分析和优化你的代码

Angular DevTools 与你的浏览器开发者工具并排就坐。它可以帮助调试和分析你的应用，包括组件树检查器、依赖注入树视图和自定义性能分析火焰图。

Angular DevTools

使用 ng update，绝不会错过任何版本

Angular CLI 的 ng update 运行自动化代码转换，自动处理例行的破坏性变更，从而大大简化了主版本更新。保持使用最新版本可使你的应用尽可能快速和安全。

ng update

通过在你最喜欢的编辑器中集成 IDE，保持高效

Angular 的 IDE 语言服务在你最喜欢的编辑器中支持代码补全、导航、重构和实时诊断。

语言服务

充满信心地交付

在 Google 庞大的单一代码仓中，逐个提交地进行验证

每个 Angular 提交都会在 Google 的内部代码仓库中针对成千上万的测试进行检查，代表着无数真实的场景。

Angular 致力于为 Google 的一些最大型产品（包括 Google Cloud）提供稳定性。这一承诺确保变更都经过充分测试、向后兼容，并在可能的情况下包含迁移工具。

了解关于 Google 的单一代码仓

清晰的支持策略和可预测的发布时间表

Angular 可预测的、基于时间的发布时间表，使你的组织对框架的稳定性和向后兼容性充满信心。长期支持 (LTS) 窗口确保你在需要时获得关键的安全修复。第一方更新工具、指南和自动化迁移原理图有助于使你的应用与框架和 Web 平台的最新进展保持同步。

版本控制与发布

适用于任何规模

通过国际化支持，覆盖全球用户

Angular 的国际化特性处理消息翻译和格式化，包括对 Unicode 标准 ICU 语法的支持。

国际化

通过默认安全保护你的用户

通过与 Google 世界一流的安全工程师合作，Angular 旨在使开发默认情况下是安全的。内置的安全特性，包括 HTML 无害化和可信类型支持，有助于保护你的用户免受常见的漏洞攻击，如跨站脚本和跨站请求伪造。

安全性

使用 Vite 和 esbuild，保持大型团队的高效

Angular CLI 包含一个使用 Vite 和 ESBuild 的快速、现代化的构建流水线。开发者报告说，构建包含数十万行代码的项目只需不到一分钟的时间。

ESBuild 和 Vite

已在 Google 的一些最大的 Web 应用中得到验证

大型 Google 产品基于 Angular 的架构构建，并帮助开发新的特性，从而进一步提高 Angular 的延展性，从 Google Fonts 到 Google Cloud。
